📌 도입: 예시 문항

예시 — 직각이 보이는 가장 쉬운 케이스

문제: 직각삼각형 $ABC$에서 $\angle C = 90°$, $\overline{AB} = 5$, $\angle B = 60°$ 일 때 $\overline{AC}$의 길이를 구하시오.

📖 풀이 보기

STEP 1: $\overline{AC}$는 $\angle B$의 대변. $\sin B = \dfrac{\overline{AC}}{\overline{AB}}$.

STEP 2: $\overline{AC} = 5\sin 60° = 5\cdot \dfrac{\sqrt{3}}{2} = \dfrac{5\sqrt{3}}{2}$.

정답: $\dfrac{5\sqrt{3}}{2}$

핵심 포인트: 직각이 보이면 삼각비 한 줄로 끝. 사인·코사인 법칙은 직각이 안 보일 때 꺼내는 도구다.

💡 개념 설명 (파인만식 4단계)

① 정의 (한 줄)

삼각형의 해석이란, 삼각형의 6요소(3각 + 3변) 중 부분 정보로 나머지를 구하는 작업. 도구는 4가지 — 피타고라스 정리, 삼각비, 사인법칙, 코사인법칙.

② 비유

삼각형은 6명이 사는 동아리방이에요. 이름표 6개(각 3 + 변 3) 중 일부만 보고도 가려진 이름표를 추리해야 해요. '직각'이라는 특별 손님이 있으면 가장 쉬운 도구(피타고라스·삼각비)가 통하고, 없으면 두 가지 새 도구(사인·코사인 법칙)를 꺼내요.

③ 작동 원리

의사결정 트리 (가장 중요)

문제에서 직각이 보이는가? ├─ YES → 피타고라스 정리 / 삼각비 (가장 간단) └─ NO → ┬─ (변, 대각) 짝 또는 외접원 R 등장? → 사인법칙 ├─ 두 변 + 끼인각, 또는 세 변 모두? → 코사인법칙 └─ 직각이 필요하면? → 수선의 발 그리기

사인법칙   $\dfrac{a}{\sin A}=\dfrac{b}{\sin B}=\dfrac{c}{\sin C}=2R$

$R$은 외접원의 반지름 (외접원 = 세 꼭짓점을 모두 지나는 원).

왜 $2R$이 튀어나오나? 한 호에 대한 원주각은 모두 같다는 중3 정리를 이용. 원주각으로 직각삼각형 $A'BC$를 만들면 빗변이 지름 $2R$이 되고, $\sin A = \sin A' = \dfrac{a}{2R}$가 자연스레 나옵니다.

코사인법칙   $a^2 = b^2 + c^2 - 2bc\cos A$ (순환적으로 3가지 형태)

변형식: $\cos A = \dfrac{b^2+c^2-a^2}{2bc}$ (세 변에서 각을 추출할 때)

왜 이렇게 되나? 한 꼭짓점에서 마주보는 변에 수선의 발을 내리면 직각삼각형 두 개가 생기고, 그 안에서 피타고라스 + 삼각비가 결합되어 위 식이 나옵니다. 즉 코사인법칙은 피타고라스 정리의 확장판입니다 ($A=90°$이면 $\cos A=0$이라 정확히 피타고라스로 환원).

삼각형 넓이   $S = \dfrac{1}{2}bc\sin A$ (두 변과 끼인각 패턴)

④ 자주 하는 실수

  1. 변의 비 ≠ 각의 비. $A:B:C = 1:2:3$ 이라고 $a:b:c = 1:2:3$ 이 아닙니다. 사인법칙에 의해 $a:b:c = \sin A:\sin B:\sin C$ — 사인 값의 비와 같습니다.
  2. 코사인법칙의 음수 부호 빠짐. $-2bc\cos A$의 마이너스 누락이 가장 흔한 실수. 둔각이면 $\cos A < 0$ 이라 부호가 다시 뒤집히니 끝까지 부호 추적!
  3. 외접원 R, 한 쌍이면 충분. $(a, A)$ 한 쌍만 알면 $2R = \dfrac{a}{\sin A}$ 로 즉시 R을 구할 수 있어요. 세 쌍 모두 필요하다고 착각 금지.

🟢 기초

기초 1 — 사인법칙 단순 적용

문제: 삼각형 $ABC$ 에서 $a=4$, $\angle A = 30°$, $\angle B = 45°$ 일 때 $b$의 값을 구하시오.

📖 풀이 보기

STEP 1: 사인법칙 $\dfrac{a}{\sin A} = \dfrac{b}{\sin B}$ 적용.

STEP 2: $\dfrac{4}{\sin 30°} = \dfrac{b}{\sin 45°}$ → $\dfrac{4}{1/2} = \dfrac{b}{\sqrt{2}/2}$ → $8 = \dfrac{2b}{\sqrt{2}}$ → $b = 4\sqrt{2}$.

정답: $4\sqrt{2}$

핵심 포인트: (변, 대각) 쌍이 보이면 사인법칙 1순위.

오답 주의: $\sin 30°$ 와 $\sin 45°$ 의 위치(분자·분모)를 바꿔 대입 — 변과 대각의 짝을 헷갈리지 말 것.

기초 2 — 코사인법칙 단순 적용

문제: 삼각형 $ABC$ 에서 $b=3$, $c=5$, $\angle A = 60°$ 일 때 $a$의 값을 구하시오.

📖 풀이 보기

STEP 1: 두 변 + 끼인각 → 코사인법칙 $a^2 = b^2 + c^2 - 2bc\cos A$.

STEP 2: $a^2 = 9 + 25 - 2\cdot 3\cdot 5\cdot \dfrac{1}{2} = 34 - 15 = 19$.

STEP 3: $a = \sqrt{19}$.

정답: $\sqrt{19}$

핵심 포인트: "두 변 + 끼인각" 패턴은 항상 코사인법칙.

오답 주의: $-2bc\cos A$ 의 음수 부호 누락이 가장 흔한 실수.

기초 3 — 외접원 R 직접 계산

문제: 외접원의 반지름이 $R=4$ 인 삼각형 $ABC$ 에서 $\angle A = 30°$ 일 때 $a$ 의 값을 구하시오.

📖 풀이 보기

STEP 1: 사인법칙의 외접원 형태 $\dfrac{a}{\sin A} = 2R$ 적용.

STEP 2: $a = 2R\sin A = 2\cdot 4 \cdot \sin 30° = 8 \cdot \dfrac{1}{2} = 4$.

정답: $4$

핵심 포인트: $a = 2R\sin A$ 는 외접원이 등장하면 즉시 꺼내는 형태.

오답 주의: $R$ 과 $2R$ 헷갈림 — 사인법칙 분모는 지름 $2R$.

🟡 기본

기본 1 — 변형식으로 각 구하기

문제: 삼각형 $ABC$ 에서 $a=7$, $b=5$, $c=3$ 일 때 $\cos A$ 의 값을 구하시오.

📖 풀이 보기

STEP 1: 세 변에서 각을 구할 땐 코사인법칙 변형식 $\cos A = \dfrac{b^2 + c^2 - a^2}{2bc}$.

STEP 2: $\cos A = \dfrac{25 + 9 - 49}{2\cdot 5\cdot 3} = \dfrac{-15}{30} = -\dfrac{1}{2}$. (따라서 $\angle A = 120°$, 둔각)

정답: $-\dfrac{1}{2}$

핵심 포인트: 세 변 모두 알면 변형식으로 각을 한 번에 추출.

오답 주의: $b^2+c^2-a^2$ 에서 $a$ 가 구하려는 각의 대변임을 혼동.

기본 2 — 변의 비 ↔ 사인의 비

문제: 삼각형 $ABC$ 에서 $\sin A : \sin B : \sin C = 3:5:7$ 이고 $b=10$ 일 때 $a + c$ 의 값을 구하시오.

📖 풀이 보기

STEP 1: 사인법칙 $\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C}$ 에서 $a:b:c = \sin A:\sin B:\sin C = 3:5:7$.

STEP 2: $b=10$ 이므로 비례상수 $k = 2$. 따라서 $a = 6$, $c = 14$.

STEP 3: $a + c = 20$.

정답: $20$

핵심 포인트: 사인의 비 = 변의 비. 각의 비가 아니라 사인 값의 비!

오답 주의: $\sin A:\sin B:\sin C$ 를 $\angle A:\angle B:\angle C$ 로 잘못 해석 — 영상이 강조한 함정.

기본 3 — 넓이 공식

문제: 삼각형 $ABC$ 에서 $b=6$, $c=8$, $\angle A = 30°$ 일 때 넓이 $S$ 를 구하시오.

📖 풀이 보기

STEP 1: 두 변 + 끼인각 → 넓이 공식 $S = \dfrac{1}{2}bc\sin A$.

STEP 2: $S = \dfrac{1}{2}\cdot 6\cdot 8\cdot \sin 30° = 24\cdot \dfrac{1}{2} = 12$.

정답: $12$

핵심 포인트: "두 변 + 끼인각" 패턴은 → 넓이까지 한 번에 처리.

오답 주의: $\sin$ 대신 $\cos$ 를 적용 — 넓이는 항상 $\sin$.

🔴 심화

심화 1 — 외접원과 사인법칙 변형식 결합

문제: 삼각형 $ABC$ 의 외접원 반지름이 $R = \sqrt{3}$ 이고 $a^2 + b^2 + c^2 = 27$ 일 때 $\sin^2 A + \sin^2 B + \sin^2 C$ 의 값을 구하시오.

📖 풀이 보기

STEP 1: 사인법칙 $\dfrac{a}{\sin A} = 2R$ 에서 $\sin A = \dfrac{a}{2R}$.

STEP 2: 양변 제곱: $\sin^2 A = \dfrac{a^2}{4R^2}$. $B$, $C$ 도 동일.

STEP 3: $\sin^2 A + \sin^2 B + \sin^2 C = \dfrac{a^2 + b^2 + c^2}{4R^2}$.

STEP 4: $= \dfrac{27}{4 \cdot 3} = \dfrac{27}{12} = \dfrac{9}{4}$.

정답: $\dfrac{9}{4}$

핵심 포인트: 사인법칙을 $\sin = \dfrac{변}{2R}$ 형태로 뒤집어 쓰면 사인 ↔ 변 통합 변환이 자유로워짐.

오답 주의: $4R^2$ 대신 $2R^2$ 또는 $R^2$ 사용 (제곱하면서 분모 처리 실수).

출제 의도: 사인법칙을 단순 적용이 아닌 변형식 형태로 다루는 능력 점검 (수능 단답형 단골 패턴).

심화 2 — 코사인법칙 + 넓이 공식 연계

문제: 삼각형 $ABC$ 에서 $a=7$, $b=8$, $c=5$ 일 때 삼각형의 넓이 $S$ 를 구하시오.

📖 풀이 보기

STEP 1: 세 변에서 각을 추출. 코사인법칙 변형식으로 $\angle A$ 부터: $\cos A = \dfrac{b^2 + c^2 - a^2}{2bc} = \dfrac{64 + 25 - 49}{2\cdot 8\cdot 5} = \dfrac{40}{80} = \dfrac{1}{2}$.

STEP 2: $\cos A = \dfrac{1}{2}$ → $\angle A = 60°$ → $\sin A = \dfrac{\sqrt{3}}{2}$.

STEP 3: 넓이 공식 $S = \dfrac{1}{2}bc\sin A = \dfrac{1}{2}\cdot 8\cdot 5\cdot \dfrac{\sqrt{3}}{2}$.

STEP 4: $S = 20\cdot \dfrac{\sqrt{3}}{2} = 10\sqrt{3}$.

정답: $10\sqrt{3}$

핵심 포인트: 세 변만 주어진 경우 코사인법칙 → 각 → 넓이 의 두 단계 콤보.

오답 주의: $A$ 의 대변이 $a$ 임을 헷갈려 분자에 다른 변을 넣는 실수.

출제 의도: 두 도구(코사인법칙·넓이 공식)를 순차 결합 — 헤론 공식 없이도 넓이를 구할 수 있는 표준 경로.

🟣 수능 대비

수능 대비 1 — 단답형 4점 유형 (외접원 결합)

문제: 원에 내접하는 삼각형 $ABC$ 에서 $\overline{AB}=8$, $\overline{AC}=6$, $\angle A = 60°$ 이다. 외접원의 넓이를 $\dfrac{p}{q}\pi$ ($p$, $q$ 는 서로소인 자연수) 로 나타낼 때 $p+q$ 의 값을 구하시오.

📖 풀이 보기

STEP 1: 두 변 + 끼인각이므로 코사인법칙으로 마주보는 변 $\overline{BC}$ 추출. $\overline{BC}^2 = 8^2 + 6^2 - 2\cdot 8\cdot 6\cdot \cos 60° = 64 + 36 - 48 = 52$. 따라서 $\overline{BC} = 2\sqrt{13}$.

STEP 2: $\overline{BC}$ 는 $\angle A$ 의 대변. 사인법칙 $\dfrac{\overline{BC}}{\sin A} = 2R$ 적용.

STEP 3: $2R = \dfrac{2\sqrt{13}}{\sin 60°} = \dfrac{2\sqrt{13}}{\sqrt{3}/2} = \dfrac{4\sqrt{13}}{\sqrt{3}} = \dfrac{4\sqrt{39}}{3}$. 따라서 $R = \dfrac{2\sqrt{39}}{3}$.

STEP 4: 외접원 넓이 $= \pi R^2 = \pi \cdot \dfrac{4\cdot 39}{9} = \dfrac{156}{9}\pi = \dfrac{52}{3}\pi$.

STEP 5: $p = 52$, $q = 3$, $\gcd(52, 3) = 1$ 이므로 서로소 ✓. $p + q = 55$.

정답: $55$

핵심 포인트: 코사인법칙(변 추출) → 사인법칙(외접원 R) 의 표준 콤보.

오답 주의: $2R$ 과 $R$ 혼동 — 사인법칙 분모는 $2R$. 외접원 넓이 공식엔 $R$ 자체.

출제 의도: 두 법칙의 연계 + 분수 정수화 처리. 수능 단답형(4점) 단골 구조.

수능 대비 2 — 21번 유형 (각의 비 + 직각 결합)

문제: 삼각형 $ABC$ 에서 $\angle A : \angle B : \angle C = 1 : 2 : 3$ 이고 외접원의 반지름이 $R = 4$ 이다. $a^2 + b^2 + c^2$ 의 값을 구하시오.

📖 풀이 보기

STEP 1: 각의 합 $= 180°$, 비 $1:2:3$ → 합 6 → 각각 $30°, 60°, 90°$. 따라서 $\angle A = 30°$, $\angle B = 60°$, $\angle C = 90°$.

STEP 2: $\angle C = 90°$ 인 삼각형은 빗변 $c$ 가 외접원의 지름. 따라서 $c = 2R = 8$, $c^2 = 64$.

STEP 3: 사인법칙으로 $a$, $b$ 추출. $a = 2R\sin A = 8\sin 30° = 4$ → $a^2 = 16$. $b = 2R\sin B = 8\sin 60° = 4\sqrt{3}$ → $b^2 = 48$.

STEP 4: $a^2 + b^2 + c^2 = 16 + 48 + 64 = 128$.

정답: $128$

핵심 포인트: 각의 비 → 실제 각 변환 후, 직각이 보이면 빗변 = 지름 이라는 원의 성질을 즉시 활용 (영상의 "지름의 원주각=$90°$" 의 역방향 사용).

오답 주의: $1:2:3$ 을 변의 비로 착각해서 $a:b:c$ 에 그대로 대입 — 각의 비는 변의 비가 아니다.

출제 의도: 직각삼각형의 외접원 = 빗변 절반 이라는 사실을 사인법칙 안에서 일관되게 다루기. 수능 21번(준킬러) 도형 진입 패턴.
