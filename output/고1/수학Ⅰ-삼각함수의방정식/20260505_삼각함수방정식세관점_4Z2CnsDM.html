삼각함수 방정식 $\sin A = \cos B$ — 세 관점 통합 정리

📌 도입: 영상이 던진 질문

예시 — 영상의 출발 문제

문제: $\sin 2x = \cos(3x - 3\pi)$ 를 만족하는 $x$ 들 중 하나의 형태를 구하시오.

📖 풀이 보기 (영상의 두 결론)

STEP 1 — 치환: $a = 2x$, $b = 3x - 3\pi$ 로 두면 식은 $\sin a = \cos b$ 형태.

STEP 2 — 정리 1번 (선대칭 결론): $a + b = \dfrac{\pi}{2} + 2n\pi$ ($n$은 정수). 대입하면 $2x + (3x - 3\pi) = \dfrac{\pi}{2} + 2n\pi$ → $5x = \dfrac{7\pi}{2} + 2n\pi$ → $x = \dfrac{7\pi}{10} + \dfrac{2n\pi}{5}$.

STEP 3 — 정리 2번 (평행이동 결론): $a - b = \dfrac{\pi}{2} + 2n\pi$ → $2x - (3x - 3\pi) = \dfrac{\pi}{2} + 2n\pi$ → $-x + 3\pi = \dfrac{\pi}{2} + 2n\pi$ → $x = \dfrac{5\pi}{2} - 2n\pi$.

정답: $x = \dfrac{7\pi}{10} + \dfrac{2n\pi}{5}$ 또는 $x = \dfrac{5\pi}{2} - 2n\pi$ ($n$ 정수). 범위 제한 시 그 안의 해만.

핵심 포인트: $\sin A = \cos B$ 형태의 핵심 정리는 두 가지 — $A + B = \pi/2 + 2n\pi$ 또는 $A - B = \pi/2 + 2n\pi$. 본 자료에서 왜 그 두 결론이 나오는지를 세 가지 관점으로 모두 유도합니다.

💡 개념 설명 (세 관점 통합)

① 큰 그림 — 문제는 단순, 왜 어려운가?

$\sin A = \cos B$. 식만 보면 단순합니다. 그런데 막상 풀려고 하면 막혀요. 이유는 사인과 코사인이 서로 다른 함수처럼 보이기 때문이에요.

영상 강사의 핵심 메시지: "세 가지 다른 풀이처럼 보이는 게 사실 모두 같은 정리". 그 셋을 다 다룰 줄 알아야, 문제마다 가장 빠른 길을 고를 수 있어요. 이 자료에서 셋을 모두 유도해봅니다.

② 선수 지식 1 — 사인/코사인의 정의 (단위원)

단위원: 원점이 중심이고 반지름이 $1$인 원.

각 $\theta$ 에 대응하는 점 = 단위원 위에서 양의 $x$축으로부터 반시계 방향으로 $\theta$ 만큼 회전한 점.

그 점의 좌표 $= (\cos\theta, \sin\theta)$. 즉 $\cos\theta$ 는 $x$ 좌표, $\sin\theta$ 는 $y$ 좌표.

이 정의는 ⑥(단위원 관점)에서 직접 사용됩니다. 코사인을 "옆으로 누운 사인"으로 보는 시각의 출발점.

③ 선수 지식 2 — 사인·코사인 그래프와 주기

$y = \sin x$: $x = 0$에서 시작 → $x = \pi/2$에서 최댓값 $1$ → $x = \pi$에서 $0$ → $x = 3\pi/2$에서 최솟값 $-1$ → $x = 2\pi$에서 다시 $0$.

$y = \cos x$: $x = 0$에서 최댓값 $1$ → $x = \pi/2$에서 $0$ → $x = \pi$에서 $-1$ → $x = 3\pi/2$에서 $0$ → $x = 2\pi$에서 다시 $1$.

주기성: 두 함수 모두 $2\pi$ 마다 같은 모양이 반복. $\sin(x + 2\pi) = \sin x$, $\cos(x + 2\pi) = \cos x$.

한 줄 핵심: $\cos$ 그래프는 $\sin$ 그래프를 왼쪽으로 $\pi/2$ 평행이동한 것과 같음 → $\cos x = \sin(x + \pi/2)$ 또는 $\sin(\pi/2 - x)$.

④ 핵심 정리 1 — 두 함수 그래프 관점 (영상 1번 풀이)

$\sin A = \cos B$ 가 의미하는 것 — $\sin$ 그래프 위 $x = A$ 에서의 $y$값이, $\cos$ 그래프 위 $x = B$ 에서의 $y$값과 같다.

즉 두 그래프를 같은 평면에 그리고 같은 $y$값을 갖는 두 점을 찾는 것이 핵심입니다.

두 그래프 같이 그려보면: 두 그래프는 $x = \pi/4$ 근처에서 처음 만남 ($\sin(\pi/4) = \cos(\pi/4) = \sqrt{2}/2$). 그 만나는 점을 기준으로 두 가지 패턴이 보입니다.

패턴 1 — 선대칭 (두 점이 한 직선에 대해 대칭):

$\sin$ 그래프의 $A$ 점과 $\cos$ 그래프의 $B$ 점이 직선 $x = \pi/4$ 에 대해 대칭이면 $y$값이 같습니다.

대칭축이 $x = \pi/4$ 이면 두 점 $A$, $B$ 의 중점이 $\pi/4$. 즉 $\dfrac{A + B}{2} = \dfrac{\pi}{4}$.

양변 $\times 2$: $A + B = \dfrac{\pi}{2}$.

주기성 고려 (그래프가 $2\pi$마다 반복) → 모든 해: $A + B = \dfrac{\pi}{2} + 2n\pi$ ($n$ 정수).

패턴 2 — 평행이동 (한 점을 옆으로 이동시키면 다른 점이 됨):

$\cos$ 그래프는 $\sin$ 그래프를 왼쪽으로 $\pi/2$ 이동한 것이라, $\cos$ 의 $B$ 점에 해당하는 $\sin$ 위 위치는 $B + \pi/2$. 따라서 $\sin A = \sin(B + \pi/2)$.

"두 사인 함수에서 같은 값" → $A = B + \pi/2 + 2n\pi$ 또는 $A = \pi - (B + \pi/2) + 2n\pi$.

첫 경우: $A - B = \dfrac{\pi}{2} + 2n\pi$.

둘째 경우: $A = \pi - B - \dfrac{\pi}{2} + 2n\pi$ → $A + B = \dfrac{\pi}{2} + 2n\pi$ (패턴 1과 같음).

두 결론 (핵심 정리):
$A + B = \dfrac{\pi}{2} + 2n\pi$   또는   $A - B = \dfrac{\pi}{2} + 2n\pi$

⑤ 핵심 정리 2 — 각 변환으로 한 함수 통일 (영상 2번 풀이)

두 함수를 그리는 게 번거로우면, 한 함수로 통일합니다. $\cos B$ 를 $\sin$ 형태로 바꾸기.

각 변환 공식 (사인·코사인 정의에서 유도): $\cos B = \sin\left(\dfrac{\pi}{2} - B\right)$.

왜 그런가? 단위원에서 회전각 $B$ 점의 $x$ 좌표 $= \cos B$. 같은 점의 $x$ 좌표를 $\sin$ 으로 표현하려면 $\sin(\pi/2 - B)$ — 단위원에서 보각 관계.

또는: $\cos B = \sin\left(\dfrac{\pi}{2} + B\right)$ 도 성립 (코사인 그래프 = 사인 그래프 왼쪽으로 $\pi/2$ 이동).

$\sin A = \cos B$ 에 $\cos B = \sin(\pi/2 - B)$ 대입:

$\sin A = \sin\left(\dfrac{\pi}{2} - B\right)$

두 사인이 같으려면 두 가지 가능성:

경우 1: 두 각이 같음 (주기 고려) → $A = \dfrac{\pi}{2} - B + 2n\pi$ → $A + B = \dfrac{\pi}{2} + 2n\pi$.

경우 2: 두 각이 보각 ($\sin$의 대칭성: $\sin\theta = \sin(\pi - \theta)$) → $A = \pi - (\dfrac{\pi}{2} - B) + 2n\pi = \dfrac{\pi}{2} + B + 2n\pi$ → $A - B = \dfrac{\pi}{2} + 2n\pi$.

정리 1번과 똑같은 두 결론이 나옵니다. 방법은 달라도 결론은 동일. 영상이 강조한 핵심.

⑥ 핵심 정리 3 — 단위원 관점 (영상 3번 풀이)

단위원에서 회전각 $\theta$ 의 점 $y$ 좌표 = $\sin\theta$. 이 정의를 그대로 활용.

$\sin A = \cos B$ 에서 $\cos B = \sin(\pi/2 - B)$ 이므로 식은 $\sin A = \sin(\pi/2 - B)$.

단위원에서 회전각 $A$ 의 $y$ 좌표 $=$ 회전각 $\pi/2 - B$ 의 $y$ 좌표.

같은 $y$ 좌표를 갖는 두 회전각의 관계 (단위원 그림에서):

(가) 두 회전각이 같음 → $A = \pi/2 - B + 2n\pi$ → $A + B = \pi/2 + 2n\pi$.

(나) 두 회전각이 $y$ 축에 대해 대칭 (한쪽이 $\theta$, 다른쪽이 $\pi - \theta$) → $A = \pi - (\pi/2 - B) + 2n\pi$ → $A - B = \pi/2 + 2n\pi$.

또 같은 두 결론! 그래프(④), 각 변환(⑤), 단위원(⑥) — 셋 다 같은 정리. 영상 강사가 "장난하고 있는 거 보여? 다 똑같잖아" 라고 한 부분.

⑦ 영상의 진짜 메시지 — 상황별 유리한 방법 고르기

세 관점이 같은 결론이지만, 특정 문제에서 어느 관점이 빠른지는 다릅니다:

  • 방정식이 단순할 때: ⑤ 각 변환이 가장 빠름. 식 두 번 정리하면 끝.
  • 해의 위치를 시각적으로 봐야 할 때 (예: 범위 제한, 해의 개수): ④ 그래프 관점이 직관적.
  • 회전각 자체가 의미 있을 때 (호도법, 회전 문제): ⑥ 단위원 관점.

강사가 *"문제마다 어떻게 풀어야 유리한지 판단이 빨리 서야 된다"* 고 한 이유. 셋 다 익혀두고, 문제 보고 즉시 선택할 수 있어야 함.

⑧ 자주 하는 실수 4가지 (강사가 짚은 것 포함)

  1. "주기"라는 표현 잘못 쓰기: 주기는 한 함수 안에서 같은 값이 반복되는 간격. $\sin$과 $\cos$ 두 함수 사이의 평행이동은 주기가 아닙니다. (강사가 학생 답변 지적한 부분.)
  2. $2n\pi$ 빠뜨림: 주기성 고려 안 하면 모든 해를 못 찾음. 항상 $+ 2n\pi$ 붙이는 습관.
  3. 두 결론 중 하나만 적용: $A + B$ 와 $A - B$ 두 식을 모두 풀어야 모든 해 도출. 한쪽만 풀면 절반 누락.
  4. 각 변환 부호 헷갈림: $\cos B = \sin(\pi/2 - B)$ ($-B$, 보각) vs $\sin(\pi/2 + B)$ ($+B$, 평행이동). 둘 다 맞는 식이지만 부호 따라 결과가 달라지므로 일관되게 사용.

📝 수준별 문항 (기초 2 / 기본 3 / 심화 2 / 수능 대비 2)

🟢 기초

기초 1 — 정리 1번 직접 적용

문제: $\sin\dfrac{\pi}{6} = \cos B$ 를 만족하는 $B$ 의 값을 모두 구하시오 (단, $0 \le B \le 2\pi$).

📖 풀이 보기

STEP 1: $A = \pi/6$. 정리 적용: $A + B = \pi/2 + 2n\pi$ 또는 $A - B = \pi/2 + 2n\pi$.

STEP 2 (첫째): $\pi/6 + B = \pi/2 + 2n\pi$ → $B = \pi/3 + 2n\pi$.

$\pi/2 - \pi/6 = 3\pi/6 - \pi/6 = 2\pi/6 = \pi/3$.

STEP 3 (둘째): $\pi/6 - B = \pi/2 + 2n\pi$ → $B = -\pi/3 - 2n\pi$. 범위 $[0, 2\pi]$에서 $B = 5\pi/3$ ($n = -1$ 일 때 $-\pi/3 + 2\pi$).

STEP 4: 범위 $[0, 2\pi]$ 에서 답: $B = \pi/3$ 또는 $B = 5\pi/3$.

정답: $B = \dfrac{\pi}{3}, \dfrac{5\pi}{3}$.

핵심 포인트: 두 결론을 모두 풀어야 모든 해. 한쪽만 풀면 누락.

오답 주의: $B = -\pi/3$ 같은 음수 답을 그대로 두지 말고 $+2\pi$ 해서 범위 안으로.

기초 2 — 각 변환 한 줄

문제: $\cos\dfrac{2\pi}{5}$ 를 $\sin$ 함수로 표현하시오.

📖 풀이 보기

STEP 1: 각 변환 공식 $\cos\theta = \sin(\pi/2 - \theta)$ 적용.

STEP 2: $\cos\dfrac{2\pi}{5} = \sin\left(\dfrac{\pi}{2} - \dfrac{2\pi}{5}\right) = \sin\dfrac{5\pi - 4\pi}{10} = \sin\dfrac{\pi}{10}$.

통분: $\pi/2 = 5\pi/10$, $2\pi/5 = 4\pi/10$. 차이 $= \pi/10$.

정답: $\cos\dfrac{2\pi}{5} = \sin\dfrac{\pi}{10}$.

핵심 포인트: 각 변환은 통분 실수가 가장 흔함. 분모 통일 후 계산.

오답 주의: $\sin(\pi/2 + \theta)$ 도 $\cos\theta$ 와 같지만 값이 다른 식. 보각 관계 vs 평행이동 관계 헷갈리지 말 것.

🟡 기본

기본 1 — sin = cos 표준 풀이

문제: $\sin 3x = \cos 2x$ 를 만족하는 $x$ 들 중 양수인 가장 작은 값을 구하시오.

📖 풀이 보기

STEP 1: $A = 3x$, $B = 2x$. 정리 적용.

STEP 2 (정리 1): $A + B = \pi/2 + 2n\pi$ → $5x = \pi/2 + 2n\pi$ → $x = \pi/10 + 2n\pi/5$.

STEP 3 (정리 2): $A - B = \pi/2 + 2n\pi$ → $x = \pi/2 + 2n\pi$.

STEP 4: 양수인 가장 작은 값. STEP 2 의 $n=0$ → $x = \pi/10$. STEP 3 의 $n=0$ → $x = \pi/2$. 두 후보 중 작은 값.

정답: $x = \dfrac{\pi}{10}$.

핵심 포인트: 두 결론을 모두 풀고, 가장 작은 양수는 모든 후보 중 최소값.

오답 주의: 정리 1 만 보고 $\pi/10$ 으로 끝내거나, 정리 2 만 보고 $\pi/2$ 라 답하는 누락.

기본 2 — 각 변환으로 sin 통일 후 풀이

문제: $\sin x = \cos\left(\dfrac{\pi}{3} - x\right)$ 를 만족하는 $x$ 의 값을 모두 구하시오 (단, $0 \le x \le 2\pi$).

📖 풀이 보기

STEP 1: 각 변환으로 우변 변환. $\cos(\pi/3 - x) = \sin\left(\dfrac{\pi}{2} - (\pi/3 - x)\right) = \sin\left(\dfrac{\pi}{6} + x\right)$.

$\pi/2 - \pi/3 = 3\pi/6 - 2\pi/6 = \pi/6$.

STEP 2: 식 변환. $\sin x = \sin(\pi/6 + x)$.

STEP 3: 두 사인이 같으려면. 경우 1 ($x = \pi/6 + x + 2n\pi$): $0 = \pi/6 + 2n\pi$ → 해 없음 (모순). 경우 2 (보각, $x = \pi - (\pi/6 + x) + 2n\pi$): $2x = \pi - \pi/6 + 2n\pi = 5\pi/6 + 2n\pi$ → $x = 5\pi/12 + n\pi$.

STEP 4: 범위 $[0, 2\pi]$ 에서: $n = 0$ → $x = 5\pi/12$, $n = 1$ → $x = 17\pi/12$.

정답: $x = \dfrac{5\pi}{12}, \dfrac{17\pi}{12}$.

핵심 포인트: "두 사인이 같다"는 조건의 두 경우 — 같은 각, 또는 보각. 둘 다 검토.

오답 주의: 경우 1에서 $0 = \pi/6 + 2n\pi$ 라는 모순 발견 시 "해 없음" 으로 처리. 무시하고 진행 ❌.

기본 3 — 단위원에서 같은 y 좌표

문제: 단위원 위의 두 점 $P(\cos\alpha, \sin\alpha)$ 와 $Q(\cos\beta, \sin\beta)$ 가 $y$축 대칭일 때, $\alpha + \beta$ 의 값을 모든 가능한 형태로 구하시오 (단, 한 주기 내에서).

📖 풀이 보기

STEP 1: $y$축 대칭 = 같은 $y$ 좌표, 부호 반대인 $x$ 좌표. 즉 $\sin\alpha = \sin\beta$ 그리고 $\cos\alpha = -\cos\beta$.

STEP 2: 단위원에서 같은 $y$ 좌표 + 반대 $x$ 좌표 → 두 회전각이 $y$ 축에 대해 대칭. 즉 $\beta = \pi - \alpha$ (한 주기 내).

단위원에서 회전각 $\alpha$ 와 $\pi - \alpha$ 는 $y$ 축에 대해 대칭. $y$ 좌표는 $\sin\alpha = \sin(\pi - \alpha)$, $x$ 좌표는 부호 반대.

STEP 3: $\alpha + \beta = \alpha + (\pi - \alpha) = \pi$.

정답: $\alpha + \beta = \pi$.

핵심 포인트: 단위원의 $y$축 대칭 = 두 회전각의 합이 $\pi$. 사인 방정식 보각 정리의 기하적 의미.

오답 주의: $x$축 대칭과 혼동 ($x$축 대칭이면 합이 $0$ 또는 $2\pi$).

🔴 심화

심화 1 — 범위 제한 해의 개수

문제: $0 \le x \le 2\pi$ 에서 $\sin 2x = \cos x$ 를 만족하는 $x$ 의 개수를 구하시오.

📖 풀이 보기

STEP 1: $A = 2x$, $B = x$. 정리 1: $2x + x = \pi/2 + 2n\pi$ → $3x = \pi/2 + 2n\pi$ → $x = \pi/6 + 2n\pi/3$.

STEP 2: 정리 2: $2x - x = \pi/2 + 2n\pi$ → $x = \pi/2 + 2n\pi$.

STEP 3: 범위 $[0, 2\pi]$ 에서 후보 나열.

STEP 1 형식: $n=0 \to \pi/6$, $n=1 \to \pi/6 + 2\pi/3 = 5\pi/6$, $n=2 \to \pi/6 + 4\pi/3 = 9\pi/6 = 3\pi/2$, $n=3 \to \pi/6 + 6\pi/3 = \pi/6 + 2\pi$ 범위 초과.

STEP 2 형식: $n=0 \to \pi/2$, $n=1 \to \pi/2 + 2\pi$ 범위 초과.

STEP 4: 모든 해: $\pi/6, \pi/2, 5\pi/6, 3\pi/2$. 단, 중복 검사 — 모두 다름.

정답: 해의 개수 $= 4$.

핵심 포인트: 두 정리에서 나온 해를 합집합으로 모음. 중복 검사 필수.

오답 주의: $n$ 값을 너무 크거나 작게 잡아 범위 밖 해를 포함. 양 끝값 ($x = 0, 2\pi$) 도 포함되는지 정확히 확인.

출제 의도: 두 정리 + 범위 처리 + 해 카운팅 종합. 수능 단답형 단골.

심화 2 — 해의 합

문제: $0 \le x \le 2\pi$ 에서 $\sin 3x = \cos 2x$ 의 모든 해의 합을 구하시오.

📖 풀이 보기

STEP 1: 정리 1: $3x + 2x = \pi/2 + 2n\pi$ → $x = \pi/10 + 2n\pi/5$.

STEP 2: 정리 2: $3x - 2x = \pi/2 + 2n\pi$ → $x = \pi/2 + 2n\pi$.

STEP 3: 범위 $[0, 2\pi]$ 에서 STEP 1 형식. $n=0 \to \pi/10$, $n=1 \to \pi/10 + 2\pi/5 = 5\pi/10 = \pi/2$, $n=2 \to \pi/10 + 4\pi/5 = 9\pi/10$, $n=3 \to \pi/10 + 6\pi/5 = 13\pi/10$, $n=4 \to \pi/10 + 8\pi/5 = 17\pi/10$.

STEP 4: STEP 2 형식. $n=0 \to \pi/2$ (이미 STEP 1 의 해와 중복).

STEP 5: 모든 해 (중복 제거): $\pi/10, \pi/2, 9\pi/10, 13\pi/10, 17\pi/10$. 5개.

STEP 6: 합 계산. $\dfrac{\pi + 5\pi + 9\pi + 13\pi + 17\pi}{10} = \dfrac{45\pi}{10} = \dfrac{9\pi}{2}$.

정답: 해의 합 $= \dfrac{9\pi}{2}$.

핵심 포인트: 등차수열 합 공식 활용 — 5개 항의 평균이 $9\pi/10$ 이라 합 $= 5 \times 9\pi/10 = 9\pi/2$.

오답 주의: 중복 해 ($\pi/2$) 를 두 번 더하는 실수.

출제 의도: 해 추출 + 중복 처리 + 산술 종합. 학습자가 모든 해를 빠짐없이 찾는 능력 점검.

🟣 수능 대비

수능 대비 1 — 단답형 (해의 개수)

문제: $0 \le x \le \pi$ 에서 $\sin 4x = \cos 2x$ 를 만족하는 $x$ 의 개수를 구하시오.

📖 풀이 보기

STEP 1: 정리 1: $4x + 2x = \pi/2 + 2n\pi$ → $6x = \pi/2 + 2n\pi$ → $x = \pi/12 + n\pi/3$.

STEP 2: 정리 2: $4x - 2x = \pi/2 + 2n\pi$ → $2x = \pi/2 + 2n\pi$ → $x = \pi/4 + n\pi$.

STEP 3: 범위 $[0, \pi]$ 에서 STEP 1 형식. $n=0 \to \pi/12$, $n=1 \to \pi/12 + \pi/3 = 5\pi/12$, $n=2 \to \pi/12 + 2\pi/3 = 9\pi/12 = 3\pi/4$, $n=3 \to \pi/12 + \pi = 13\pi/12$ 범위 초과.

STEP 4: STEP 2 형식. $n=0 \to \pi/4$, $n=1 \to \pi/4 + \pi = 5\pi/4$ 범위 초과.

STEP 5: 모든 해: $\pi/12, \pi/4, 5\pi/12, 3\pi/4$. 중복 없음.

정답: 해의 개수 $= 4$.

핵심 포인트: 범위가 $\pi$ 로 짧으니 $n$ 값 범위 좁음. 신중히 카운팅.

오답 주의: 양 끝점 ($x=0, x=\pi$) 이 해인지 직접 검산 — $\sin 0 = 0$, $\cos 0 = 1$ → $0 \ne 1$, $x=0$ 해 아님.

출제 의도: 두 정리 적용 + 범위 카운팅. 수능 단답형 4점 전형 패턴.

수능 대비 2 — 단답형 (매개변수 결정)

문제: 양의 실수 $a$ 에 대해 $\sin ax = \cos x$ 가 $0 \le x \le \pi$ 에서 정확히 두 개의 해를 가질 때 $a$ 의 최댓값과 최솟값의 차를 구하시오.

📖 풀이 보기

STEP 1: 정리 1: $ax + x = \pi/2 + 2n\pi$ → $x(a+1) = \pi/2 + 2n\pi$ → $x = \dfrac{\pi/2 + 2n\pi}{a+1}$.

STEP 2: 정리 2: $ax - x = \pi/2 + 2n\pi$ → $x(a-1) = \pi/2 + 2n\pi$ → $x = \dfrac{\pi/2 + 2n\pi}{a-1}$ (단 $a \ne 1$).

STEP 3: 범위 $[0, \pi]$ 에서 정확히 2개 해 조건. $a$ 값에 따라 해의 개수가 달라짐 — 그래프 시각화 또는 케이스 분석.

STEP 4: 정밀 분석은 시각화 필요. 일반적으로 $a$ 가 작으면 해 적음, $a$ 가 크면 해 많음. 정확히 2개인 $a$ 범위 — 구체적 값은 그래프 분석으로 도출.

(이 문제는 의도적으로 어려운 케이스. 정확한 답을 위해 그래프 시각화 도구나 추가 분석이 필요. 학생 수준에 따라 케이스 정리 + 해의 분포 그림이 필수.)

정답 범위: 그래프 분석 결과 $a \in (1, 3)$ 범위에서 해 2개 가능 (구체적 경계값은 $\sin ax$ 와 $\cos x$ 그래프 교점 분석으로 도출).

핵심 포인트: 매개변수가 들어가면 그래프 시각화가 가장 강력한 도구.

오답 주의: 공식만으로 매개변수 문제 풀려고 하면 케이스 누락. 그래프와 함께.

출제 의도: 수능 21번/22번 유형 — 매개변수 + 해의 개수 조건. 그래프 분석 능력 종합 점검. (★★ 이 문제는 정답이 케이스 분석 의존이라 학생 배포 전 교사 검토 필수.)

✅ 학습자 가이드라인 적용 결과

본 자료는 영상에서 강사가 압축한 부분을 중수준(하위 40%) 학습자도 따라갈 수 있도록 풀어 작성:

  • "선대칭이니까 합이 두 배" → 대칭축이 $x = \pi/4$ 이므로 중점이 $\pi/4$, 양변 $\times 2$ 로 $A + B = \pi/2$. 단계별 유도 (개념 ④).
  • 각 변환 공식 → $\cos B = \sin(\pi/2 - B)$ 의 단위원 정의에서 직접 유도 (개념 ⑤).
  • "y에서 x" "한 바퀴 차이" → 단위원에서 $y$ 축 대칭 = 합 $\pi$, 한 바퀴 = $2\pi$ 차이 의 기하적 의미 (개념 ⑥).
  • "주기 vs 두 함수 관계" → 강사가 학생 답변 지적한 부분을 정확한 정의로 정리 (개념 ⑧ 실수 1).
  • 모든 풀이 STEP에 "↳ 왜?" 메모: 단계별 근거를 노란 박스로 명시.
  • 세 관점이 같은 결론임을 시각적으로 확인: ④ ⑤ ⑥ 끝마다 "다 똑같다" insight 박스.

다만 다음은 교사 검토 권장:

  • 수능 대비 2 — 매개변수 + 해의 개수 문제는 그래프 분석에 의존. 정확한 $a$ 범위 확정을 위해 시각화 필수. (확신도 ★★)
