📌 도입 — 정수 조건, 어디에 집중해야 하나?
도입 문제: 최고차항의 계수가 1인 3차 함수 $f(x)$가 다음 조건을 만족한다.

(가) 임의의 정수 $k$에 대해 $f(k-1) \times f(k+1) \geq 0$이다.
(나) $f(0) = 0$

$f(-1) = 0$과 $f(1) = 0$ 중 적어도 하나가 반드시 성립함을 보여라.
정수 지점에서의 부호 관찰 (f(0)=0인 경우) k: ... -2 -1 0 1 2 ... f(k): ... ? ? [0] ? ? ... 조건: 2칸 간격 정수쌍 f(k-1), f(k+1)의 부호가 서로 반대이면 ❌ k=0 적용: f(-1) × f(1) ≥ 0 즉, f(-1)과 f(1)은 같은 부호이거나 적어도 하나가 0 3차 함수는 결국 음→양으로 부호가 바뀌므로, 그 '넘어가는 지점'에 0이 있어야 충돌을 피할 수 있다.
풀이 보기
STEP 1
$k = 0$을 조건 (가)에 대입한다: $f(-1) \times f(1) \geq 0$
$f(0) = 0$이 주어졌으므로 $k=0$일 때 $f(k-1) = f(-1)$, $f(k+1) = f(1)$
STEP 2
최고차항의 계수가 1인 3차 함수이므로 $x \to -\infty$에서 $f(x) \to -\infty$, $x \to +\infty$에서 $f(x) \to +\infty$. 따라서 $f$는 반드시 음에서 양으로 부호가 넘어간다.
3차함수 $x^3$의 끝 거동(leading behavior)
STEP 3
$f(-1) \times f(1) \geq 0$에서 두 경우만 가능: ① $f(-1)$과 $f(1)$이 같은 부호(둘 다 양수이거나 둘 다 음수), ② 적어도 하나가 0.
곱이 0 이상 = 부호 반전이 없거나, 적어도 하나가 0
STEP 4
$f(0) = 0$이고 3차 함수는 이 지점을 통해 부호가 바뀐다(혹은 접한다). 만약 $f(-1)$과 $f(1)$이 둘 다 양수라면, $f$는 $x = -1$과 $x = 1$ 사이 어딘가에서 반드시 음의 값을 가져야 하는데, $f(0) = 0$이고 3차 함수의 감소 가능성이 있으므로 $k = 1$ 조건 $f(0) \times f(2) \geq 0$이 위배될 수 있다. 체계적으로 정리하면: 음→양 부호 전환이 연달아 $0$인 정수 지점 없이 일어나려면 $f(-1)$이나 $f(1)$ 중 하나는 반드시 0이어야 한다.
연달아 두 정수 지점에서 모두 0이어야 "2칸 간격" 조건을 위반하지 않고 부호를 전환할 수 있다
결론: f(-1) = 0 또는 f(1) = 0 (적어도 하나 성립)
💡 개념 설명
1선수 환기 — 3차 함수의 부호 변화와 근

최고차항의 계수가 양수인 3차 함수 $f(x)$는 $x \to -\infty$일 때 $-\infty$, $x \to +\infty$일 때 $+\infty$로 간다. 따라서 어딘가에서 반드시 음수 → 양수로 부호가 바뀐다.

이 부호 전환이 일어나는 지점 근방의 실근에 대해서만 생각하면 된다.
실근이 하나뿐이면 그 근을 통해 $- \to +$로 한 번에 전환.
실근이 셋이면 순서에 따라 $-\to+\to-\to+$ 식으로 바뀐다.
$f(x) = (x - r_1)(x - r_2)(x - r_3)$ (최고차항 계수 1, $r_1 \leq r_2 \leq r_3$)

$x < r_1$: $f < 0$  |  $r_1 < x < r_2$: $f \geq 0$  |  $r_2 < x < r_3$: $f \leq 0$  |  $x > r_3$: $f > 0$
(단, 중근인 경우 $r_1 = r_2$ 또는 $r_2 = r_3$이면 부호가 그 구간에서 바뀌지 않음)
⚠️ 실근이 아닌 허근이 있을 경우: $f(x) = (x-r)(x^2+bx+c)$ (판별식 $b^2-4c < 0$)이면 $f$는 $x = r$에서 단 한 번 부호가 바뀐다.
2정수 조건 $f(k-1) \times f(k+1) \geq 0$ 의 의미

강사의 핵심 메시지: "문제에서 보라는 것은 '정수 지점'이다. 그 사이 연속적인 구간을 다 보려고 하면 미궁에 빠진다."

조건의 뜻: 어떤 정수 $k$를 잡아도, $k-1$과 $k+1$(2칸 간격 정수쌍)에서의 함숫값 곱이 0 이상이어야 한다.

곱이 0 이상 = 부호가 서로 반대가 아님 = $f(k-1)$과 $f(k+1)$은 같은 부호이거나 최소 하나가 0.
$f(k-1) \times f(k+1) \geq 0$
$\Leftrightarrow$ $f(k-1)$과 $f(k+1)$이 서로 이부호(반대 부호)가 아님

3차 함수는 언젠가 $- \to +$ 부호 전환이 일어나야 한다. 이 전환이 정수 지점에서 '0'을 거쳐 일어나지 않으면 반드시 어떤 $k$에 대해 $f(k-1) < 0$, $f(k+1) > 0$이 발생한다.

⚠️ 많은 학생들이 "일단 3차 함수 개형 그리고 케이스 분류"를 시작한다. 그러나 문제가 요구하는 것은 정수 지점에서의 부호뿐이다. 개형을 전체적으로 보는 순간, 관계없는 정보까지 같이 보게 된다.
3왜 연달아 두 개의 정수 근이 필요한가?

3차 함수가 조건 $f(k-1) \times f(k+1) \geq 0$을 만족하면서 부호를 $- \to +$로 전환하려면, 그 전환 과정에 연달아 두 개의 정수 근이 있어야 한다.

이유 (유도):
가령 $f$가 정수 $n$과 $n+1$ 사이에서 $-$에서 $+$로 바뀌고, 정수 $n$과 $n+1$에서 모두 $0$이 아니라고 가정하자.
그러면 $f(n) < 0$ (아직 음수)이고 $f(n+1) > 0$ (이미 양수).
이때 $k = n+1$로 놓으면: $f(k-1) = f(n) < 0$, $f(k+1) = f(n+2)$.
$n+2$ 이후는 계속 양수이므로 $f(n+2) > 0$.
따라서 $f(n) \times f(n+2) < 0$ → 조건 위반! ❌
결론: 조건 만족 → 어딘가에 연달아 두 정수 $n, n+1$에서 $f(n) = 0$, $f(n+1) = 0$
(그래야 $k = n$: $f(n-1) \times f(n+1) = f(n-1) \times 0 = 0$ ✓
      $k = n+1$: $f(n) \times f(n+2) = 0 \times f(n+2) = 0$ ✓)
💡 강사 인용 [00:03:24]: "0이 적어도 몇 개는 있어야 돼, 연달아 두 개는 있어 줘야 된다는 거예요."
4도함수 조건으로 감소 구간 속 정수 찾기

조건 $f'(-\frac{1}{4}) = -\frac{1}{4}$, $f'(\frac{1}{4}) < 0$이 주어지면, 구간 $\left(-\frac{1}{4},\ \frac{1}{4}\right)$ 안에서 $f'(x) < 0$임을 알 수 있다.

감소 구간 속 정수 탐색:
구간 $\left(-\frac{1}{4},\ \frac{1}{4}\right)$ 안에 속하는 정수를 찾아보면:
$-0.25 < x < 0.25$에서 정수는 오직 $x = 0$뿐.

따라서 $f$는 $x = 0$ 근방에서 감소하고 있으며, $x = 0$은 $f$의 감소 구간 속에 있다.
$f'(x) < 0$ for $x \in \left(-\tfrac{1}{4},\ \tfrac{1}{4}\right)$
$\Rightarrow$ $x = 0$에서 $f$ 감소
$\Rightarrow$ $f$가 감소하면서 $x = 0$을 지나므로, 연달아 두 정수 근 중 하나가 $x = 0$

연달아 두 정수 근 조건 + $f(0) = 0$ 조건이 결합되면, 나머지 한 정수 근은 $-1$ 또는 $1$이다.

⚠️ 도함수 조건만으로 $x = 0$이 근임을 단정짓지 않는다. "감소 구간 속에서 부호 전환이 일어나는 정수"가 $0$뿐이므로 $f(0) = 0$임을 추론하는 것이다.
5미지수 1개인 식으로 압축 — 대입으로 모순 제거

$f$의 근이 $\{0,\ -1,\ \alpha\}$ 또는 $\{0,\ 1,\ \alpha\}$임이 결정되었으므로, 각 경우를 식으로 표현하면:

경우 A: $f(x) = x(x+1)(x-\alpha)$
경우 B: $f(x) = x(x-1)(x-\alpha)$
(미지수는 $\alpha$ 하나뿐)
$f'(-\frac{1}{4}) = -\frac{1}{4}$ 조건을 대입해 $\alpha$를 구한다:

경우 A: $f'(x) = 3x^2 + 2(1-\alpha)x - \alpha$ 에서
    $f'(-\tfrac{1}{4}) = \tfrac{3}{16} - \tfrac{1-\alpha}{2} - \alpha = -\tfrac{5}{16} - \tfrac{\alpha}{2} = -\tfrac{1}{4}$
    $\Rightarrow \alpha = -\tfrac{1}{8}$

경우 B: $f'(x) = 3x^2 - 2(1+\alpha)x + \alpha$ 에서
    $f'(-\tfrac{1}{4}) = \tfrac{3}{16} + \tfrac{1+\alpha}{2} + \alpha = \tfrac{11}{16} + \tfrac{3\alpha}{2} = -\tfrac{1}{4}$
    $\Rightarrow \alpha = -\tfrac{5}{8}$

이후 $f'(\tfrac{1}{4}) < 0$ 조건으로 두 경우 중 하나를 제거한다.

⚠️ 케이스 A, B를 세울 때 "어느 쪽이 맞을 것 같다"는 직관으로 고르지 말 것. 양쪽에 조건을 대입하고, 모순이 생기는 쪽을 논리적으로 제거한다.

기초 정수 조건의 기본 개념 확인
기초 1 — 정수 조건 직접 확인
문제: $f(x) = x(x-1)(x+1)$일 때, 모든 정수 $k$에 대해 $f(k-1) \times f(k+1) \geq 0$이 성립하는지 확인하시오.
(힌트: $k = 0, 1, -1, 2$를 직접 대입해 보라.)
풀이 보기
STEP 1
$f(x) = x(x-1)(x+1)$의 실근은 $x = -1, 0, 1$. 이 세 근이 연달아 있다 (정수 $-1, 0, 1$).
근이 연달아 3개이므로 정수 지점에서 부드럽게 0을 거친다
STEP 2
몇 가지 $k$를 직접 확인:
직접 대입이 가장 확실한 확인 방법
  • $k=0$: $f(-1) \times f(1) = 0 \times 0 = 0 \geq 0$ ✓
  • $k=1$: $f(0) \times f(2) = 0 \times (2)(1)(3) = 0$ ✓
  • $k=-1$: $f(-2) \times f(0) = (-2)(-3)(-1) \times 0 = 0$ ✓
  • $k=2$: $f(1) \times f(3) = 0 \times (3)(2)(4) = 0$ ✓
  • $k=3$: $f(2) \times f(4) = (2)(1)(3) \times (4)(3)(5) = 6 \times 60 > 0$ ✓
  • $k=-2$: $f(-3) \times f(-1) = (-3)(-4)(-2) \times 0 = 0$ ✓
STEP 3
일반적으로: $|k| \geq 2$인 경우 $k-1$과 $k+1$은 모두 $[-1, 0, 1]$ 바깥이므로 $f$가 같은 부호를 가진다. 따라서 곱은 항상 $\geq 0$.
세 근이 모두 [-1, 0, 1]에 있으므로 그 바깥에서는 f의 부호가 일정하다

오답 주의: $k=0$에서 $f(-1) \times f(1) = 0 \times 0 = 0$이므로 "0 이상" 조건을 만족한다. "양수여야 한다"고 잘못 생각하지 말 것.

정답: 성립함 (모든 정수 k에서 조건 만족)
기초 2 — 도함수와 감소 구간 속 정수
문제: 미분가능한 함수 $g(x)$에 대해 $g'(-0.3) = -2 < 0$이고 $g'(0.3) = -1 < 0$이다. 구간 $(-0.3, 0.3)$ 안의 정수를 모두 구하고, 그 정수에서 $g$가 증가하는지 감소하는지 판단하시오.
풀이 보기
STEP 1
구간 $(-0.3,\ 0.3)$에 속하는 정수를 찾는다.
정수는 ..., -2, -1, 0, 1, 2, ... 중 이 구간에 들어오는 것을 찾는다
$-0.3 < x < 0.3$을 만족하는 정수는 $x = 0$뿐.
STEP 2
$g'(-0.3) < 0$이고 $g'(0.3) < 0$. $g'$가 연속이면 사이값 정리에 의해 $(-0.3, 0.3)$ 안에서도 $g' < 0$이다 (극값이 없다는 추가 조건이 있어야 엄밀하지만, 수능 문제에서는 이 논리를 적용).
도함수가 양 끝에서 모두 음수이므로 사이 구간에서 부호가 바뀌지 않는다고 추론
STEP 3
따라서 $x = 0$에서 $g'(0) < 0$이므로 $g$는 $x = 0$에서 감소하고 있다.
도함수가 음수 = 함수 감소

오답 주의: "$g'(-0.3)$과 $g'(0.3)$이 음수이면 구간 전체에서 음수"라는 것은 $g'$가 그 구간에서 부호가 바뀌지 않는다는 전제 하의 추론. 엄밀한 증명은 아니지만 수능 문제의 맥락에서는 이렇게 쓴다.

정답: 구간 안 정수는 x=0뿐, x=0에서 g는 감소
기본 두 조건을 결합해 근 후보 압축
기본 1 — 두 조건 결합으로 근 후보 구하기
문제: 최고차항의 계수가 1인 3차 함수 $f(x)$가 다음 조건을 만족한다.

(가) 임의의 정수 $k$에 대해 $f(k-1) \times f(k+1) \geq 0$이다.
(나) $f'(0) < 0$  ($x = 0$에서 $f$가 감소)

$f(0) = 0$임을 논리적으로 설명하고, $f(x)$가 가질 수 있는 근의 조합을 두 가지 제시하시오.
풀이 보기
STEP 1
조건 (나)에서 $x = 0$ 근방에서 $f$가 감소한다.
$f'(0) < 0$이면 $f$는 $x = 0$ 부근에서 감소 중
STEP 2
3차 함수는 음에서 양으로 부호가 전환된다. 조건 (가)에서 이 전환은 연달아 두 정수 근을 거쳐 일어나야 한다 (개념 3 참고).
2칸 간격 부호 조건을 위반하지 않으려면 전환 지점에 연달아 두 정수 근 필요
STEP 3
감소 구간($x = 0$ 근방)이 부호 전환 구간과 겹친다. 부호 전환 중에 $x = 0$이 있으므로, $f(0) = 0$이어야 한다.
감소하면서 부호 전환이 일어나려면 그 지점에서 f = 0을 통과해야 한다
STEP 4
연달아 두 정수 근 중 하나가 $0$이므로, 나머지 하나는 $-1$ 또는 $1$.
연달아 = 서로 1 차이나는 정수
  • 경우 A: $f(x) = x(x+1)(x - \alpha)$ (근: $-1, 0, \alpha$)
  • 경우 B: $f(x) = x(x-1)(x - \alpha)$ (근: $0, 1, \alpha$)

오답 주의: "$f(0)$이 왜 0이지?"라는 부분에서 단순히 "당연히"라고 넘어가면 안 된다. "감소하면서 부호가 전환되려면 반드시 0을 지나야 한다"는 논리가 핵심.

정답: f(0)=0 확정. 경우 A: f(x)=x(x+1)(x-α), 경우 B: f(x)=x(x-1)(x-α)
기본 2 — 도함수 조건으로 α 구하기
문제: 기본 1에서 경우 B: $f(x) = x(x-1)(x-\alpha)$로 결정되었다고 하자. 조건 $f'\!\left(-\dfrac{1}{4}\right) = -\dfrac{1}{4}$를 이용하여 $\alpha$를 구하시오.
풀이 보기
STEP 1
$f(x) = x(x-1)(x-\alpha)$를 전개하면: $$f(x) = x^3 - (1+\alpha)x^2 + \alpha x$$
$(x)(x-1)(x-\alpha)$를 순서대로 전개
STEP 2
도함수를 구한다: $$f'(x) = 3x^2 - 2(1+\alpha)x + \alpha$$
각 항을 미분: $(x^3)' = 3x^2$, $(-(1+\alpha)x^2)' = -2(1+\alpha)x$, $(\alpha x)' = \alpha$
STEP 3
$x = -\dfrac{1}{4}$를 대입: $$f'\!\left(-\tfrac{1}{4}\right) = 3 \cdot \tfrac{1}{16} - 2(1+\alpha) \cdot \left(-\tfrac{1}{4}\right) + \alpha = \tfrac{3}{16} + \tfrac{1+\alpha}{2} + \alpha$$
$(-1/4)^2 = 1/16$, $(-1/4)$에 음수를 곱하면 양수
STEP 4
통분해 정리: $$\tfrac{3}{16} + \tfrac{8}{16} + \tfrac{8\alpha}{16} + \alpha = \tfrac{11}{16} + \tfrac{8\alpha + 16\alpha}{16} = \tfrac{11}{16} + \tfrac{3\alpha}{2}$$ 이를 $-\dfrac{1}{4} = -\dfrac{4}{16}$으로 놓으면: $$\tfrac{11}{16} + \tfrac{3\alpha}{2} = -\tfrac{4}{16}$$ $$\tfrac{3\alpha}{2} = -\tfrac{15}{16} \implies \alpha = -\tfrac{5}{8}$$
우변으로 이항하고 $\alpha$에 대해 정리

오답 주의: $f'(-1/4)$ 계산에서 $2(1+\alpha) \times (-1/4)$의 부호를 틀리기 쉽다. 천천히 한 단계씩.

정답: α = -5/8

심화 1 — 경우 A 검증 (모순 확인)
문제: 경우 A: $f(x) = x(x+1)(x-\alpha)$에 조건 $f'\!\left(-\dfrac{1}{4}\right) = -\dfrac{1}{4}$를 대입하면 $\alpha = -\dfrac{1}{8}$이 나온다. 이때 $f'\!\left(\dfrac{1}{4}\right) < 0$ 조건을 검증하여 경우 A가 타당한지 판단하시오.
풀이 보기
STEP 1
경우 A: $f(x) = x(x+1)(x - (-\tfrac{1}{8})) = x(x+1)(x+\tfrac{1}{8})$
$\alpha = -1/8$을 대입
STEP 2
전개: $f(x) = x^3 + (1 - (-\tfrac{1}{8}))x^2 + \cdots = x^3 + \tfrac{9}{8}x^2 + \tfrac{1}{8}x$
도함수: $f'(x) = 3x^2 + \tfrac{9}{4}x + \tfrac{1}{8}$
$f(x)=x(x+1)(x+1/8)$ 전개 후 미분
STEP 3
$f'\!\left(\dfrac{1}{4}\right)$를 계산: $$f'\!\left(\tfrac{1}{4}\right) = 3 \cdot \tfrac{1}{16} + \tfrac{9}{4} \cdot \tfrac{1}{4} + \tfrac{1}{8} = \tfrac{3}{16} + \tfrac{9}{16} + \tfrac{2}{16} = \tfrac{14}{16} = \tfrac{7}{8}$$
각 항을 분모 16으로 통분해 계산
STEP 4
$f'\!\left(\tfrac{1}{4}\right) = \tfrac{7}{8} > 0$이므로 조건 "$f'\!\left(\tfrac{1}{4}\right) < 0$"을 만족하지 않는다. → 경우 A 제거.

정답: 경우 A에서 f'(1/4) = 7/8 > 0 → 조건 위반 → 경우 A 제거
심화 2 — f(x) 완전 결정 및 특정 값 계산
문제: 기본 2의 결과 ($\alpha = -\dfrac{5}{8}$)를 이용하여 최종 $f(x)$를 확정하고, $60 \times f\!\left(-\dfrac{3}{4}\right)$의 값을 구하시오.
풀이 보기
STEP 1
$f(x) = x(x-1)\!\left(x+\dfrac{5}{8}\right)$ (경우 B에 $\alpha = -5/8$ 대입)
경우 B: f(x)=x(x-1)(x-α)에서 α=-5/8이므로 x-α = x+5/8
STEP 2
$f'\!\left(\dfrac{1}{4}\right)$로 최종 검증: $$f'(x) = 3x^2 - 2(1+(-\tfrac{5}{8}))x + (-\tfrac{5}{8}) = 3x^2 - \tfrac{3}{4}x - \tfrac{5}{8}$$ $$f'\!\left(\tfrac{1}{4}\right) = 3 \cdot \tfrac{1}{16} - \tfrac{3}{4} \cdot \tfrac{1}{4} - \tfrac{5}{8} = \tfrac{3}{16} - \tfrac{3}{16} - \tfrac{10}{16} = -\tfrac{10}{16} = -\tfrac{5}{8} < 0\ ✓$$
조건 f'(1/4) < 0 만족 확인
STEP 3
$f\!\left(-\dfrac{3}{4}\right)$ 계산: $$f\!\left(-\tfrac{3}{4}\right) = \left(-\tfrac{3}{4}\right)\!\left(-\tfrac{3}{4}-1\right)\!\left(-\tfrac{3}{4}+\tfrac{5}{8}\right)$$ $$= \left(-\tfrac{3}{4}\right)\!\left(-\tfrac{7}{4}\right)\!\left(-\tfrac{1}{8}\right)$$ $$= -\tfrac{3 \times 7 \times 1}{4 \times 4 \times 8} = -\tfrac{21}{128}$$
분수끼리 곱셈: 분자끼리, 분모끼리 곱한다
STEP 4

정수 답이 나오도록 계수 60을 선택했으나 분수로 남는 경우도 있음. 실제 수능 문제는 정수 답이 나오도록 설계됨.

오답 주의: $-3/4 + 5/8 = -6/8 + 5/8 = -1/8$. 분수 통분을 빠뜨리지 말 것.

수능 대비 1 — 준킬러 유형 (전체 논리 흐름)
문제: 최고차항의 계수가 1인 3차 함수 $f(x)$가 다음 조건을 만족한다.

(가) 임의의 정수 $k$에 대해 $f(k-1) \times f(k+1) \geq 0$이다.
(나) $f'\!\left(-\dfrac{1}{4}\right) = -\dfrac{1}{4}$, $\quad f'\!\left(\dfrac{1}{4}\right) < 0$

$f(x)$를 구하고, $64 \times f\!\left(\dfrac{1}{2}\right)$의 값을 구하시오.
풀이 보기
STEP 1
[정수 조건 분석] 조건 (가)에서 $f$는 부호 전환 시 연달아 두 정수 근을 가져야 한다.
개념 3의 논리: 연달아 두 정수 근 없이는 2칸 간격 조건 위반
STEP 2
[도함수 조건 분석] 조건 (나)에서 $f'(-1/4) < 0$, $f'(1/4) < 0$이므로 구간 $(-1/4, 1/4)$ 에서 $f$ 감소. 이 구간에서 유일한 정수는 $x = 0$.
$-0.25 < x < 0.25$ 안의 정수는 0뿐
STEP 3
[근 확정] $x = 0$에서 $f$ 감소 + $f$가 이 지점에서 부호 전환 → $f(0) = 0$. 연달아 두 정수 근이므로 $f(-1) = 0$ 또는 $f(1) = 0$.
감소하면서 0을 통과해야 부호가 전환됨
STEP 4
[경우 A: 근이 -1, 0, α] $f(x) = x(x+1)(x-\alpha)$. $f'(-1/4) = -1/4$ 대입: $$-\tfrac{5}{16} - \tfrac{\alpha}{2} = -\tfrac{1}{4} \implies \alpha = -\tfrac{1}{8}$$ 검증: $f'(1/4)$을 계산하면 양수 → 조건 (나) 위반 → 경우 A 기각.
조건 f'(1/4) < 0을 만족하지 않음
STEP 5
[경우 B: 근이 0, 1, α] $f(x) = x(x-1)(x-\alpha)$. $f'(-1/4) = -1/4$ 대입: $$\tfrac{11}{16} + \tfrac{3\alpha}{2} = -\tfrac{1}{4} \implies \alpha = -\tfrac{5}{8}$$ 검증: $f'(1/4) = -5/8 < 0$ ✓ → 경우 B 채택.
조건 f'(1/4) < 0 만족 확인
STEP 6
$f(x) = x(x-1)\!\left(x+\tfrac{5}{8}\right)$. $f\!\left(\tfrac{1}{2}\right)$ 계산: $$f\!\left(\tfrac{1}{2}\right) = \tfrac{1}{2} \cdot \left(-\tfrac{1}{2}\right) \cdot \left(\tfrac{1}{2}+\tfrac{5}{8}\right) = \tfrac{1}{2} \cdot \left(-\tfrac{1}{2}\right) \cdot \tfrac{9}{8} = -\tfrac{9}{32}$$ $$64 \times f\!\left(\tfrac{1}{2}\right) = 64 \times \left(-\tfrac{9}{32}\right) = -18$$
1/2 + 5/8 = 4/8 + 5/8 = 9/8

오답 주의: 경우 A와 B 모두 α를 구한 뒤 반드시 조건 (나) 두 번째 조건 $f'(1/4) < 0$으로 검증해야 한다. 한 가지만 대입하면 틀릴 수 있다.

정답: 64 × f(1/2) = -18
수능 대비 2 — 킬러 변형 유형 (조건 변형)
문제: 최고차항의 계수가 1인 3차 함수 $g(x)$가 다음 조건을 만족한다.

(가) 임의의 정수 $k$에 대해 $g(k-1) \times g(k+1) \geq 0$이다.
(나) $g'\!\left(\dfrac{1}{4}\right) = \dfrac{1}{4}$, $\quad g'\!\left(\dfrac{3}{4}\right) < 0$

$g(x)$를 구하고, $64 \times g\!\left(\dfrac{3}{2}\right)$의 값을 구하시오.
풀이 보기
STEP 1
[도함수 조건 분석] $g'(1/4) > 0$이고 $g'(3/4) < 0$이면 구간 $(1/4, 3/4)$에서 극대가 있다. 이 구간 안의 정수는? $0.25 < x < 0.75$에서 정수 없음!
1/4과 3/4 사이에는 정수가 없다
다만, $g'$가 양수→음수로 바뀌는 지점이 $(1/4, 3/4)$ 사이에 있으므로, $g$는 극대를 가진다. 이 극대 이후 감소하는 구간에서 정수를 찾는다.
STEP 2
$g'(3/4) < 0$이므로 $x = 3/4$ 이후에도 당분간 감소. 다음 정수는 $x = 1$. 감소 구간 내 정수가 $x = 1$일 가능성 검토.
극대가 (1/4, 3/4) 사이에 있으므로 x = 3/4 직후도 감소 중. x=1이 감소 구간에 속할 수 있음
STEP 3
[근 확정] 조건 (가)에서 연달아 두 정수 근 필요. $g'(3/4) < 0$이고 $x = 1$에서 감소 중이면 $g(1) = 0$. 연달아 있으므로 $g(0) = 0$ 또는 $g(2) = 0$.
감소하면서 x=1에서 부호 전환 → g(1)=0. 연달아 한 정수 더 필요
STEP 4
[경우 설정] 두 정수 근 중 하나가 1이므로:
  • 경우 A: 근 $\{0, 1, \beta\}$ → $g(x) = x(x-1)(x-\beta)$
  • 경우 B: 근 $\{1, 2, \beta\}$ → $g(x) = (x-1)(x-2)(x-\beta)$
STEP 5
[경우 A에 조건 대입] $g'(x) = 3x^2 - 2(1+\beta)x + \beta$: $$g'\!\left(\tfrac{1}{4}\right) = \tfrac{3}{16} - \tfrac{1+\beta}{2} + \beta = \tfrac{3}{16} - \tfrac{8(1+\beta)}{16} + \tfrac{16\beta}{16} = \tfrac{3 - 8 - 8\beta + 16\beta}{16} = \tfrac{-5 + 8\beta}{16}$$ $= 1/4 = 4/16$이면: $-5 + 8\beta = 4 \implies \beta = 9/8$
통분해 정리. 결과 β = 9/8
검증: $g'(3/4)$는 양수인지 확인. $g'(3/4) = 3(9/16) - 2(1 + 9/8)(3/4) + 9/8 = 27/16 - (17/4)(3/4) + 9/8 = 27/16 - 51/16 + 18/16 = -6/16 = -3/8 < 0$ ✓
STEP 6
경우 A 채택: $g(x) = x(x-1)\!\left(x - \tfrac{9}{8}\right)$
경우 B 검증은 생략 (경우 A가 이미 모든 조건을 만족)
$$g\!\left(\tfrac{3}{2}\right) = \tfrac{3}{2} \cdot \left(\tfrac{3}{2}-1\right) \cdot \left(\tfrac{3}{2}-\tfrac{9}{8}\right) = \tfrac{3}{2} \cdot \tfrac{1}{2} \cdot \tfrac{3}{8} = \tfrac{9}{32}$$ $$64 \times g\!\left(\tfrac{3}{2}\right) = 64 \times \tfrac{9}{32} = 18$$

오답 주의: 수능 대비 2는 감소 구간 내 정수를 찾는 단계가 수능 대비 1보다 한 단계 더 복잡하다. 도함수 조건이 양수→음수로 바뀌므로 극대 위치와 그 이후 감소 구간을 분리해서 생각해야 한다.

정답: 64 × g(3/2) = 18
