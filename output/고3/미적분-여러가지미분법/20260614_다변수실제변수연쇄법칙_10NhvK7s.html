🎯 학습 목표: 다변수 식에서 독립·종속 변수를 구별하고, 도형(원)에서는 각 $\theta$ 를 실제 변수로 삼아 연쇄법칙 $\dfrac{df}{dt}=\dfrac{df/d\theta}{dt/d\theta}$ 로 미분한다. · 대상: 중수준(하위 40%)도 따라갈 수 있는 호흡

📌 도입 — "$t$ 로 미분하지 말고, 각 $\theta$ 로 미분하라"
문제: 원 $x^2+y^2=25$ 위 제1사분면 점 $\mathrm P$ 의 $y$ 좌표를 $t$ 라 하자. $\mathrm P$ 의 $x$ 좌표를 $x(t)$ 라 할 때, $t=4$ 인 순간 $\dfrac{dx}{dt}$ 를 ($t=5\sin\theta$ 치환으로) 구하고 직접 미분과 비교하시오.
y │ ● P = (5cosθ, 5sinθ) t=5sinθ │ ╱┊ │ ╱ ┊ 5 (반지름) │ ╱ θ ┊ ─────┼────────┴──── x O 5cosθ 각 θ "하나"가 점 P의 위치를 결정 → θ가 실제 변수
풀이 보기
STEP 1
원 위의 점은 $(5\cos\theta,\,5\sin\theta)$. 즉 $x=5\cos\theta,\ t=5\sin\theta$.
좌표 $t$ 를 고집하면 $x=\sqrt{25-t^2}$ 처럼 √ 가 끼어 지저분해진다.
STEP 2
각각 $\theta$ 로 미분: $\dfrac{dx}{d\theta}=-5\sin\theta,\quad \dfrac{dt}{d\theta}=5\cos\theta.$
STEP 3
연쇄법칙: $\dfrac{dx}{dt}=\dfrac{dx/d\theta}{dt/d\theta}=\dfrac{-5\sin\theta}{5\cos\theta}=-\tan\theta.$
STEP 4
$t=5\sin\theta=4\Rightarrow\sin\theta=\tfrac45,\ \cos\theta=\tfrac35$ → $\dfrac{dx}{dt}=-\tfrac{4/5}{3/5}=-\tfrac43.$
검산
직접: $x=\sqrt{25-t^2}$, $\dfrac{dx}{dt}=\dfrac{-t}{\sqrt{25-t^2}}=\dfrac{-4}{3}$ ✓ 일치.
정답: $-\dfrac43$
💡 개념 설명
⚡ 선수 학습 1분 환기

독립 변수는 내가 자유롭게 정하는 문자, 종속 변수는 그에 따라 끌려가는 문자, 상수는 이번 문제에서 안 변하는 문자.

원 위의 점은 매개변수로 $(r\cos\theta,\ r\sin\theta)$ 로 쓸 수 있고, 삼각함수 미분 $(\sin\theta)'=\cos\theta,\ (\cos\theta)'=-\sin\theta$ 만 알면 된다.

1'다변수'는 변수가 많은 게 아니다

문자가 $x,y,a,t$ 처럼 여러 개 보여도, 무엇을 독립 변수로 볼지 정하면 나머지는 자동으로 상수가 된다.

강사: "다변수는 변수가 여러 개라는 게 아니라 그냥 문자가 여러 개 있다는 느낌이거든."
'상수 취급'이라는 말은 피하라. 모호하다. 대신 "무엇을 독립 변수로 삼겠다"를 먼저 선언하면 상수는 저절로 정해진다.
2같은 식도 '누구로 미분하느냐'에 따라 다르다

곡선 $y=x^2+a$ 위 점 $(t,\,t^2+a)$ 의 접선 $y=2tx-t^2+a$ 의 $y$ 절편은 $g=-t^2+a$.

t
$t$ 로 보면(점을 움직이면): $\dfrac{dg}{dt}=-2t$.
a
$a$ 로 보면(포물선을 위아래로): $\dfrac{dg}{da}=1$.

같은 $g$ 인데 독립 변수를 무엇으로 잡느냐에 따라 미분이 완전히 달라진다.

3함수는 $x$, 도형은 각 $\theta$ 가 '실제 변수'
강사: "함수면 실제 변수는 무조건 $x$ 좌표야. 오히려 도형이 어려운 거야 … $t$ 가 실제 변수가 아니라 각이 실제 변수여야 된다."

함수 $y=f(x)$ 는 $x$ 가 실제 변수라 깔끔. 원은 점을 좌표 두 개로 말하지만 중심각 $\theta$ 하나면 충분 → $\theta$ 가 실제 변수.

원 $x^2+y^2=r^2$ → $x=r\cos\theta,\quad y=r\sin\theta$
4핵심 도구 — "각으로 미분한 것끼리 나눈다" (유도)
1
연쇄법칙: $\dfrac{df}{d\theta}=\dfrac{df}{dt}\cdot\dfrac{dt}{d\theta}$.
2
$\dfrac{dt}{d\theta}$ 로 나누면: $\dfrac{df}{dt}=\dfrac{df/d\theta}{dt/d\theta}$.
$\dfrac{df}{dt}=\dfrac{\,df/d\theta\,}{\,dt/d\theta\,}\quad(dt/d\theta\ne0)$
도입 예시가 이거였다: $\dfrac{dx}{dt}=\dfrac{-5\sin\theta}{5\cos\theta}=-\tan\theta$. √ 가 사라지고 삼각함수 미분만 남는다.
5자주 하는 실수
① $x=\sqrt{25-t^2}$ 를 그대로 $t$ 로 밀어붙이기 → 도형이면 각 치환부터 의심. ② 사분면 보고 $\cos\theta$ 부호 정하기. ③ 함수($y=f(x)$)에는 각 치환 불필요 — 이미 $x$ 가 실제 변수.
📝 수준별 문항 (기초 2 / 기본 2 / 심화 2 / 수능 대비 2)
기초
기초 1 — 관계식 미분 $dt/d\theta$
문제: $t=5\sin\theta$ 일 때 $\dfrac{dt}{d\theta}$ 를 구하고, $t=4$(제1사분면)에서의 값을 구하시오.
풀이 보기

$\dfrac{dt}{d\theta}=5\cos\theta$. $t=4\Rightarrow\sin\theta=\tfrac45,\ \cos\theta=\tfrac35$ → $5\cdot\tfrac35=3$.

오답 주의: $\sin\theta=\tfrac45$ 에서 $\cos\theta=\tfrac45$ 로 쓰지 말 것($\cos\theta=\sqrt{1-\sin^2\theta}$).

정답: $5\cos\theta$, $t=4$ 일 때 $3$
기초 2 — 두 방법 일치 확인 ($t=3$)
문제: $x=\sqrt{25-t^2}$ 일 때 $\dfrac{dx}{dt}\big|_{t=3}$ 를 (a) 직접, (b) $t=5\sin\theta$ 치환으로 구해 비교하시오.
풀이 보기

(a) 직접: $\dfrac{-t}{\sqrt{25-t^2}}=\dfrac{-3}{4}$.

(b) 치환: $-\tan\theta$, $t=3\Rightarrow\sin\theta=\tfrac35,\cos\theta=\tfrac45$ → $-\tfrac34$.

정답: 두 방법 모두 $-\dfrac34$
기본
기본 1 — 반지름이 다른 원
문제: 원 $x^2+y^2=100$ 제1사분면 점의 $y$ 좌표 $t$, $x$ 좌표 $x(t)$. $t=6$ 에서 $\dfrac{dx}{dt}$ 를 구하시오.
풀이 보기

$x=10\cos\theta,\ t=10\sin\theta$ → $\dfrac{dx}{dt}=-\tan\theta$. $t=6\Rightarrow\sin\theta=\tfrac35,\cos\theta=\tfrac45$ → $-\tfrac34$.

검산: $\dfrac{-6}{\sqrt{100-36}}=\dfrac{-6}{8}=-\tfrac34$ ✓.

정답: $-\dfrac34$
기본 2 — 무엇으로 미분하느냐 ($y$ 절편)
문제: $y=x^2+a$ 위 점 $(t,t^2+a)$ 의 접선의 $y$ 절편 $g$ 에 대해 $\dfrac{dg}{dt}$ 와 $\dfrac{dg}{da}$ 를 구하시오.
풀이 보기

접선 $y=2tx-t^2+a$ → $x=0$ 에서 $g=-t^2+a$.

$\dfrac{dg}{dt}=-2t$ ($a$ 상수), $\quad\dfrac{dg}{da}=1$ ($t$ 상수).

오답 주의: $\dfrac{dg}{dt}$ 에서 $a$ 까지 미분하지 말 것 — 이 관점에서 $a$ 는 상수.

정답: $g=-t^2+a$, $\dfrac{dg}{dt}=-2t$, $\dfrac{dg}{da}=1$
심화
심화 1 — 부채꼴 넓이의 변화율
문제: 반지름 $5$ 원에서 중심각 $\theta$ 부채꼴 넓이 $S$, $t=5\sin\theta$. $t=4$ 에서 $\dfrac{dS}{dt}$ 를 구하시오.
풀이 보기

$S=\tfrac12 r^2\theta=\tfrac{25}{2}\theta$ → $\dfrac{dS}{d\theta}=\tfrac{25}{2}$, $\dfrac{dt}{d\theta}=5\cos\theta$.

$\dfrac{dS}{dt}=\dfrac{25/2}{5\cos\theta}=\dfrac{5}{2\cos\theta}$. $t=4\Rightarrow\cos\theta=\tfrac35$ → $\dfrac{5}{2\cdot3/5}=\dfrac{25}{6}$.

검산: $\dfrac{25}{2}\cdot\dfrac{1}{\sqrt{25-t^2}}=\dfrac{25}{2}\cdot\tfrac13=\tfrac{25}{6}$ ✓.

정답: $\dfrac{25}{6}$
심화 2 — 각 치환이 '필요 없는' 경우
문제: $y=x^3-x$ 에서 $\dfrac{dy}{dx}\big|_{x=2}$ 를 구하고, 각 $\theta$ 치환이 적절한지 판단하시오.
풀이 보기

함수는 이미 $x$ 가 실제 변수 → 그냥 $\dfrac{dy}{dx}=3x^2-1$. $x=2$ → $11$.

각 치환은 불필요(좌표가 얽혀 √ 가 생기는 도형 상황이 아님).

정답: $11$, 각 치환 부적절
수능 대비
수능 대비 1 — 단답형 (미분계수)
문제: 원 $x^2+y^2=169$ 제1사분면 점의 $y$ 좌표 $t$, $x$ 좌표 $x(t)$. $\dfrac{dx}{dt}\big|_{t=5}=-\dfrac{q}{p}$ ($p,q$ 서로소 자연수)일 때 $p+q$ 를 구하시오.
풀이 보기

$x=13\cos\theta,\ t=13\sin\theta$ → $-\tan\theta$. $t=5\Rightarrow\sin\theta=\tfrac{5}{13},\cos\theta=\tfrac{12}{13}$(5-12-13) → $-\tfrac{5}{12}$.

$p=12,q=5$, 서로소 → $p+q=17$.

정답: $17$
수능 대비 2 — 단답형 (넓이 변화율)
문제: 반지름 $5$ 부채꼴 넓이 $S=\tfrac{25}{2}\theta$, $t=5\sin\theta$. $t=3$ 에서 $\dfrac{dS}{dt}=\dfrac{q}{p}$ ($p,q$ 서로소)일 때 $p+q$ 를 구하시오.
풀이 보기

$\dfrac{dS}{dt}=\dfrac{5}{2\cos\theta}$. $t=3\Rightarrow\cos\theta=\tfrac45$ → $\dfrac{5}{2\cdot4/5}=\dfrac{25}{8}$.

$p=8,q=25$, 서로소 → $p+q=33$.

정답: $33$
✅ 검증 / 교사 검토 권장

영상의 핵심(독립·종속 변수 구별, 도형은 각 $\theta$ 가 실제 변수, 연쇄법칙 $\dfrac{df}{dt}=\dfrac{df/d\theta}{dt/d\theta}$)을 반영하고, 영상 수치($t=5\sin\theta$, $t=4\to(\sin,\cos)=(\tfrac45,\tfrac35)$)를 활용했습니다. 각 치환 결과는 모두 직접 미분으로 검산했습니다.
