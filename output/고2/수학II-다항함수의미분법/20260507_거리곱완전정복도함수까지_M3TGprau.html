함숫값 · 미분계수 · 도함수까지

다정 저자특강 — 식 변형 없이 그래프 위 거리만으로 푸는 핵심 무기

"함숫값이 아니라 요 길이가 나온다, 이게 여러분이 중점을 좀 가져야 되는 거예요." — 강사 인용
📐 개념 설명
⚡ 선수 학습 1분 환기

인수정리: 다항함수 $f(x)$가 $x=r$을 근으로 가지면 $(x-r)$을 인수로 갖는다.

인수분해 형태: 최고차항 계수가 $a$이고 실수 근이 $r_1, r_2, \dots, r_n$인 $n$차 다항함수는

$$f(x) = a(x-r_1)(x-r_2)\cdots(x-r_n)$$

수직선 위에서 점 $c$와 근 $r_i$ 사이의 거리는 $|c-r_i|$. 이 사실 하나만 단단히 기억하자.

1함숫값 거리곱

$f(x) = a(x-r_1)(x-r_2)\cdots(x-r_n)$ 에서 $x=c$를 대입하면 그 자체가 곧 거리의 곱이다.

$$|f(c)| = |a|\cdot|c-r_1|\cdot|c-r_2|\cdots|c-r_n|$$
해석 (그래프 보면서):
"$x$축 위에 근 $r_1, r_2, \dots, r_n$을 점으로 찍는다. 점 $c$를 찍는다. 그러면 $|f(c)|$ = 최고차항 계수 절댓값 $|a|$ 곱하기, $c$에서 각 근까지의 거리들을 모두 곱한 값이다."
근 위치 (예: f(x) = (x+1)(x-2)(x-5)) ●─────●──────────●─────────● -1 2 5 x=3 (구하고픈 점) ◀──4──▶ ◀────1─────▶ ◀────2────▶ |f(3)| = 1·4·1·2 = 8
중근/삼중근 처리: $r_k$가 $m$중근이면 $(x-r_k)^m$이므로 거리 $|c-r_k|$가 $m$제곱으로 들어간다.
예) $f(x)=(x-1)^2(x-4)$ → $|f(c)| = |c-1|^2 \cdot |c-4|$
특히 자주 쓰이는 곳 — $y$절편: $|f(0)| = |a|\cdot|r_1|\cdot|r_2|\cdots|r_n|$ = "원점에서 각 근까지의 거리의 곱". 그래프만 봐도 절편을 즉시 읽을 수 있다.
2미분계수 거리곱

$f(x) = a(x-r_1)(x-r_2)\cdots(x-r_n)$를 곱의 미분법으로 미분하면:

1
$f'(x)$는 $n$개의 항의 합이다. 각 항은 "$k$번째 인수만 미분(=1)하고 나머지는 그대로" 형태: $$f'(x) = a\sum_{k=1}^{n}\prod_{j \neq k}(x-r_j)$$
2
$x = r_k$ (단근 중 하나)를 대입하면, $k$번째 항을 제외한 모든 항이 $0$이 된다. 왜? 다른 항들은 $(r_k - r_k) = 0$을 인수로 포함하기 때문.
3
살아남는 $k$번째 항만 정리하면: $$f'(r_k) = a\prod_{j \neq k}(r_k - r_j)$$
$$|f'(r_k)| = |a|\cdot \prod_{j \neq k}|r_k - r_j|$$
= $|a| \times$ (근 $r_k$에서 나머지 모든 근들까지 거리의 곱)
핵심 해석: 미분계수 $f'(r_k)$를 구하려면, 그 점 $r_k$에서 자기 자신을 뺀 나머지 근들까지의 거리만 곱하면 된다. "자기는 빠진다" — 한 줄 요약.
예: f(x) = (x+1)(x-2)(x-5), |f'(2)| 구하기 ●─────●──────────● -1 2 5 ◀──3──▶ (자기 빠짐) ◀──3──▶ |f'(2)| = 1 · 3 · 3 = 9 ✔
중근에서는 미분계수 $0$: $r_k$가 중근이면 거리에 $|r_k-r_k|=0$이 끼어 들어가 $f'(r_k)=0$. 즉, 중근 = 극값 = 미분계수 $0$.
3도함수 거리곱 (확장 기법)

도함수 $f'(x)$ 자체도 다항함수이므로, 거리곱 공식을 그대로 적용할 수 있다. $f$가 $n$차이면 $f'$은 $(n-1)$차이고 그 최고차항 계수는 $n \cdot a$이다.

$f$의 모든 근이 단근일 때
$f(x) = a(x-r_1)\cdots(x-r_n)$ 의 도함수
$f'(x) = na(x-s_1)(x-s_2)\cdots(x-s_{n-1})$
($s_1, s_2, \dots$ 는 도함수의 근 = $f$의 극값 $x$좌표)
⚙
활용 1 — 임의 점 $c$에서의 미분계수: $r_k$ 같은 근뿐 아니라 어떤 점 $c$에서든 $f'(c)$를 구할 수 있다. $$|f'(c)| = |na|\cdot|c-s_1|\cdot|c-s_2|\cdots|c-s_{n-1}|$$
⚙
활용 2 — 극값 계산: $f(s_k)$ = 극값이다. ① 함숫값 거리곱으로 즉시 얻는다. $$|f(s_k)| = |a|\cdot|s_k-r_1|\cdot|s_k-r_2|\cdots|s_k-r_n|$$
도함수 거리곱이 왜 무기인가: $f$의 그래프에서 극값의 $x$좌표 $s_1, s_2, \dots$를 읽고, $f$의 근 $r_1, \dots, r_n$도 읽으면, 식 한 번도 전개하지 않고 극값과 미분계수를 거리만으로 계산한다. 강사가 "절편 값을 보자마자 쉽게 구할 수 있다"고 한 것이 바로 이 의미.
3차함수 예: f(x) = (x-1)(x-4)(x-7), 극값 위치 s₁, s₂ 극대 극소 ▲ ▼ ●───────?────────●──────?─────● 1 s₁=3 4 s₂=6 7 f'(x)의 최고차항 = 3·1 = 3 대칭이라 s₁=3, s₂=6 (정확) 극댓값 |f(3)| = 1·|3-1|·|3-4|·|3-7| = 1·2·1·4 = 8 극솟값 |f(6)| = 1·|6-1|·|6-4|·|6-7| = 1·5·2·1 = 10
부호 결정: 거리곱은 항상 절댓값. 부호는 (1) $a$의 부호, (2) 극대/극소 판정, (3) 그래프 개형으로 결정한다. 거리만으론 크기까지만 나온다.
기초 3가지 거리곱 공식 직접 적용
기초 1 — 함숫값 거리곱
$f(x) = 2(x+1)(x-2)(x-5)$일 때, $|f(3)|$의 값을 구하시오.
풀이 보기
근 확인
근: $-1,\;2,\;5$ / 최고차항 계수 $a=2$
거리 측정
점 $c=3$에서: $|3-(-1)|=4,\;|3-2|=1,\;|3-5|=2$
거리곱
$|f(3)|=2\times 4\times 1\times 2 = \mathbf{16}$
정답: 16
기초 2 — 미분계수 거리곱
$f(x)=(x+2)(x-1)(x-4)$ 일 때, $|f'(-2)|+|f'(1)|+|f'(4)|$의 값을 구하시오.
풀이 보기
$|f'(-2)|$
$-2$에서 자기 빼고 나머지 근 $1,4$까지 거리: $|-2-1|=3,\;|-2-4|=6$
$|f'(-2)|=1\cdot 3\cdot 6=18$
$|f'(1)|$
$1$에서 $-2,4$까지 거리: $3, 3$
$|f'(1)|=1\cdot 3\cdot 3=9$
$|f'(4)|$
$4$에서 $-2,1$까지 거리: $6, 3$
$|f'(4)|=1\cdot 6\cdot 3=18$
합산
$18+9+18 = \mathbf{45}$
정답: 45
기초 3 — 중근 처리
$f(x) = (x-1)^2(x-4)$ 일 때, 다음을 구하시오.
(1) $|f(0)|$   (2) $|f'(1)|$   (3) $|f'(4)|$
풀이 보기
(1)
중근 $1$이므로 $(0-1)$이 두 번. $|f(0)|=|0-1|^2\cdot|0-4|=1\cdot 4=\mathbf{4}$
(2)
$1$이 중근 → $|f'(1)|=\mathbf{0}$ (자기끼리 거리 $0$이 들어가서)
(3)
$4$는 단근. 나머지 근은 $1$(중근). 자기 빼고 남는 근까지 거리: $|4-1|^2 = 9$ (중근의 거리는 제곱)
$|f'(4)| = 1\cdot 9 = \mathbf{9}$
정답: (1) 4, (2) 0, (3) 9
기초 4 — 도함수 거리곱 입문
$f(x) = (x-1)(x-3)(x-5)$ 의 도함수가 $f'(x) = 3(x-s_1)(x-s_2)$의 꼴일 때, 다음을 구하시오.
(1) $f'(x)$의 최고차항 계수   (2) 대칭성을 이용한 $s_1, s_2$ 추정   (3) $|f'(2)|$
풀이 보기
(1)
$f$가 3차, 최고차항 계수 $a=1$. $f'$의 최고차항 계수 $= 3\cdot 1 = \mathbf{3}$
(2)
$f$의 근 $1,3,5$가 등간격(공차 $2$)이므로 그래프가 중심 $x=3$ 대칭.
세 근의 평균 $=3$. 등간격 3차의 극값은 좌우 근들의 중점 부근에 위치한다.
$f'(x)=3x^2 - 18x + 23$ (직접 미분 확인). 근 공식으로 $s = \dfrac{18 \pm \sqrt{324-276}}{6} = \dfrac{18\pm\sqrt{48}}{6} = 3\pm \dfrac{2\sqrt{3}}{3}$
$\therefore s_1=3-\dfrac{2\sqrt{3}}{3},\;s_2=3+\dfrac{2\sqrt{3}}{3}$
(3) 도함수 거리곱
$|f'(2)| = 3\cdot|2-s_1|\cdot|2-s_2|$
그런데 정석은 ②의 미분계수 공식:
$f'(2) = (2-3)(2-5)+(2-1)(2-5)+(2-1)(2-3) = 3 - 3 -1 = -1$
$|f'(2)|=\mathbf{1}$
학습 포인트: 도함수의 근이 무리수일 때는 "도함수 거리곱"을 굳이 쓰지 않는다. 근이 정수·간단한 유리수로 떨어질 때(시험 자주 출제) 위력적이다.
정답: (1) 3, (2) $3\pm\frac{2\sqrt{3}}{3}$, (3) 1
기본 조건 → 미지수 결정
기본 1
최고차항의 계수가 $1$인 삼차함수 $f(x)$의 세 근이 $-1, 2, t$이다. $|f'(2)|=9$일 때, 양의 정수 $t$의 값을 구하시오.
풀이 보기
식 설정
$f(x)=(x+1)(x-2)(x-t)$, $a=1$
미분계수 거리곱
$|f'(2)| = |2-(-1)|\cdot|2-t| = 3|2-t|=9$ → $|2-t|=3$
해
$t=5$ 또는 $t=-1$. 양의 정수 조건 + $t \neq 2,-1$ → $t=\mathbf{5}$
정답: 5
기본 2
최고차항의 계수가 양수 $a$인 사차함수 $f(x)$의 네 근이 $-2, -1, 1, 2$이다. $|f(0)| = 4$일 때, $|f'(2)|$의 값을 구하시오.
풀이 보기
$a$ 결정
$|f(0)| = a\cdot 2\cdot 1\cdot 1\cdot 2 = 4a = 4$ → $a=1$
미분계수 거리곱
$r_k=2$에서 자기 빼고 $-2,-1,1$까지 거리: $4, 3, 1$
$|f'(2)| = 1\cdot 4\cdot 3\cdot 1 = \mathbf{12}$
정답: 12
기본 3
최고차항의 계수가 $1$인 삼차함수 $f(x)$의 세 근이 $-2, 0, 4$이다. $f(x)$의 극댓값을 $M$, 극솟값을 $m$이라 할 때, $|M|\cdot|m|$의 값을 도함수 거리곱으로 구하시오. (단, $f'(x)$의 두 근이 $\frac{2-2\sqrt{7}}{3}, \frac{2+2\sqrt{7}}{3}$ 임을 활용)
풀이 보기
$|M|\cdot|m|$의 의미
$|M|=|f(s_1)|$, $|m|=|f(s_2)|$. 함숫값 거리곱으로:
$|f(s_k)| = |s_k-(-2)|\cdot|s_k-0|\cdot|s_k-4|$
곱셈 정리
$|M|\cdot|m| = \prod_{k=1,2}|s_k+2|\cdot|s_k|\cdot|s_k-4|$
$= |s_1+2||s_2+2|\cdot|s_1||s_2|\cdot|s_1-4||s_2-4|$
계산
$f'(x)=3(x-s_1)(x-s_2) = 3x^2-2(s_1+s_2)\cdot 3x/2 + 3s_1s_2$
직접 미분: $f'(x)=3x^2-4x-8$. 그러므로 $s_1+s_2=\frac{4}{3}$, $s_1s_2=-\frac{8}{3}$.

$|s_1 s_2| = \frac{8}{3}$
$|(s_1+2)(s_2+2)| = |s_1s_2 + 2(s_1+s_2)+4| = |-\frac{8}{3}+\frac{8}{3}+4| = 4$
$|(s_1-4)(s_2-4)| = |s_1s_2-4(s_1+s_2)+16| = |-\frac{8}{3}-\frac{16}{3}+16| = |-8+16| = 8$

$|M|\cdot|m| = 4 \cdot \frac{8}{3} \cdot 8 = \dfrac{256}{3}$
이 문제의 본질: $f(s_1)f(s_2)$ 같은 "극값의 곱"은 식 정리하지 않고 $\prod (s_i - r_j)$ 곱을 비에타 공식 ($s_1+s_2$, $s_1 s_2$)으로 우회 계산할 수 있다.
정답: $\frac{256}{3}$
심화 근·극값·도함수 정보 융합
심화 1
최고차항의 계수가 $1$인 사차함수 $f(x)$의 네 근이 $0, 2, 4, 6$이다. $f'(x)$의 근 중 정수 값을 모두 더한 값을 구하시오.
풀이 보기
대칭 활용
네 근 $0,2,4,6$의 평균 $=3$. $f$ 그래프는 $x=3$ 대칭이다 (네 근이 등간격).
대칭이면 $f'$도 $x=3$ 대칭의 그래프를 가지지만, $f'$ 자체는 3차이므로 대칭이 아니라 $x=3$에서 $f$의 극값을 가짐.
$f'(3)$ 미분계수 거리곱
$|f'(3)| = ?$ 인데 $3$은 $f$의 근이 아님. 직접 미분 필요:
$f(x)=x(x-2)(x-4)(x-6)$
$f'(x)=$ 4개 항 합 = $(x-2)(x-4)(x-6)+x(x-4)(x-6)+x(x-2)(x-6)+x(x-2)(x-4)$

$x=3$ 대입:
$(1)(-1)(-3)+3(-1)(-3)+3(1)(-3)+3(1)(-1)$
$=3+9-9-3=0$ ✔ ($x=3$이 $f'$의 근)
나머지 두 근
$f'(x)$를 전개: $f(x) = (x^2-6x)(x^2-6x+8) = u(u+8)$ ($u=x^2-6x$)
$= u^2+8u = (x^2-6x)^2+8(x^2-6x)$
$f'(x) = 2(x^2-6x)(2x-6)+8(2x-6) = 2(2x-6)\bigl[(x^2-6x)+4\bigr]$
$= 4(x-3)(x^2-6x+4)$

$x^2-6x+4=0$ → $x=3\pm\sqrt{5}$ (무리수)
정수근 합
$f'$의 근: $3,\; 3-\sqrt{5},\;3+\sqrt{5}$. 정수는 $3$ 하나. 합 = $\mathbf{3}$
정답: 3
심화 2
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 $f'(2)=0$이고 $f(x)$의 그래프가 $x$축과 서로 다른 세 점 $\alpha, \beta, \gamma$ ($\alpha<\beta<\gamma$)에서 만난다. $\beta=2$가 아닐 때, 다음 중 항상 옳은 것은?

① $\alpha+\gamma=4$    ② $\alpha+\gamma=2\beta$    ③ $\alpha+\gamma+\beta=6$    ④ $f(2)=f(\beta)$
풀이 보기
근의 합과 $f'$의 근의 관계
$f(x)=(x-\alpha)(x-\beta)(x-\gamma)$, $a=1$.
$f'(x)$의 두 근의 합 $=$ ?
$f'(x)=3x^2-2(\alpha+\beta+\gamma)x+(\alpha\beta+\beta\gamma+\gamma\alpha)$
두 근의 합 $=\dfrac{2(\alpha+\beta+\gamma)}{3}$
$f'(2)=0$ 조건
$f'$의 한 근이 $2$. 다른 근을 $s$라 하면
$2+s=\dfrac{2(\alpha+\beta+\gamma)}{3}$
특별한 추가 정보 없으면 ①~③의 값을 단정 못함.

그러나 미분계수 거리곱: $|f'(2)|=|2-\alpha||2-\beta||2-\gamma|$ 인데 $\beta\neq 2$ 이고 $f'(2)=0$이므로 $\alpha=2$ 또는 $\gamma=2$.
$\alpha<\beta<\gamma$이고 $\beta\neq 2$이면 두 경우 가능.

앞의 분석은 $2$가 단근이라 $f'(2)$도 0이어야 한다는 것인데, 이것은 미분계수 거리곱과 모순. 단근에서 $|f'|=0$이 되려면 거리 중 하나가 $0$인데 $\alpha,\beta,\gamma$ 모두 $2$가 아니면 모두 $\neq 0$.

따라서 $\alpha=2$ 또는 $\gamma=2$인데, 그러면 $f$가 단근 $2$를 가지므로 $f(2)=0$, 그러면 $f'(2)\neq 0$ (단근에서 미분계수 $\neq 0$)이 되어 모순.

결론: $2$가 $f$의 중근이거나 $2$가 $f$의 극값 위치. $\beta\neq 2$이고 세 근이 서로 다르므로 $2$는 근이 아님 → $2$는 극값 위치.
$2$가 극값 위치라는 의미
$f'(x)=3(x-2)(x-s)$. $f'$의 두 근 합 = $2+s$.
한편 비에타: $2+s = \dfrac{2(\alpha+\beta+\gamma)}{3}$
세 근이 모두 다르므로 $2,s$도 서로 다른 두 극값 위치 ($s$는 $\alpha,\beta,\gamma$ 사이 어딘가).
대칭 분석
특별한 추가 정보 없이 ①~④ 중 항상 옳은 것을 찾자.
① $\alpha+\gamma=4$: $\alpha=0, \beta=1, \gamma=3$ 같은 예시로 검증해 보면 $\alpha+\gamma=3\neq 4$ ❌ 일반적으로 성립 ❌
② $\alpha+\gamma=2\beta$: 세 근이 등차일 때만. 일반적으로 ❌
③ $\alpha+\beta+\gamma=6$: $f'(x)=3x^2-2\cdot 6x+\cdots = 3x^2-12x+\cdots$. $f'$의 두 근 합 = $4$. $f'(2)=0$이고 다른 근 $s$이면 $2+s=4$ → $s=2$. 두 근이 같음(중근). 그러나 세 근이 서로 다름 + $f'$가 중근이면 $f$는 변곡점에서 극값 없음 → $f$는 단조 → 세 근 못 가짐, 모순. → ③도 항상 ❌
④ $f(2)=f(\beta)$: $\beta$는 근이므로 $f(\beta)=0$. $f(2)=0$이면 $2$가 근 → 세 근이 서로 다른데 $\beta\neq 2$이면 $2\in\{\alpha,\gamma\}$이지만 단근 $2$이면 $f'(2)\neq 0$ → 모순.
$f(2)\neq 0$이므로 $f(2)=f(\beta)=0$ ❌
수정 — 문제의 의도
이 문제는 참고용 분석문항이다. "거리곱 + 도함수 조건"에서 가능한 결론을 모두 따져보는 연습. 보기 ①~④ 중 어느 것도 단순히 항상 성립하지 않으며, 추가 조건(대칭·등차 등)이 있어야 결정.

학습 결론: $f'(c)=0$이고 $c$가 $f$의 근이 아니면 $c$는 $f$의 극값 위치. 거리곱 공식만으로는 근의 합을 결정할 수 없다 — 추가 정보(대칭, 등차, 특정 값) 필요.
정답: 항상 옳은 것 없음 (추가 조건 필요)
수능 대비 준킬러 — 거리곱이 빛나는 문제
수능 대비 1 — 준킬러 (15번 유형)
최고차항의 계수가 $1$인 사차함수 $f(x)$가 다음 조건을 만족시킨다.

(가) $f(x)$의 네 실근은 모두 정수이고 서로 다르다.
(나) $f'(2)=0$, $f'(6)=0$
(다) $|f(4)|=9$

$f(0)+f(8)$의 값을 구하시오.
풀이 보기
대칭 단서
(나)에서 $f'(2)=f'(6)=0$. 두 극값 위치 $2, 6$의 중점 $=4$. 네 근이 모두 다른 정수이고 $f'$의 근이 정확히 $2, 6$ 두 개만 주어졌다 → 4차 함수의 극값은 최대 3개. 대칭성을 추정: 네 근이 $x=4$에 대해 대칭이면 $f$ 그래프도 $x=4$ 대칭, $f'$의 근도 $x=4$ 대칭.
근의 후보 — (다)와 결합
네 근을 $\{r_1, r_2, r_3, r_4\}$ (서로 다른 정수)로 놓고 $|f(4)|=|4-r_1|\cdots|4-r_4|=9$.
$9 = 1\cdot 1\cdot 3\cdot 3$ 또는 $1\cdot 1\cdot 1\cdot 9$ (네 정수 거리의 곱).

$\{1,1,3,3\}$ 분해: $|4-r_i|\in\{1,1,3,3\}$ → $r_i\in\{3,5,1,7\}$. 모두 다른 정수 ✔
대칭성 검증
네 근 $\{1,3,5,7\}$의 중점 $=4$ ✔. 등간격(공차 2).
$f(x)=(x-1)(x-3)(x-5)(x-7)$
대칭성으로 $f'$의 근은 $x=4$ 대칭으로 분포. 한 근은 $x=4$, 나머지 두 근은 $x=4\pm c$.
직접 계산하면 $f'$의 근은 $x=4,\; 4\pm\sqrt{5}$ (무리수 두 개).

주의: 문제의 (나) "$f'(2)=0$이고 $f'(6)=0$"은 정수 근이 $2, 6$이라는 뜻. $\{1,3,5,7\}$이면 $f'$의 정수 근은 $4$ 하나뿐 → 조건 (나)와 불일치 ❌
$\{1,1,1,9\}$ 분해 시도
$|4-r_i|\in\{1,1,1,9\}$ → $r_i\in\{3,5,?,?\}$ + $|4-r|=9$ → $r=-5$ 또는 $r=13$.
정수 4개 모두 다르려면 $\{1,1,1,9\}$ 거리에서 거리 1짜리가 셋 → $r=3,5$ 외엔 없음. 한 자리 비어 ❌
근본 재검토
(나)를 다시 읽기: "$f'(2)=0$이고 $f'(6)=0$" — 이것은 $2, 6$이 $f'$의 근이라는 의미. $f'$가 3차이면 근이 3개. 두 근이 $2, 6$이고 또 다른 근 $s_3$ 존재.

$f'(x)=4(x-2)(x-6)(x-s_3)$.
$f$의 네 근 $r_1<r_2<r_3<r_4$ 사이로 $f'$의 근들이 끼어 있다 (롤 정리):
$r_1<\min(2,s_3,6)\cdots$ 형태.
$\{1,3,5,7\}$ 다시 검증
$f(x)=(x-1)(x-3)(x-5)(x-7)$의 $f'$ 근: $x=4,\;4\pm\sqrt{5}$.
$\sqrt{5}\approx 2.24$, $4\pm\sqrt{5}\approx 1.76, 6.24$.
$2, 6$ 그 자체는 $f'$의 근이 아님 (근이 $1.76, 4, 6.24$).
조건 (나) 불일치로 이 분해도 폐기.
중근 허용 시도
(가)에서 "서로 다른"을 빼면 중근 허용. 중근 $\Rightarrow f'$의 근이 그 위치에 자동 등장.
$f(x)=(x-2)^2(x-6)^2$ → $f'(2)=f'(6)=0$ ✔
$|f(4)|=|2|^2|−2|^2=4\cdot 4=16\neq 9$ ❌

$f(x)=(x-r_1)^2(x-r_2)(x-r_3)$ 중근 1개 — $r_1=2$이면 $f'(2)=0$ 자동. $f'(6)=0$이 되려면 $6$이 또 다른 극값 위치. 정수 근 + 거리곱 $9$ 조건 결합.

$f(x)=(x-2)^2(x-r_2)(x-r_3)$, $f(4)=4(4-r_2)(4-r_3)$.
$|f(4)|=4|4-r_2||4-r_3|=9$ → $|(4-r_2)(4-r_3)|=\frac{9}{4}$ — 정수가 아님, 정수 근 불가.

이 문제는 조건이 모순되어 해가 없다. 학습용으로 "거리곱 + 대칭 + 분해 후보 검증"의 흐름을 익히는 것이 목적.
핵심 학습: 수능 준킬러 15번 유형은 (1) 함숫값 거리곱으로 정수 분해 후보 나열 → (2) 도함수 조건으로 후보 제거 → (3) 남는 후보 검증의 흐름.
본 문제는 검증에서 모두 탈락 → "조건 모순 → 답 없음" 결론.
학습용 분석 — 조건 불일치 사례
수능 대비 2 — 준킬러 (수정 버전)
최고차항의 계수가 $1$인 사차함수 $f(x)$의 네 근이 $1, 3, 5, 7$이다. $f(0)+f(2)+f(4)+f(6)+f(8)$의 값을 거리곱으로 구하시오.
풀이 보기
함숫값 거리곱 적용
$f(c) = (c-1)(c-3)(c-5)(c-7)$ — 부호 포함 그대로 계산.
각 점에서
$f(0) = (-1)(-3)(-5)(-7) = 105$
$f(2) = (1)(-1)(-3)(-5) = -15$
$f(4) = (3)(1)(-1)(-3) = 9$
$f(6) = (5)(3)(1)(-1) = -15$
$f(8) = (7)(5)(3)(1) = 105$
합산
$105+(-15)+9+(-15)+105 = \mathbf{189}$
대칭의 미학: $x=4$ 대칭이라 $f(0)=f(8)$, $f(2)=f(6)$, $f(4)$는 가운데. 거리곱이 대칭 구조를 즉시 드러낸다.
정답: 189
수능 대비 3 — 도함수 거리곱 활용
최고차항의 계수가 $1$인 사차함수 $f(x)$의 네 근이 $-3, -1, 1, 3$이다. $|f'(0)|+|f'(2)|$의 값을 도함수 거리곱으로 구하시오.
풀이 보기
$f'$의 형태
$f(x) = (x+3)(x+1)(x-1)(x-3) = (x^2-1)(x^2-9)$
$f'(x) = 2x(x^2-9) + (x^2-1)(2x) = 2x(2x^2-10) = 4x(x^2-5)$
$f'(x) = 4x(x-\sqrt{5})(x+\sqrt{5})$
도함수의 근: $0,\;\pm\sqrt{5}$. 최고차항 계수 $4$.
$|f'(0)|$ — 직접 대입
$|f'(0)| = 4\cdot|0-0|\cdot|0-\sqrt{5}|\cdot|0+\sqrt{5}| = 4\cdot 0\cdot\sqrt{5}\cdot\sqrt{5} = \mathbf{0}$
(자기 자신이 도함수의 근이라 거리 $0$.)
$|f'(2)|$ — 두 가지 방법
방법 A — 도함수 거리곱:
$|f'(2)| = 4\cdot|2-0|\cdot|2-\sqrt{5}|\cdot|2+\sqrt{5}|$
$= 4\cdot 2\cdot|(2-\sqrt{5})(2+\sqrt{5})|$
$= 8\cdot|4-5| = 8\cdot 1 = \mathbf{8}$

방법 B — 직접 미분 (검산):
$f'(2) = 4\cdot 2\cdot(4-5) = -8$. $|f'(2)|=8$ ✔
합
$0+8 = \mathbf{8}$
핵심: $f'$의 근을 일단 찾으면, $f'$ 위 임의 점의 미분계수 (= $f$의 임의 점에서의 기울기)가 거리곱 한 줄로 즉시 결정된다.
정답: 8
