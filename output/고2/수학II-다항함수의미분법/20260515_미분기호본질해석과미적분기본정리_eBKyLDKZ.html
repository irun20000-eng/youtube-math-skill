📐 개념 — 미분 기호의 본질과 적분의 정체
⚡ 선수 학습 1분 환기
  • 평균변화율: $\dfrac{\Delta y}{\Delta x} = \dfrac{f(b)-f(a)}{b-a}$ — 두 점 사이 기울기 (실제 분수).
  • 순간변화율 = 미분계수: $f'(a) = \displaystyle\lim_{\Delta x \to 0}\dfrac{\Delta y}{\Delta x}$ — 극한값 (분수의 극한).
  • 도함수: $f'(x) = \dfrac{df}{dx} = \dfrac{dy}{dx}$ (라이프니츠 표기).
  • 정적분: $\displaystyle\int_a^b f(x)\,dx$ — $x$축과 그래프 사이 부호 있는 넓이.
1$\dfrac{dy}{dx}$의 본질 — "분수처럼 다루는 직관"

엄밀한 정의: $\dfrac{dy}{dx}$는 극한 $\displaystyle\lim_{\Delta x\to 0}\dfrac{\Delta y}{\Delta x}$의 결과로 나온 기호다. 분자·분모 따로 떼어서 분수처럼 곱하고 나누면 안 된다 (라이프니츠 표기법).

그런데도 마음속으로 분수로 보면 미적분의 거의 모든 공식이 약분·역수 같은 사칙연산으로 풀린다. 이 직관이 라이프니츠가 만든 표기법의 진짜 위력.

$$\dfrac{dy}{dx} = \dfrac{\text{"무한히 작은 } y \text{의 변화량"}}{\text{"무한히 작은 } x \text{의 변화량"}}$$
엄밀히는 ❌ / 직관적으로는 ✔
왜 분수로 보면 편한가: 평균변화율 $\dfrac{\Delta y}{\Delta x}$는 진짜 분수다. 극한을 취해도 그 "분수의 모양"이 그대로 유지되도록 라이프니츠가 일부러 $d$ 기호를 도입했다. 그래서 분수의 성질이 결과적으로 통한다.
📌 합성함수 미분 (연쇄법칙) — 분수의 약분

$y = f(u),\;u = g(x)$일 때 $\dfrac{dy}{dx}$를 구하라.

암기 X
"$\dfrac{dy}{dx} = \dfrac{dy}{du} \cdot \dfrac{du}{dx}$" 공식을 외우지 말고…
직관 ✓
$\dfrac{dy}{dx} = \dfrac{dy}{\cancel{du}} \cdot \dfrac{\cancel{du}}{dx}$ — $du$가 약분되어 자연스럽게 성립.
적용: $y = (x^2+1)^3$에서 $u = x^2+1$로 두면 $\dfrac{dy}{du} = 3u^2$, $\dfrac{du}{dx} = 2x$.
$\dfrac{dy}{dx} = 3u^2 \cdot 2x = 6x(x^2+1)^2$
학습 범위 안내: 매개변수 미분 $\dfrac{dy}{dx} = \dfrac{dy/dt}{dx/dt}$, 역함수 미분 $\dfrac{dx}{dy} = \dfrac{1}{dy/dx}$, 음함수 미분 등은 선택 미적분 과목 영역이라 현 고2 수능 범위 밖. 그러나 분수 직관은 동일하게 작동 — 미적분Ⅰ 안에서 합성함수 미분(다항함수)에 적용해 충분.
2미적분의 제1기본정리 — "넓이를 미분하면 원래 함수"

구간 $[a, x]$에서 $f(t)$가 만드는 넓이를 $S(x)$라 하면:

$$S(x) = \int_a^x f(t)\,dt, \qquad S'(x) = f(x)$$
넓이 함수를 미분하면 원래 함수!
왜 적분이 "넓이"가 되는가: 적분의 정의 그 자체가 "넓이"여서가 아니다. "넓이를 미분하면 원래 함수다"라는 놀라운 사실을 거꾸로 이용 — 즉 넓이를 구하려면 "미분해서 $f(x)$가 되는 함수(역도함수)"를 찾으면 된다는 것이 핵심.
1
$S(x)$ 정의: 구간 $[a, x]$에서 $f(t)$가 $t$축과 만드는 넓이.
2
$S(x)$의 변화량: $x$를 $\Delta x$만큼 늘리면 새로 생기는 넓이 $\Delta S \approx f(x)\cdot\Delta x$ (얇은 직사각형).
3
미분계수: $\displaystyle\lim_{\Delta x \to 0}\dfrac{\Delta S}{\Delta x} = f(x)$ → 따라서 $S'(x) = f(x)$.
y │ ┌──────y=f(t)────── │ ┃ ←Δx만큼 늘림 │ ┃ ↓ ●━━━━━━━━━━━━━━━● 새 넓이 ΔS ≈ f(x)·Δx ◀═══ S(x) ═══▶◀━Δx━▶ a x ↓ ΔS / Δx → f(x) (Δx → 0) ⇒ S'(x) = f(x)
정적분의 기본정리 (계산용):
$F(x)$가 $f(x)$의 역도함수이면 ($F'(x) = f(x)$)
$$\int_a^b f(x)\,dx = F(b) - F(a)$$
📌 활용 예 — 다항함수 정적분

$\displaystyle\int_0^2 (3x^2 + 1)\,dx$ 를 구하시오.

역도함수
$3x^2$의 역도함수는 $x^3$, $1$의 역도함수는 $x$ → $F(x) = x^3 + x$.
$F(b) - F(a)$
$F(2) - F(0) = (8 + 2) - 0 = 10$
본질: 답 $10$은 "$y = 3x^2+1$이 $[0,2]$에서 만드는 넓이"임. 미분이 넓이의 변화율이라서 거꾸로 역도함수로 넓이 계산 가능.
기초 분수 직관·기본정리 적용
기초 1 — 합성함수 미분
$y = (2x + 3)^4$일 때 $\dfrac{dy}{dx}$를 구하시오.
풀이 보기
치환
$u = 2x + 3$ → $y = u^4$.
분수 직관
$\dfrac{dy}{dx} = \dfrac{dy}{du}\cdot\dfrac{du}{dx} = 4u^3 \cdot 2 = 8(2x+3)^3$
정답: $8(2x+3)^3$
기초 2 — 정적분 기본정리
$\displaystyle\int_1^3 (2x - 1)\,dx$의 값을 구하시오.
풀이 보기
역도함수
$F(x) = x^2 - x$ (미분하면 $2x-1$ 확인)
계산
$F(3) - F(1) = (9-3) - (1-1) = 6 - 0 = 6$
정답: 6
기초 3 — 넓이 함수 미분
$f(t) = t^2 + 1$일 때 $S(x) = \displaystyle\int_0^x (t^2+1)\,dt$로 정의된 함수 $S(x)$의 도함수 $S'(x)$를 구하시오.
풀이 보기
기본정리
$S(x)$가 $f(t)$의 넓이 함수이므로 $S'(x) = f(x) = x^2 + 1$.
직접 확인
$S(x) = \dfrac{x^3}{3} + x$ → $S'(x) = x^2 + 1$ ✔
정답: $x^2 + 1$
기본 분수 직관 + 기본정리 연계
기본 1 — 합성함수 연쇄
$y = (x^3 - 2x)^5$일 때 $\dfrac{dy}{dx}$를 구하시오.
풀이 보기
치환
$u = x^3 - 2x$ → $\dfrac{du}{dx} = 3x^2 - 2$. $y = u^5$ → $\dfrac{dy}{du} = 5u^4$.
분수 약분
$\dfrac{dy}{dx} = \dfrac{dy}{du}\cdot\dfrac{du}{dx} = 5(x^3-2x)^4 \cdot (3x^2 - 2)$
정답: $5(x^3-2x)^4(3x^2-2)$
기본 2 — 정적분 + 도형 해석
$\displaystyle\int_0^a (4 - x^2)\,dx = \dfrac{16}{3}$을 만족하는 양수 $a$의 값을 구하시오.
풀이 보기
역도함수
$F(x) = 4x - \dfrac{x^3}{3}$
방정식
$F(a) - F(0) = 4a - \dfrac{a^3}{3} = \dfrac{16}{3}$
$\dfrac{12a - a^3}{3} = \dfrac{16}{3}$ → $12a - a^3 = 16$ → $a^3 - 12a + 16 = 0$
인수분해
$a = 2$ 대입: $8 - 24 + 16 = 0$ ✔ → $(a-2)$ 인수.
$a^3 - 12a + 16 = (a-2)(a^2+2a-8) = (a-2)(a+4)(a-2) = (a-2)^2(a+4)$
양수 해
$a = 2$ (중근). $a = -4$ 양수 아님.
정답: $a = 2$
기본 3 — 넓이 함수 + 미분
$S(x) = \displaystyle\int_1^x (3t^2 + 2t)\,dt$일 때 $S(2)$와 $S'(2)$의 값을 구하시오.
풀이 보기
$S(2)$
역도함수 $t^3 + t^2$.
$S(2) = (8 + 4) - (1 + 1) = 12 - 2 = 10$
$S'(2)$ — 기본정리
$S'(x) = 3x^2 + 2x$. $S'(2) = 12 + 4 = 16$
핵심: $S'(x)$는 적분 안의 함수($f$)에 $x$ 그대로 대입. 적분 결과를 일일이 정리할 필요 없음.
정답: $S(2) = 10$, $S'(2) = 16$
심화 기본정리의 변형
심화 1 — 변수 적분 한계
$S(x) = \displaystyle\int_0^{x^2} (2t + 1)\,dt$일 때 $S'(x)$를 구하시오.
풀이 보기
분수 직관 — 연쇄법칙
적분 한계가 $x^2$이므로 $S = F(x^2) - F(0)$, $F$가 $2t+1$의 역도함수.
$\dfrac{dS}{dx} = F'(x^2)\cdot\dfrac{d(x^2)}{dx} = f(x^2) \cdot 2x$
$f(x^2) = 2x^2 + 1$.
$S'(x) = (2x^2 + 1)\cdot 2x = 4x^3 + 2x$
직접 확인
$S(x) = (x^2)^2 + x^2 = x^4 + x^2$ → $S'(x) = 4x^3 + 2x$ ✔
정답: $4x^3 + 2x$
심화 2 — 적분과 미분의 환원
함수 $f(x)$가 $\displaystyle\int_0^x f(t)\,dt = x^3 - 2x^2 + x$를 만족할 때 $f(2)$의 값을 구하시오.
풀이 보기
양변 미분 (기본정리)
좌변 = $S(x)$, $S'(x) = f(x)$.
우변 미분: $3x^2 - 4x + 1$.
$\therefore f(x) = 3x^2 - 4x + 1$
$f(2)$
$f(2) = 12 - 8 + 1 = 5$
정답: 5
수능 대비 준킬러 — 기본정리 종합
수능 대비 1 — 준킬러 (미정계수 + 기본정리)
다항함수 $f(x)$가 다음 두 조건을 만족시킨다.

(가) $\displaystyle\int_0^x f(t)\,dt = x^3 + ax^2 + b$
(나) $f(0) = -1$

$f(3)$과 상수 $a, b$의 값을 구하시오.
풀이 보기
$x=0$ 대입 → $b$
좌변 $\int_0^0 = 0$ → $b = 0$.
양변 미분
$f(x) = 3x^2 + 2ax$
$f(0) = -1$?
$f(0) = 0 \neq -1$ → 모순! 조건 재해석 필요.

실제 풀이: (가)에서 $x=0$ 대입이 $0 = b$가 아니라, $f(t)$ 의 적분 자체가 $f(0)$ 정보를 잃을 수 있음. (가)의 식이 다항함수라면 (가)를 미분한 $f(x) = 3x^2 + 2ax$는 자동으로 $f(0) = 0$. (나)와 충돌.

→ 조건 (가)가 "$\int_0^x f(t)dt + C = x^3 + ax^2 + b$" 형태로 상수 $C$ 포함 가능. 또는 (가)를 약간 다르게 해석.

학습 결론: "$\int_0^x f(t)dt = $ 다항식"이면 그 다항식의 상수항 = 0이어야 일관. 추가 정보 (나)는 이 경우 (가)의 미분 결과로 직접 유도되지 않으므로 모순 검출.
정합 조건으로 수정
(나)를 "$f(0) = 0$"으로 가정하면:
$f(x) = 3x^2 + 2ax$, $f(0) = 0$ 자동.
$a$ 결정 위해 (다) 추가 필요. 예: $f(1) = 5$ → $3 + 2a = 5$ → $a = 1$.
$f(3) = 27 + 6 = 33$.
핵심 학습: 적분식의 양변 미분 + 적분 한계 대입 두 가지로 미정계수 결정. 정합성 검증 필수.
학습용 — 양변 미분 + 한계 대입 흐름
수능 대비 2 — 준킬러 (적분 = 다항식)
다항함수 $f(x)$에 대해 $\displaystyle\int_1^x f(t)\,dt = x^2 + ax + b$가 성립하고 $f(2) = 5$일 때, $a + b$의 값을 구하시오.
풀이 보기
$x=1$ 대입
좌변 $0$. $1 + a + b = 0$ → $a + b = -1$ ... ①
양변 미분
$f(x) = 2x + a$. $f(2) = 4 + a = 5$ → $a = 1$.
$b$
①: $1 + b = -1$ → $b = -2$.
$a + b$
$1 + (-2) = -1$ (또는 ①에서 즉시 확인)
정답: $a + b = -1$
수능 대비 3 — 준킬러 (넓이 함수의 도함수)
함수 $S(x) = \displaystyle\int_0^x (t^2 - 4t + 3)\,dt$ 의 극값을 구하시오.
풀이 보기
$S'(x)$ — 기본정리
$S'(x) = x^2 - 4x + 3 = (x-1)(x-3)$
극값 위치
$S'(x) = 0$인 $x = 1, 3$. 증감표:
$x < 1$: $S' > 0$ (증가) → $x = 1$ 극대.
$1 < x < 3$: $S' < 0$ (감소) → $x = 3$ 극소.
극값 계산
$S(x) = \dfrac{x^3}{3} - 2x^2 + 3x$.
$S(1) = \dfrac{1}{3} - 2 + 3 = \dfrac{4}{3}$ (극대)
$S(3) = 9 - 18 + 9 = 0$ (극소)
핵심: $S'(x)$ 식을 풀 때 $S(x)$를 직접 적분 안 해도 미분 결과는 적분 안의 함수. 단 극값 자체 계산은 $S$를 구해야.
극대 $\dfrac{4}{3}$ ($x=1$), 극소 $0$ ($x=3$)
📝 한눈에 보는 정리
도구핵심 식활용
합성함수 미분 $\dfrac{dy}{dx} = \dfrac{dy}{du}\cdot\dfrac{du}{dx}$ $du$ 약분 — 분수 직관
기본정리 (미분) $\dfrac{d}{dx}\displaystyle\int_a^x f(t)dt = f(x)$ 적분식 양변 미분
기본정리 (적분) $\displaystyle\int_a^b f\,dx = F(b)-F(a)$ 역도함수 차로 정적분 계산
변수 한계 적분 $\dfrac{d}{dx}\displaystyle\int_a^{g(x)} f(t)dt = f(g(x))g'(x)$ 연쇄법칙 + 기본정리 결합
💡 시험장 체크리스트
① $\dfrac{dy}{dx}$ 공식 외울 때 분수의 약분으로 이해.
② $\int_a^x f(t)dt = $ 다항식 형태 → 즉시 양변 미분하여 $f(x)$ 추출.
③ 적분 식에서 한계가 변수면 한계 미분도 곱해주기.
④ $x=a$ 대입 ($\int_a^a = 0$)으로 상수항 정보 얻기.
⑤ 넓이 함수의 극값 = $f(x) = 0$인 점.
