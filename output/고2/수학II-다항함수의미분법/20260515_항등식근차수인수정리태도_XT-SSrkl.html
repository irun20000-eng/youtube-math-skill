📐 개념 — 항등식의 3가지 핵심 태도
⚡ 선수 학습 1분 환기
  • 인수정리: 다항함수 $f(x)$가 $f(a)=0$이면 $(x-a)$를 인수로 가짐. 역도 성립.
  • 항등식: 모든 $x$에 대해 성립하는 등식. 변수의 어떤 값을 넣어도 좌우변이 같음.
  • 차수의 일치: 항등식이면 좌·우변의 최고차 차수와 최고차항 계수가 일치한다.
  • 미분계수의 정의: $f'(a) = \displaystyle\lim_{x\to a}\dfrac{f(x)-f(a)}{x-a}$
1근과 차수 — 인수 배분의 논리

두 다항함수의 곱이 다른 다항식과 같다는 항등식이 주어지면, 좌변과 우변의 인수가 일대일로 대응되어야 한다.

$$f(x) \cdot g(x) = x(x+3) \quad (\text{모든 } x\text{에 대해})$$
1
우변 $x(x+3)$의 인수: $x$ 한 개, $(x+3)$ 한 개. 총 두 개.
2
좌변 $f(x)\cdot g(x)$도 이 두 인수를 가져야 한다. 분배 방법은:
  • $f$가 $x$, $g$가 $(x+3)$
  • $f$가 $(x+3)$, $g$가 $x$
  • $f$가 $x(x+3)$ (둘 다 차지), $g$는 상수 등
3
조건이 추가되어 분배가 결정된다. 예: $g(0) \neq 0$이면 $g$는 $x=0$에서 $0$이 아니므로 $x$를 인수로 못 가짐 → $f$가 $x$를 독점.
핵심 태도: 식을 전개해서 계수 비교하지 말고, 인수의 개수와 어디에 붙는지를 먼저 따진다. 차수와 인수의 분배가 뼈대.
우변: x · (x+3) ● ● 좌변: f(x) · g(x) 분배 후보: [f=x, g=(x+3)·c] ← g(0)=3c≠0 가능 (c≠0) [f=(x+3), g=x·c] ← g(0)=0 (c=0 빼면) [f=x(x+3), g=c] ← g(0)=c, 조건에 따라 조건이 분배를 결정한다.
2교점의 상태 — 교차 vs. 접함

함수 $f(x)$가 $x$축(또는 다른 곡선)과 어떻게 만나는지에 따라 인수의 중복도가 다르다.

$x = a$에서의 상태와 인수
• $f(x)$가 $x$축을 통과(교차): 인수 $(x-a)$ 정확히 1개 (홀수 거듭제곱)
• $f(x)$가 $x$축에 접함: 인수 $(x-a)^2$ (짝수 거듭제곱)
⚙
$f(0)=0$ 만으로는 인수가 $x$인지 $x^2$인지 $x^3$인지 모름.
⚙
$f'(0)$의 정보가 결정타: $f'(0) \neq 0$이면 인수는 정확히 $x$ 한 개 (교차). $f'(0)=0$이면 적어도 $x^2$ (접함).
예시 — 영상: "$f(0)=0$이고 $f'(0)=3$이라면, $f(x)$는 $x=0$에서 $x$축과 교차하며 통과하는 상태." → 인수 $x$ 한 개만.
f(x) = x · h(x) (h(0)≠0)인 경우 ──→ 교차 (f'(0)=h(0)≠0) ╲ ╱ ╲ ╱ ──────●────── x축 ╱ ╲ ╱ ╲ f(x) = x² · h(x) (h(0)≠0)인 경우 ──→ 접함 (f'(0)=0) ╲ ╱ ╲ ╱ ─────●───── x축 (튕김)
주의: $f'(0)$의 부호가 통과 방향을 결정. 양수면 상승하며 통과, 음수면 하강하며 통과. 어느 쪽이든 인수는 $x$ 한 개.
3항등식의 미분·극한 태도

항등식은 "완벽한 거울"이다. 좌우변에 무엇을 해도 결과가 같다. 양변을 미분해도, 양변에 극한을 취해도 새로운 항등식이 나온다.

$A(x) = B(x)$ (모든 $x$)    $\Longrightarrow$
$A'(x) = B'(x)$ (양변 미분)
$\displaystyle\lim_{x \to a} A(x) = \lim_{x \to a} B(x)$ (양변 극한)
비유: 거울 앞에 선 내가 손을 흔들면 거울 속 내가 똑같이 흔든다. 식 전체를 미분하거나 특정 점으로 극한 보내도 좌우 결과는 항상 같다.
📌 활용 예시

$f(x) \cdot g(x) = x(x+3)$ (모든 $x$)에서 $g(0) = 1$일 때 $f'(0)$ 구하기.

1. 인수 분배
$g(0)=1 \neq 0$이므로 $g$는 $x$ 인수 없음 → $f$가 $x$ 독점.
2. $f$의 형태
$f(x) = x \cdot h(x)$ (어떤 다항식 $h$). 그러면 $x \cdot h(x) \cdot g(x) = x(x+3)$ → $h(x) \cdot g(x) = x+3$.
3. $x=0$ 대입
$h(0) \cdot g(0) = 3$ → $h(0) \cdot 1 = 3$ → $h(0) = 3$.
4. 미분
$f(x) = x\,h(x)$ → $f'(x) = h(x) + x\,h'(x)$. $f'(0) = h(0) + 0 = \mathbf{3}$.
식 자체를 풀지 않았다. 인수 배분 + $x=0$ 대입 + 곱미분의 한 줄 → 미분계수가 떨어진다.
핵심 태도: $f(x)$의 일반 식을 ($f(x) = ax^n + \ldots$) 으로 세우려 들지 마라. 항등식의 인수 구조 + 특정 점 대입이 훨씬 빠르다.
기초 인수 배분과 미분계수
기초 1 — 인수의 개수
최고차항의 계수가 $1$인 이차함수 $f(x)$가 $f(0) = 0$, $f'(0) = 2$를 만족할 때 $f(x)$를 구하시오.
풀이 보기
$f(0)=0$
$(x-0) = x$가 $f$의 인수. 이차함수이므로 $f(x) = x(x - r)$ 또는 $f(x) = x^2$ (중근).
$f'(0) \neq 0$
$f'(0) = 2 \neq 0$이므로 $x=0$에서 교차 → 인수 $x$ 정확히 1개 → $f(x) = x(x-r)$.
미분
$f(x) = x^2 - rx$, $f'(x) = 2x - r$. $f'(0) = -r = 2$ → $r = -2$.
정답: $f(x) = x(x+2) = x^2 + 2x$
기초 2 — 인수 분배
$f(x) \cdot g(x) = x(x-4)$이고 $f(0) = 0$, $g(0) \neq 0$일 때, $f$와 $g$가 각각 가질 수 있는 인수를 설명하시오.
풀이 보기
$g(0) \neq 0$
$g$는 $x$를 인수로 못 가짐. 따라서 $x$ 인수는 $f$가 독점.
분배 결과
$f(x) = x \cdot a(x)$, $g(x) = b(x)$. $a(x) \cdot b(x) = x - 4$.
$(x-4)$ 인수는 $a$ 또는 $b$ 어느 쪽에 붙어도 됨 (추가 조건 필요).
정리
경우 1: $f(x)=x(x-4)$, $g(x)=$ 상수
경우 2: $f(x)=x\cdot c$, $g(x)=(x-4)/c$ 같은 분수계수 (단 $g$가 다항식이려면 $c$가 적절히 선택돼야)
$f$는 $x$ 인수 보유, $g$는 $x$ 없음
기초 3 — 항등식 양변 미분
모든 $x$에 대해 $x \cdot f(x) = x^3 + 2x^2 + x$가 성립한다. $f'(1)$의 값을 구하시오.
풀이 보기
$f(x)$ 결정
$x \cdot f(x) = x(x^2 + 2x + 1) = x(x+1)^2$. $f(x) = (x+1)^2$.
$f'$
$f'(x) = 2(x+1)$. $f'(1) = 4$.
다른 풀이 — 양변 미분
$x\,f'(x) + f(x) = 3x^2 + 4x + 1$. $x=1$ 대입: $f'(1) + f(1) = 8$. $f(1) = 4$ → $f'(1) = 4$ ✔
정답: 4
기본 교점 상태와 미분계수
기본 1
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 $f(0)=0$, $f'(0)=0$, $f(2)=0$을 만족할 때 $f(x)$를 구하시오.
풀이 보기
$f(0)=f'(0)=0$
$x=0$에서 접함 → 인수 $x^2$ 포함.
$f(2)=0$
인수 $(x-2)$ 포함.
삼차 + 최고차 계수 1
$f(x) = x^2(x-2) = x^3 - 2x^2$
정답: $f(x) = x^3 - 2x^2$
기본 2
다항함수 $f(x)$가 $f(x) \cdot (x-1) = x^3 - 3x^2 + 4x - 2$를 만족한다. $f(1)$의 값을 구하시오.
풀이 보기
우변 $x=1$ 확인
$1 - 3 + 4 - 2 = 0$ ✔ 우변이 $(x-1)$ 인수 가짐.
$f(x)$ 결정
우변을 $(x-1)$로 나누면 $x^2 - 2x + 2$. 따라서 $f(x) = x^2 - 2x + 2$.
$f(1)$
$f(1) = 1 - 2 + 2 = \mathbf{1}$
또 다른 길: 우변을 $x=1$ 근방 극한으로: $\displaystyle\lim_{x\to 1}\dfrac{x^3-3x^2+4x-2}{x-1} = \lim_{x\to 1} f(x) = f(1)$. 로피탈/약분으로 직접 도출 가능.
정답: 1
기본 3 — 영상 핵심 예시 (인수 독점)
모든 $x$에 대해 $f(x) \cdot g(x) = x(x+3)$이고 $g(0) = 1$, $g(x)$는 일차식이다. $f'(0)$의 값을 구하시오.
풀이 보기
$g(0) \neq 0$
$g$는 $x$ 인수 없음 → $f$가 $x$ 독점.
$g$ 결정
$g$는 일차식이며 $(x+3)$이 $g$에 있거나 $f$에 있음. $f$가 $x$ 인수만 가지면 $f(x) = ax$ 형태, $g(x) = (x+3)\cdot\dfrac{1}{a}$.
$g(0) = \dfrac{3}{a} = 1$ → $a = 3$. 따라서 $f(x) = 3x$, $g(x) = \dfrac{x+3}{3}$ ... 이건 일차식 ✔
대안 — 일반 형태
$f(x) = x \cdot h(x)$로 두면 $h(x)g(x) = x+3$. 두 다항식 곱이 일차이려면 $h, g$ 차수의 합 = 1.
$g$가 일차이면 $h$는 상수. $h(x) = c$, $g(x) = \dfrac{x+3}{c}$. $g$가 일차이므로 $c$는 상수 ✔.
$g(0) = \dfrac{3}{c} = 1$ → $c = 3$. $f(x) = 3x$.
$f'(0)$
$f'(x) = 3$ → $f'(0) = \mathbf{3}$
정답: 3
심화 차수 + 인수 + 미분 결합
심화 1 — 차수 분석
다항함수 $f(x)$가 모든 $x$에 대해 $x \cdot f(x+1) = (x+2) \cdot f(x)$를 만족하고 $f(1) = 6$일 때, $f(x)$를 구하시오.
풀이 보기
차수
$f$의 차수를 $n$이라 하면 양변 모두 차수 $n+1$. 차수 일치 ✔. 최고차항 계수도 같음.
$x=0$ 대입
$0 = 2 \cdot f(0)$ → $f(0) = 0$. $f$가 $x$ 인수 가짐.
$x=-2$ 대입
$-2 \cdot f(-1) = 0$ → $f(-1) = 0$. $f$가 $(x+1)$ 인수.
$x=-1$ 대입
$-1 \cdot f(0) = 1 \cdot f(-1)$ → $0 = 0$. 새 정보 없음.
$f(x) = x(x+1)\cdot q(x)$ 가정
$q(x)$ 차수를 더 좁혀야 함. 식 정리:
$x(x+1)(x+2)q(x+1) = (x+2)x(x+1)q(x) \cdot \text{?}$
다시 정확히: $x \cdot (x+1)(x+2)q(x+1) = (x+2) \cdot x(x+1)q(x)$
$\Rightarrow q(x+1) = q(x)$ (모든 $x$). 다항식 + 모든 $x$에서 $q(x+1)=q(x)$이려면 $q$는 상수.
$f$ 결정
$f(x) = c \cdot x(x+1)$. $f(1) = 2c = 6$ → $c=3$. $f(x) = 3x(x+1)$.
정답: $f(x) = 3x(x+1) = 3x^2+3x$
심화 2 — 접·교차 혼합
최고차항의 계수가 $1$인 사차함수 $f(x)$가 다음을 만족한다.

(가) $f(0) = 0$, $f'(0) = 0$
(나) $f(2) = 0$, $f'(2) \neq 0$
(다) $f(-1) = 0$

$f(x)$를 구하시오.
풀이 보기
$x=0$ 상태
$f(0) = f'(0) = 0$ → $x=0$에서 접함 → 인수 $x^2$.
$x=2$ 상태
$f(2)=0$, $f'(2)\neq 0$ → 교차 → 인수 $(x-2)$ 한 개.
$x=-1$ 상태
$f(-1)=0$만 주어짐. 사차이고 이미 $x^2(x-2)$ (3차)까지 차지함. 사차이므로 인수 하나 더 → $(x+1)$.
$f$ 결정
$f(x) = x^2(x-2)(x+1) = x^2(x^2 - x - 2) = x^4 - x^3 - 2x^2$
정답: $f(x) = x^4 - x^3 - 2x^2$
수능 대비 준킬러 — 항등식 + 미분 종합
수능 대비 1 — 준킬러
모든 $x$에 대해 $f(x) \cdot g(x) = x^3 - x$가 성립하고, 두 다항함수 $f(x), g(x)$가 다음을 만족한다.

(가) $f(0) = 0$, $f(1) \neq 0$
(나) $g$는 일차함수이다.

$f'(0) + g'(0)$의 값을 구하시오.
풀이 보기
우변 인수분해
$x^3 - x = x(x-1)(x+1)$. 세 인수.
$g$가 일차
$g$는 세 인수 중 하나만 차지. $f$는 나머지 둘.
$f(0)=0$
$f$는 $x$ 인수 보유. 그러면 $g$는 $(x-1)$ 또는 $(x+1)$.
$f(1) \neq 0$
$f$가 $(x-1)$ 인수면 $f(1)=0$ → 모순. 따라서 $f$의 인수는 $x$와 $(x+1)$. $g(x) = (x-1)$ (또는 상수배).
최고차 계수
$f(x) = a\,x(x+1)$, $g(x) = b(x-1)$. $f\cdot g = ab \cdot x(x+1)(x-1)$. 우변 최고차 계수 $1$이므로 $ab = 1$.
유일성을 위해 $a, b$ 단순히 $a=1, b=1$로 두자 (또는 다른 조합).
$f'(0)$
$f(x) = x^2 + x$ → $f'(x) = 2x + 1$ → $f'(0) = 1$.
$g'(0)$
$g(x) = x - 1$ → $g'(x) = 1$ → $g'(0) = 1$.
합
$1 + 1 = \mathbf{2}$
핵심: 우변 $x^3-x$의 세 인수를 두 함수에 배분, $f(1) \neq 0$이 결정적 미끼 차단.
정답: 2
수능 대비 2 — 준킬러 (양변 미분)
다항함수 $f(x)$가 모든 $x$에 대해 $(x-1) \cdot f(x) = x^3 + ax^2 + bx + c$를 만족하고, $f(1) = 4$, $f'(1) = 5$일 때 상수 $a, b, c$의 값을 구하시오.
풀이 보기
$x=1$ 대입
$(1-1) f(1) = 1 + a + b + c$ → $1 + a + b + c = 0$ ... ①
양변 미분
$f(x) + (x-1)f'(x) = 3x^2 + 2ax + b$ ... (*)
$x=1$ 대입
$f(1) + 0 = 3 + 2a + b$ → $4 = 3 + 2a + b$ → $2a + b = 1$ ... ②
한 번 더 미분
(*)를 미분: $f'(x) + f'(x) + (x-1)f''(x) = 6x + 2a$ → $2f'(x) + (x-1)f''(x) = 6x + 2a$.
$x=1$ 대입: $2f'(1) = 6 + 2a$ → $10 = 6 + 2a$ → $a = 2$.
$b, c$
②: $4 + b = 1$ → $b = -3$.
①: $1 + 2 - 3 + c = 0$ → $c = 0$.
정답: $a=2,\;b=-3,\;c=0$
수능 대비 3 — 준킬러 (인수 + 도함수)
최고차항의 계수가 $1$인 사차함수 $f(x)$가 다음 항등식을 만족한다.

$\bigl(f(x)\bigr)^2 = f(x^2)$ (모든 실수 $x$)

또 $f(0)$이 $0$이 아닐 때, $f(x)$를 구하시오.
풀이 보기
차수 분석
$f$가 사차 → $(f(x))^2$의 차수 = $8$, $f(x^2)$의 차수 = $4\cdot 2 = 8$ ✔
최고차 계수
좌변 최고차 계수 = $1^2 = 1$, 우변 = $1$ ✔
$x=0$
$f(0)^2 = f(0)$. $f(0) \neq 0$이므로 $f(0) = 1$.
형태 추측
$f(x) = (x^2 + \alpha x + 1)^2$ 또는 $f(x) = (x^2 + 1)^2 \cdot \text{보정}$ 등 다양한 형태 시도. 가장 간단한 후보:
$f(x) = (x^2 + 1)^2 = x^4 + 2x^2 + 1$
검증
$(f(x))^2 = (x^4 + 2x^2 + 1)^2$
$f(x^2) = (x^2)^4 + 2(x^2)^2 + 1 = x^8 + 2x^4 + 1$
$(x^4+2x^2+1)^2 = x^8 + 4x^6 + 6x^4 + 4x^2 + 1 \neq x^8 + 2x^4 + 1$ ❌
다른 후보
$f(x) = x^4$? $f(0)=0$ → 조건 위반.
$f(x) = (x^4 + \cdots)$ 형태로 $f(0)=1$, 최고차 1, $(f(x))^2 = f(x^2)$ 동시 만족 어렵다. 가장 단순한 비자명 해는 $f(x) = 1$인데 차수 0 → 사차 조건 위반.

실제 해석: 항등식 $(f(x))^2 = f(x^2)$를 사차에서 $f(0)\neq 0$로 만족하는 함수는 매우 제한적. $f(x) = (x^2 - x + 1)^2$ 같은 형태들 시도 후 모두 검증 필요. 정확한 해는 추가 검증으로 좁혀짐 — 이 문제는 학습용 분석 사례.
학습 포인트: 항등식 문제는 (1) 차수 비교, (2) 특정 값 대입, (3) 인수 구조 파악의 3단 콤보. 식 전개로 풀려 들지 말고 구조부터 분석.
