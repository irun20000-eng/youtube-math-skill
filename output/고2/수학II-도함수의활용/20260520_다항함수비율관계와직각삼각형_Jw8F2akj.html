1다항함수의 고유 비율 관계

각 차수의 다항함수가 가지는 고정된 비율들. 식 모르고도 그래프 한 번 보고 즉시 적용.

▸ 2차 함수 — 넓이 2:1

이차함수 $y = a(x-\alpha)(x-\beta)$ 의 꼭짓점에서 $x$축으로 내린 수선
→ 곡선 아래 넓이를 2 : 1 로 분할
y = ax² 의 꼭짓점 ● │ ● ← 꼭짓점 │╱ ╲ 꼭짓점 수직선이 곡선과 만든 ╱ ╲ 전체 넓이를 2:1로 나눈다 ❌ ╱ S₁ ╲ (실제로는 정확히 같은 두 부분 1:1) ╱ │ ╲ ─●───╱│╲───●─ x축 α │ β ※ 사실: 꼭짓점에서 수선 내리면 두 부분 1:1 (좌우대칭). 비율 "2:1"은 다른 분할 — 예: 일차함수와 둘러싸인 영역에서 적용. 참고: 곡선 + 일차 직선 분할이 2:1 등은 영상의 변형 예.

▸ 3차 함수 — 1:1:1:1 & 1:$\sqrt 3$

점대칭 + 극값 간격: 3차함수 $f(x) = a(x^3 + \cdots)$의 변곡점·극대점·극소점·근 사이의 거리 비가 정해진 패턴.
극대 ↔ 변곡점 ↔ 극소 사이 $x$좌표 간격: 1 : 1 (변곡점이 정중앙)
극대(또는 극소)에서 곡선이 다시 같은 $y$값에 도달하는 거리: 극값 간격의 1 : $\sqrt 3$
⚙
변곡점 점대칭: 모든 3차함수는 변곡점에 대해 점대칭. 극대점과 극소점은 변곡점에 대해 대칭 위치 — 즉 변곡점이 두 극값의 중점.

▸ 4차 함수 — 대칭축 1:$\sqrt 2$ & 3:1

좌우대칭 4차함수 $f(x) = a(x^2 - p^2)^2 + c$ 같이 좌우 대칭이면 대칭축 기준으로 극값·근의 거리가 단순 비율.
극소(중앙) ↔ 극대 사이 $x$ 거리 : 극대 ↔ 인접 극소 = 1 : 1
핵심 태도: 다항함수 그래프 보면 "좌표값" 대신 "거리 간격 $k$"로 본다. 모든 좌표를 $\alpha, \alpha+k, \alpha+2k, \ldots$ 형태로. 이후 비율 관계로 미지수 결정.
2도함수 넓이 = 원함수 함숫값 차

두 그래프(원함수 $f$, 도함수 $f'$)를 따로 보지 말고 한 정보의 두 표현으로 본다.

$$\int_a^b f'(x)\,dx = f(b) - f(a)$$
$f'$ 그래프의 부호 있는 넓이 = $f$의 함숫값 변화량 $h$
원함수 f 도함수 f' y y │ 극대 ● │ + │ ╱╲ │ h │╱─╲ 양 넓이 = h │ ╱ ╲ │ │ ╲ ●╱ ╲↓ ●────●──── x → 음 넓이 = h' ╲ ╲ ╲ ╲ 극소 ● ─╲ 높이 차 h = 도함수의 부호 있는 넓이
⚙
각종 넓이 공식의 통일: 6분 공식·12분 공식 등은 모두 "도함수 넓이"의 특수 사례. 원함수 입장에서는 극값 차이를 만든 양.
📌 영상 핵심 관점

"빨간색 길이"(= 원함수의 함숫값 변화량 $h$)를 도함수 입장에서 보면 → 넓이.

강사 인용: "이 빨간색 길이가 의미하는 건 뭐야... 도함수 입장에서 뭘 보면 되죠? 얼마나 떨어졌어? 얼마나 변화했어? 넓이를 보면 돼요."
실전 적용: 문제에 "극값의 차이가 N", "두 점에서 같은 함숫값" 같은 조건이 나오면 → 즉시 도함수 그래프의 넓이로 환원.
3접선 문제 — 직각삼각형 3요소

접선이 등장하면 곡선 식을 안 쓰고 접점·접선·수직선이 만드는 직각삼각형의 3요소만 추적.

3요소: 밑변 (가로 $\Delta x$) · 높이 (세로 $\Delta y$) · 기울기 ($\tan \theta = \dfrac{\Delta y}{\Delta x}$)
두 개를 알면 나머지 하나 자동.
접선 + 곡선의 직각삼각형 │ ╱ ← 접선 (기울기 = f'(a)) │ ╱ │ ╱ │ ● │ ← 높이 = Δy = f'(a)·Δx 접점│ │ │ ●──● x축 a a+Δx ◀─▶ 밑변 = Δx 세 정보: • 밑변 = a+Δx - a = Δx • 높이 = f'(a)·Δx (접선 위 변화) • 기울기 = f'(a) (접점에서 도함수 값)
⚙
접선과 다음 교점 거리: 삼차함수 접선과 곡선의 다른 교점 사이 $x$ 거리 = $k$로 두면, 12분 공식이 $\dfrac{|a|k^4}{12}$ 형태가 자연스럽게.
📌 직각삼각형 활용 예

$f(x) = x^3 - 3x$의 $x = 1$에서 접선이 $x$축과 만나는 점까지의 거리.

기울기
$f'(1) = 3 - 3 = 0$. 접선 수평. 특수 케이스.
$x=2$ 접선
$f(2) = 2$, $f'(2) = 9$. 접선 $y = 9(x-2) + 2 = 9x - 16$.
$x$축 교점: $9x = 16$ → $x = \dfrac{16}{9}$.
직각삼각형: 밑변 $= 2 - \dfrac{16}{9} = \dfrac{2}{9}$, 높이 $= 2$, 기울기 $= 9$ → $\dfrac{2}{2/9} = 9$ ✔
강사 인용: "이동준은 접선 문제를 항상 어떻게 푸냐... 직각삼각형의 3요소를 기억하세요. 밑변, 높이, 기울기. 세 개 중에 두 개를 알면 나머지 하나 따라와요."
기초 간격 변수·직각삼각형
기초 1 — 거리 간격 변수
삼차함수 $f(x) = x(x-2)(x-4)$의 변곡점·극대점·극소점의 $x$좌표를 각각 구하시오.
풀이 보기
근의 평균
근 $0, 2, 4$. 평균 = $2$. 변곡점 $x$좌표 = $2$ (점대칭 중심).
$f'(x)$
$f(x) = x^3 - 6x^2 + 8x$
$f'(x) = 3x^2 - 12x + 8 = 0$
$x = \dfrac{12 \pm \sqrt{144 - 96}}{6} = \dfrac{12 \pm \sqrt{48}}{6} = 2 \pm \dfrac{2\sqrt 3}{3}$
극값 위치
$x = 2 - \dfrac{2\sqrt 3}{3}$ 극대, $x = 2 + \dfrac{2\sqrt 3}{3}$ 극소.
간격 $k = \dfrac{2\sqrt 3}{3}$.
변곡점 $x=2$, 극대 $x=2-\dfrac{2\sqrt 3}{3}$, 극소 $x=2+\dfrac{2\sqrt 3}{3}$
기초 2 — 직각삼각형 3요소
곡선 $y = x^2$ 위의 점 $(2, 4)$에서의 접선이 $x$축과 만나는 점의 $x$좌표를 직각삼각형 관점으로 구하시오.
풀이 보기
3요소
접점 $(2, 4)$. 기울기 $f'(2) = 4$. 높이 $4$ (접점에서 $x$축까지).
밑변
밑변 $= \dfrac{높이}{기울기} = \dfrac{4}{4} = 1$.
$x$축 교점
접점 $x = 2$, 밑변 $1$만큼 떨어진 곳. $x = 2 - 1 = 1$ (접선이 내려가는 방향).
정답: $x = 1$
기초 3 — 도함수 넓이 = 극값 차
$f(x) = x^3 - 12x$의 극댓값과 극솟값의 차이를 구하시오.
풀이 보기
$f'(x)$
$f'(x) = 3x^2 - 12 = 3(x-2)(x+2)$. 극대 $x = -2$, 극소 $x = 2$.
도함수 넓이
$f$의 극값 차 = $\left|\int_{-2}^{2} f'(x)dx\right| = \dfrac{|3|}{6}(2-(-2))^3 = \dfrac{1}{2}\cdot 64 = 32$.
검산
$f(-2) = -8 + 24 = 16$, $f(2) = 8 - 24 = -16$. 차 = $32$ ✔
정답: 32
기본 간격 변수 + 비율
기본 1 — 3차함수 점대칭
삼차함수 $f(x) = ax^3 + bx + c$가 극대점 $x = -1$, 극소점 $x = 3$을 가질 때, 변곡점의 $x$좌표를 구하시오.
풀이 보기
중점
변곡점은 극대·극소의 중점 = $\dfrac{-1 + 3}{2} = 1$.
검증
$f$가 $bx^2$ 항 없으면 $f''(x) = 6ax$ → $x=0$. 모순 ($b$가 있어야 함).
정확히: $f(x) = ax^3 + bx^2 + cx + d$에서 $f''(x) = 6ax + 2b = 0$ → $x = -\dfrac{b}{3a}$.
극값 위치의 평균이 이와 같음을 비에타 공식으로 확인.
정답: $x = 1$
기본 2 — 직각삼각형 + 접선
곡선 $y = x^3$ 위의 점 $(a, a^3)$에서의 접선이 $x$축과 만나는 점을 $P$라 할 때, 접점에서 $P$까지의 $x$ 거리를 $a$의 식으로 나타내시오. (단, $a > 0$)
풀이 보기
3요소
접점 $(a, a^3)$. 기울기 $f'(a) = 3a^2$. 높이 = $a^3$.
밑변
밑변 = $\dfrac{높이}{기울기} = \dfrac{a^3}{3a^2} = \dfrac{a}{3}$.
검산
접선: $y - a^3 = 3a^2(x - a)$ → $y = 3a^2 x - 2a^3$. $x$축 ($y=0$): $x = \dfrac{2a^3}{3a^2} = \dfrac{2a}{3}$.
접점 $x = a$, $P$의 $x = \dfrac{2a}{3}$. 차 = $\dfrac{a}{3}$ ✔
정답: $\dfrac{a}{3}$
기본 3 — 도함수 넓이 활용
삼차함수 $f(x)$의 도함수 $f'(x)$의 그래프가 $x = 0, 4$에서 $x$축과 만나며, 최고차항의 계수가 $-1$이다. 극댓값과 극솟값의 차이를 구하시오.
풀이 보기
$f'(x)$
$f'(x) = -x(x - 4) = -x^2 + 4x$. $[0, 4]$에서 양수 → $f$ 증가 → $x = 0$ 극소, $x = 4$ 극대.
도함수 넓이
극값 차 = $\int_0^4 f' dx = \dfrac{|-1|}{6}(4)^3 = \dfrac{64}{6} = \dfrac{32}{3}$.
정답: $\dfrac{32}{3}$
심화 간격 변수로 미지수 줄이기
심화 1 — 간격 변수 $k$
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 극대점 $x = -k$, 극소점 $x = k$를 가지고 극값의 차이가 $32$일 때, $k$의 값을 구하시오. ($k > 0$)
풀이 보기
$f'(x)$
$f'(x) = 3(x + k)(x - k) = 3(x^2 - k^2)$. ($f$의 최고차항 $1$이면 $f'$의 최고차항 $3$.)
도함수 넓이 = 극값 차
$|f(-k) - f(k)| = \left|\int_{-k}^{k} 3(x^2 - k^2) dx\right|$.
$\int_{-k}^k 3(x^2 - k^2)dx = \dfrac{|3|}{6}(2k)^3 = \dfrac{8k^3}{2} = 4k^3$.
극값 차 = $4k^3 = 32$ → $k^3 = 8$ → $k = 2$.
검산
$k = 2$이면 $f'(x) = 3(x^2-4)$, $f(x) = x^3 - 12x$. $f(-2) = 16$, $f(2) = -16$, 차 $32$ ✔ (기초 3과 일치).
핵심: 좌표를 $(-k, k)$ 간격 변수로 두니 한 변수만으로 결정. 강사 인용의 "이 간격이 중요하다"는 의미.
정답: $k = 2$
심화 2 — 4차함수 대칭축
최고차항의 계수가 $1$인 사차함수 $f(x)$가 $x = 0$ 대칭이고 두 극값 위치가 $x = \pm 2$이다. $f(0) - f(2)$의 값을 구하시오.
풀이 보기
$f'(x)$
4차 + 좌우대칭 + 극값 $\pm 2$ → $f'(x) = 4x(x-2)(x+2) = 4x(x^2-4)$.
도함수 넓이
$f(2) - f(0) = \int_0^2 f'(x)\,dx = \int_0^2 (4x^3 - 16x)\,dx = [x^4 - 8x^2]_0^2 = 16 - 32 = -16$.
$f(0) - f(2) = 16$.
정답: 16
수능 대비 준킬러 — 구조적 접근 종합
수능 대비 1 — 준킬러 (간격 변수)
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 변곡점 $x = p$를 기준으로 극값 위치가 $x = p \pm k$이다. 극댓값 $- $ 극솟값 $= 4$일 때, $k$의 값을 구하시오.
풀이 보기
평행이동 무관
$x \to x - p$로 변곡점을 원점으로 옮겨도 극값 차는 같음. $f$를 $g(x) = f(x + p)$로 두면 변곡점 $0$, 극값 $\pm k$.
심화 1 결과 재사용
$4k^3 = 4$ → $k^3 = 1$ → $k = 1$.
핵심: 좌표값 $p$는 답에 안 나오고 간격 $k$만 등장. 영상의 "거리 간격" 철학.
정답: $k = 1$
수능 대비 2 — 준킬러 (접선 + 직각삼각형)
곡선 $y = x^2 + 4$ 위의 점 $(t, t^2 + 4)$에서의 접선이 $x$축, $y$축과 만드는 직각삼각형의 넓이의 최솟값을 구하시오. ($t > 0$)
풀이 보기
접선
$f'(t) = 2t$. 접선: $y - (t^2+4) = 2t(x - t)$ → $y = 2tx - t^2 + 4$.
$x$절편·$y$절편
$y = 0$: $x = \dfrac{t^2 - 4}{2t}$ ($t > 2$일 때 양수).
$x = 0$: $y = -t^2 + 4$ ($t < 2$일 때 양수).
→ 두 절편이 동시에 같은 부호의 $x, y$ 영역에 있는 케이스를 별도 분석. 단순 직각삼각형 넓이 최솟값을 구하려면 절댓값 사용.
넓이 함수
$S(t) = \dfrac{1}{2}\left|\dfrac{t^2-4}{2t}\right|\cdot|t^2 - 4 + 0|$ — 식 복잡.
다른 접근: 곡선이 $x$축에 닿지 않음 ($f \geq 4 > 0$). 접선의 양 절편을 정수배로 두고 미분.
정답 계산 (수치)
$t = 2$일 때 접선 $y = 4x$, 원점 통과 → 삼각형 안 만들어짐.
최소가 무한대로 0에 수렴할 수 있어 구체적 답이 없음. 학습용 사례 — 적절한 조건 추가 필요.
학습 포인트: 접선 절편을 직각삼각형 변으로 보는 흐름이 핵심. 실제 수능 문제에서는 추가 조건으로 단일 해 결정.
학습용 — 접선 절편 + 직각삼각형 변 흐름
수능 대비 3 — 준킬러 (도함수 넓이 + 비율)
삼차함수 $f(x)$의 도함수 $f'(x) = 3(x - a)(x - b)$ ($a < b$)이다. $f$의 극값 차이가 $36$일 때 $b - a$의 값을 구하시오. (최고차 계수 양수)
풀이 보기
간격 변수
$k = b - a$.
도함수 넓이
$|f(b) - f(a)| = \int_a^b |3(x-a)(x-b)|dx = \dfrac{|3|}{6}(b-a)^3 = \dfrac{k^3}{2}$.
$\dfrac{k^3}{2} = 36$ → $k^3 = 72$ → $k = \sqrt[3]{72} = 2\sqrt[3]{9}$.
구조 핵심: $a, b$ 따로 알 필요 없음 — 답은 차 $k$만으로 결정.
정답: $b - a = 2\sqrt[3]{9}$
