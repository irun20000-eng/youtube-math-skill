⚡ 선수 학습 1분 환기
  • 방정식 실근 = 교점: $f(x) = t$의 실근 개수 = $y = f(x)$와 $y = t$ (수평선)의 교점 개수.
  • 좌극한·우극한: $\lim_{t\to c^-} g(t)$, $\lim_{t\to c^+} g(t)$. 둘과 $g(c)$가 모두 같아야 연속.
  • 극대·극소: $f'(x) = 0$이고 부호 바뀌는 점.
  • 3차함수 극값: 극대·극소 두 개 (또는 없음).
1교점 개수로 정의된 함수 $g(t)$

$y = f(x)$의 그래프와 수평선 $y = t$가 만나는 교점의 개수를 $g(t)$로 정의. 이 $g(t)$는 $f$의 극값을 경계로 불연속이 발생한다.

$$g(t) = (\text{방정식 } f(x) = t \text{의 서로 다른 실근의 개수})$$
1
"세워서 보기": $f$를 옆으로 내리지 말고, 수평선 $y = t$를 $t$축 방향으로 위아래 이동시킨다고 생각. $t$가 극값을 통과하는 순간 교점 개수가 점프.
2
극값에서 불연속: 수평선이 극댓값·극솟값과 정확히 같아지면 그 순간 "접하는" 교점이 생겨 개수가 달라짐.
3차함수 f와 수평선 y=t 의 교점 개수 g(t) y g(t) │ 극대 M │ │ ╱╲ 3├──● │ ╱ ╲ y=t (이동) │ ┊ │─╱────╲─── ─────── 2├ ●━━● (불연속 점프) │╱ ╲ │ ┊ ┊ ● ╲ 극소 m 1├──● ●── ╲╱ └──┴──┴──── t m M t < m: 교점 1개 t = m: 교점 2개 (접점 발생) ← 불연속 m < t < M: 교점 3개 t = M: 교점 2개 ← 불연속 t > M: 교점 1개
핵심: $g(t)$의 불연속점 = $f$의 극댓값 $M$, 극솟값 $m$. 거꾸로 $g(t)$의 불연속 정보를 보고 $f$의 극값을 역추적할 수 있다.
📌 영상 핵심 인용

"얘를 옆으로 내리는 게 아니라 얘를 세워. 그냥 약간 생각만 약간 바꾸면 되거든." — 수평선을 $t$축에 세워 시각화.

23차함수 극점 높이차 공식

3차함수의 극댓값과 극솟값의 차이(높이 차)를 적분 없이 즉시 구하는 공식.

최고차항 계수 $a$, 두 극점의 $x$좌표 차이 $l$ 일 때
$$(\text{극댓값}) - (\text{극솟값}) = \dfrac{|a|}{2}\,l^3$$
⚙
유도: $f'(x) = 3a(x - p)(x - q)$, $q - p = l$. 높이차 $= \left|\int_p^q f'(x)\,dx\right| = |3a|\cdot\dfrac{l^3}{6} = \dfrac{|a|l^3}{2}$.
도함수 넓이와 연결: 극값 차 = 도함수 $f'$가 두 극점 사이에서 $x$축과 이루는 넓이. (6분 공식의 $f'$ 버전.)
📌 높이차 공식 예시

$f(x) = x^3 - 3x$ (최고차 $a = 1$). 극점 $x = \pm 1$ → $l = 2$.

공식
높이차 $= \dfrac{|1|}{2}\cdot 2^3 = \dfrac{8}{2} = 4$.
검산
$f(-1) = -1 + 3 = 2$ (극대), $f(1) = 1 - 3 = -2$ (극소). 차 = $4$ ✔
강사 인용: "3차 함수에서 그 유용한 공식 중 하나가 높이 차이 공식이야... 2분의 절댓값 $a$에다가 $l$의 3승."
33차함수 2:1 비율 관계

3차함수의 극점을 지나는 수평선이 그래프와 다시 만나는 교점까지의 거리 비.

극점(극대 또는 극소)을 지나는 수평선이 곡선과 다시 만나는 점까지의 $x$ 거리
= 두 극점 사이 거리($l$)의 2배 만큼 반대편.
→ 접점 ↔ 다른 극점 ↔ 교점의 $x$ 거리 비 = $2 : 1$... 정확히는 $1 : 2$ 패턴
3차함수 극대점 지나는 수평선 y │ 극대 ●━━━━━━━━━━━━● ← 교점 (다시 만남) │ ╱ ╱ │ ╱ ╲ ╱ │ ╱ ╲ ╱ │ ╱ 극소 ●╱ │ ╱ ● ◀── l ──▶◀──── 2l ────▶ 극대 극소 교점 극대에서 극소까지 거리 = l 극소를 지나 교점까지 거리 = 2l → 비율 1 : 2 (또는 관점에 따라 2:1)
왜 2:1인가: 3차함수는 변곡점 점대칭. 극대점을 지나는 수평선이 곡선과 만나는 다른 점은, 변곡점 대칭성과 인수분해 구조로 항상 극점 간격의 2배 위치에 놓인다.
📌 2:1 비율 예시

$f(x) = x^3 - 3x$. 극대점 $(-1, 2)$를 지나는 수평선 $y = 2$가 곡선과 다시 만나는 점을 구하라.

방정식
$x^3 - 3x = 2$ → $x^3 - 3x - 2 = 0$ → $(x+1)^2(x-2) = 0$.
근
$x = -1$ (이중근, 극대점 접함), $x = 2$ (다른 교점).
비율
극대 $x=-1$, 극소 $x=1$ (거리 $l = 2$). 교점 $x = 2$는 극대에서 거리 $3 = 1.5 l$... 극소($x=1$)에서 교점($x=2$)까지 $1$, 극대($-1$)에서 극소($1$)까지 $2$ → 비 $2:1$ ✔
주의: "2:1"의 기준점이 문제마다 다를 수 있음. 극대↔극소 거리와 극점↔교점 거리의 비를 정확히 따질 것. 핵심은 항상 정수비로 떨어진다는 점.
기초 높이차 공식·교점 개수
기초 1 — 높이차 공식
$f(x) = x^3 - 12x$의 극댓값과 극솟값의 차이를 높이차 공식으로 구하시오.
풀이 보기
극점
$f'(x) = 3x^2 - 12 = 3(x-2)(x+2)$. 극점 $x = \pm 2$ → $l = 4$. $a = 1$.
공식
높이차 $= \dfrac{|1|}{2}\cdot 4^3 = \dfrac{64}{2} = 32$.
정답: 32
기초 2 — 교점 개수
$f(x) = x^3 - 3x$에 대해 방정식 $f(x) = t$가 서로 다른 세 실근을 가질 $t$의 범위를 구하시오.
풀이 보기
극값
극대 $f(-1) = 2$, 극소 $f(1) = -2$.
세 실근 조건
수평선 $y = t$가 극솟값과 극댓값 사이에 있을 때 교점 3개.
$-2 < t < 2$.
정답: $-2 < t < 2$
기초 3 — 2:1 비율
$f(x) = x^3 - 3x$의 극솟값을 지나는 수평선이 곡선과 다시 만나는 점의 $x$좌표를 구하시오.
풀이 보기
극솟값
극소 $(1, -2)$. 수평선 $y = -2$.
방정식
$x^3 - 3x = -2$ → $x^3 - 3x + 2 = 0$ → $(x-1)^2(x+2) = 0$.
$x = 1$ (이중근), $x = -2$ (다른 교점).
2:1 확인
극소 $x=1$, 극대 $x=-1$ (거리 2), 교점 $x=-2$는 극대에서 거리 1. 비 $2:1$ ✔
정답: $x = -2$
기본 높이차·교점 결합
기본 1 — 높이차로 미지수
최고차항의 계수가 $2$인 삼차함수 $f(x)$의 두 극점의 $x$좌표 차이가 $3$일 때, 극댓값과 극솟값의 차이를 구하시오.
풀이 보기
공식
$a = 2$, $l = 3$. 높이차 $= \dfrac{|2|}{2}\cdot 3^3 = 1 \cdot 27 = 27$.
정답: 27
기본 2 — 교점 개수 함수 불연속
$f(x) = x^3 - 3x^2$에 대해 $g(t)$를 방정식 $f(x) = t$의 서로 다른 실근의 개수라 하자. $g(t)$가 불연속인 $t$의 값을 모두 구하시오.
풀이 보기
극값
$f'(x) = 3x^2 - 6x = 3x(x-2)$. 극대 $f(0) = 0$, 극소 $f(2) = 8 - 12 = -4$.
불연속점
$g(t)$는 극값에서 불연속. $t = 0$ (극대), $t = -4$ (극소).
정답: $t = 0,\; t = -4$
기본 3 — 높이차 + 교점
최고차항의 계수가 $1$인 삼차함수 $f(x)$의 극댓값이 $5$, 극솟값이 $1$이다. 두 극점의 $x$좌표 차이 $l$을 구하시오.
풀이 보기
높이차
극댓값 $-$ 극솟값 $= 5 - 1 = 4$.
공식 역산
$\dfrac{|1|}{2}l^3 = 4$ → $l^3 = 8$ → $l = 2$.
정답: $l = 2$
심화 교점 개수 함수 역추적
심화 1 — $g(t)$로 극값 역추적
삼차함수 $f(x)$ (최고차 계수 1)에 대해 $g(t)$ = ($f(x) = t$의 실근 개수)가 $t = 6$과 $t = 2$에서만 불연속이다. 두 극점의 $x$좌표 차이 $l$을 구하시오.
풀이 보기
불연속 = 극값
$g(t)$ 불연속점 = $f$의 극댓값 $6$, 극솟값 $2$.
높이차
$6 - 2 = 4 = \dfrac{|1|}{2}l^3$ → $l^3 = 8$ → $l = 2$.
정답: $l = 2$
심화 2 — 좌극한·우극한 합
삼차함수 $f(x) = x^3 - 3x$에 대해 $g(t)$ = ($f(x) = t$의 실근 개수)라 하자. $\displaystyle\lim_{t \to 2^-} g(t) + \lim_{t \to 2^+} g(t) + g(2)$의 값을 구하시오.
풀이 보기
$t = 2$ = 극댓값
극대 $f(-1) = 2$. $t = 2$ 근처 분석.
좌극한 ($t \to 2^-$)
$t$가 $2$보다 약간 작음 ($-2 < t < 2$) → 교점 3개. $\lim_{t\to 2^-} g(t) = 3$.
우극한 ($t \to 2^+$)
$t > 2$ → 교점 1개. $\lim_{t\to 2^+} g(t) = 1$.
$g(2)$
$t = 2$ 정확히 = 극댓값 → 접점 1 + 다른 교점 1 = 2개. $g(2) = 2$.
합
$3 + 1 + 2 = 6$.
정답: 6
수능 대비 준킬러 — 종합
수능 대비 1 — 준킬러 (높이차 + 2:1)
최고차항의 계수가 $1$인 삼차함수 $f(x)$의 극대점이 $(0, 4)$이고, 극대점을 지나는 수평선이 곡선과 다시 만나는 점의 $x$좌표가 $3$이다. 극솟값을 구하시오.
풀이 보기
2:1 비율
극대 $x = 0$, 교점 $x = 3$. 2:1 비율로 극소 $x$좌표 결정.
극대 → 극소 거리 $l$, 극소 → 교점 거리... 패턴: 교점은 극대에서 $\frac{3}{2}l$ 위치.
$f(x) - 4 = (x - 0)^2(x - 3) = x^2(x-3)$ (극대 접점 이중근 + 교점 단근).
$f(x)$
$f(x) = x^2(x-3) + 4 = x^3 - 3x^2 + 4$.
극점
$f'(x) = 3x^2 - 6x = 3x(x-2)$. 극대 $x=0$ ✔, 극소 $x = 2$. $l = 2$.
극솟값
$f(2) = 8 - 12 + 4 = 0$.
검산 (높이차): $4 - 0 = 4 = \dfrac{1}{2}\cdot 2^3$ ✔
정답: 극솟값 0
수능 대비 2 — 준킬러 (교점 개수 함수 + 미정)
최고차항의 계수가 $1$인 삼차함수 $f(x)$에 대해 $g(t)$ = ($f(x) = t$의 실근 개수)가 다음을 만족한다.

(가) $g(t)$는 $t = k$와 $t = k - 4$에서 불연속
(나) $g(0) = 2$

상수 $k$의 값을 구하시오.
풀이 보기
불연속 = 극값
극댓값 $k$, 극솟값 $k - 4$ (차이 $4$). 높이차 공식: $\dfrac{1}{2}l^3 = 4$ → $l = 2$ (참고용).
$g(0) = 2$
$g(t) = 2$가 되는 $t$ = 극값 그 자체 (접점 발생). 즉 $t = 0$이 극댓값 또는 극솟값.
$0 = k$ 또는 $0 = k - 4$.
→ $k = 0$ 또는 $k = 4$.
정합 검증
$k = 0$: 극댓값 $0$, 극솟값 $-4$. $g(0) = 2$ ✔ (극대에서 접).
$k = 4$: 극댓값 $4$, 극솟값 $0$. $g(0) = 2$ ✔ (극소에서 접).
두 경우 모두 가능. 추가 조건 없으면 둘 다 답.
정답: $k = 0$ 또는 $k = 4$
수능 대비 3 — 준킬러 (높이차 + 넓이)
최고차항의 계수가 $1$인 삼차함수 $f(x)$의 두 극점의 $x$좌표가 $1, 3$이다. 도함수 $f'(x)$의 그래프와 $x$축으로 둘러싸인 부분의 넓이를 구하고, 그것이 극값 차이와 같음을 확인하시오.
풀이 보기
$f'(x)$
극점 $1, 3$ → $f'(x) = 3(x-1)(x-3)$ (최고차 $3$, $f$가 최고차 $1$이므로).
넓이 (6분 공식)
$\left|\int_1^3 3(x-1)(x-3)dx\right| = \dfrac{|3|}{6}(3-1)^3 = \dfrac{1}{2}\cdot 8 = 4$.
높이차 공식 검증
$l = 3 - 1 = 2$, $a = 1$. 높이차 $= \dfrac{1}{2}\cdot 2^3 = 4$ ✔ — 도함수 넓이와 일치.
핵심 통합: "도함수 넓이 = 극값 차 = $\frac{|a|}{2}l^3$" 세 표현이 모두 같음.
정답: 넓이 = 극값 차 = 4
