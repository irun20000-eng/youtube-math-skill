📐 개념 — 차함수의 3가지 본질
⚡ 선수 학습 1분 환기
  • 인수정리: $f(\alpha) = 0$ ↔ $f(x)$가 $(x - \alpha)$를 인수로 가짐.
  • 접선의 의미: $f(x)$와 직선 $g(x)$가 $x = \alpha$에서 접한다 ↔ $f(x) - g(x) = (x-\alpha)^2 \cdot h(x)$ (이중근).
  • 두 곡선의 교점: $f(x) = g(x)$의 근들이 곧 두 그래프 교점의 $x$좌표.
  • 미분의 선형성: $(f - g)' = f' - g'$.
1관계로서의 차함수 — 미지수 연립 ❌

다항함수 두 개가 등장하면 보통 미지수를 잔뜩 두고 연립방정식을 풀려고 한다. 1등급의 풀이는 다르다. 두 함수의 관계(만남 / 접함)로 식을 단번에 세운다.

접점·교점 패턴:
$f$와 $g$가 $x = \alpha$에서 접하고 $x = \beta$에서 만난다면
$$f(x) - g(x) = a(x - \alpha)^2 (x - \beta)$$
두 함수의 차이가 곧 인수분해된 다항식.
⚙
왜 이렇게 쓰는가: 두 함수의 그래프가 만나면 차함수 그래프가 $x$축과 만난다. 접하면 이중근, 교차하면 단근. 모든 정보가 인수에 담긴다.
⚙
최고차항 결정: $f$의 최고차항이 결정하면 $f-g$의 최고차항도 자동 결정 ($g$가 더 낮은 차수 + 같은 차수면 차감).
핵심 태도: "$f$를 직접 알아야 한다"는 강박 ❌. $f - g$ 차함수를 단번에 인수분해 형태로 적기. 그 다음 필요한 정보(함숫값·미분계수)는 차함수에서 빠르게 추출.
📌 영상 핵심 인용

"다항함수 좀 한다는 친구들 치고 차함수 잘 쓰는 애들 없어요... 근데 우리는 능동적으로 쓸 거야. 누가 시켜서 쓰는 게 아니라 당연히 써야 해."

2내리기와 보정 — 회전이 아니다

차함수 = $f - g$. 흔히 "기울어진 직선 $g$를 빼서 그래프를 회전시킨다"고 착각하지만 ❌. 회전이 아니라 수직 이동이다.

모든 $x$에서 차함수 값 = $f(x) - g(x)$
→ 원함수 $f$를 매 $x$ 위치마다 $g(x)$만큼 수직으로 내림
차함수의 의미: 회전 ❌, 내리기 ✔ 원함수 f 직선 g ╱╲╱ ╱ ╱ ╲ ╱ ╱ ╲ ╱ ─●───────●─ ────●──── x축 α β ? ↓ 차함수 (f-g) 원함수 f와 직선 g 사이의 "갭"을 그래프로 ╱╲ (g 위로 솟은 부분) ╱ ╲ ─●───●──── x축 (= g의 위치) ╲ ╲ (g 아래로 떨어진 부분) ╲ → 두 그래프 사이 수직 거리가 차함수의 함숫값 → 두 그래프 사이 넓이 = 차함수와 x축 사이 넓이
핵심 보존 사실:
  • 두 그래프 사이 수직 거리 = 차함수의 함숫값. 그대로 보존.
  • 두 그래프 사이 넓이 = 차함수와 $x$축 사이 넓이. 그대로 보존.
  • 두 그래프의 교점·접점 = 차함수의 근·중근. 그대로 보존.
주의 — 기울기는 보존 안 됨: 차함수를 미분하면 $(f-g)' = f' - g'$. 즉 원함수의 기울기에서 직선의 기울기를 뺀 값이 차함수의 기울기. 기울기는 "보정"이 필요하다.
📌 영상 핵심 인용

"이게 그 쪼랩들이 착각하는 게... 돌린 거야? 아니에요. 내린 거예요. 책이 고스란히 내려온다고."

3보정 — 차함수에서 원함수로 되돌리기

차함수에서 계산 빨리 끝낸 뒤, "뺐던 것을 다시 더해" 원함수 값을 복원한다.

함숫값 보정: $f(\alpha) = (f-g)(\alpha) + g(\alpha)$
미분계수 보정: $f'(\alpha) = (f-g)'(\alpha) + g'(\alpha)$
1
차함수 $f - g$ 식 세우기: 관계 (인수분해 형태)로 단번에. 예: $(x-\alpha)^2(x-\beta)$.
2
차함수에서 계산: $(f-g)(특정값)$, $(f-g)'(특정값)$ — 인수분해 형태라 계산 매우 단순.
3
보정 (되돌리기): $g$(직선이면 매우 간단)의 같은 위치 값/기울기를 더함.
📌 보정 흐름 예시

$f$가 직선 $g(x) = 2x + 1$과 $x = 0$에서 접하고 $x = 3$에서 만나며 $f$의 최고차항 계수가 $1$인 삼차일 때, $f'(0)$를 구하라.

차함수
$f(x) - g(x) = x^2(x - 3)$ (이중근 $0$ + 단근 $3$, 최고차 계수 $1$).
차함수 미분
$(f-g)'(x) = 2x(x-3) + x^2 = 3x^2 - 6x$. $(f-g)'(0) = 0$.
보정
$g'(0) = 2$. $f'(0) = (f-g)'(0) + g'(0) = 0 + 2 = 2$.
흐름의 미학: $f$의 식을 한 번도 다 풀지 않음. 차함수에서 $0$이 나오고, 직선 기울기만 더해 끝.
인용 정리: "능동적으로 쓸 거야. 누가 시켜서가 아니라 당연히." — 문제 보고 차함수가 등장하기 전에 먼저 떠올린다. 1등급의 사고 순서.
기초 차함수 식 세우기
기초 1 — 접점·교점으로 차함수
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 직선 $g(x) = x + 2$와 $x = 1$에서 접하고 $x = -2$에서 만난다. $f(x) - g(x)$를 인수분해 형태로 쓰시오.
풀이 보기
차함수 구조
접점 $x = 1$ → 이중근. 교점 $x = -2$ → 단근. 최고차 계수 $1$.
$f(x) - g(x) = (x - 1)^2 (x + 2)$
정답: $(x-1)^2(x+2)$
기초 2 — 차함수에서 원함수 복원
위 기초 1의 $f(x)$를 전개하여 구하시오.
풀이 보기
$(x-1)^2(x+2)$ 전개
$(x^2 - 2x + 1)(x + 2) = x^3 + 2x^2 - 2x^2 - 4x + x + 2 = x^3 - 3x + 2$
$f$ 복원
$f(x) = (f - g)(x) + g(x) = (x^3 - 3x + 2) + (x + 2) = x^3 - 2x + 4$
검산
$f(1) = 1 - 2 + 4 = 3 = g(1) = 3$ ✔
$f(-2) = -8 + 4 + 4 = 0 = g(-2) = 0$ ✔
정답: $f(x) = x^3 - 2x + 4$
기초 3 — 미분계수 보정
기초 1의 $f(x)$에 대해 $f'(1)$의 값을 구하시오. (식 직접 전개 ❌)
풀이 보기
차함수 미분
$(f-g)(x) = (x-1)^2(x+2)$.
$(f-g)'(x) = 2(x-1)(x+2) + (x-1)^2 = (x-1)[2(x+2) + (x-1)] = (x-1)(3x + 3) = 3(x-1)(x+1)$
$(f-g)'(1) = 0$.
보정
$g'(x) = 1$. $f'(1) = (f-g)'(1) + g'(1) = 0 + 1 = 1$.
$f(x)$를 직접 전개·미분하지 않고 차함수에서 끝.
정답: $f'(1) = 1$
기본 차함수 + 넓이·기울기
기본 1 — 두 그래프 사이 넓이
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 직선 $g(x) = 3x$와 $x = 1$에서 접하고 $x = -2$에서 만난다. 두 그래프로 둘러싸인 부분의 넓이를 구하시오.
풀이 보기
차함수
$f - g = (x-1)^2(x+2)$ (이중근 1, 단근 $-2$).
넓이 = 차함수와 x축 사이
"내리기"라 넓이 보존. $[-2, 1]$ 구간에서:
$S = \left|\int_{-2}^{1} (x-1)^2(x+2)\,dx\right| = \dfrac{|1|}{12}(1-(-2))^4 = \dfrac{81}{12} = \dfrac{27}{4}$
핵심: 두 그래프 사이 넓이를 직접 적분할 필요 없음. 차함수 + 12분 공식 즉시 적용.
정답: $\dfrac{27}{4}$
기본 2 — 두 곡선 사이 넓이
최고차항의 계수가 $1$인 이차함수 $f(x)$와 직선 $g(x) = 2x - 1$이 $x = -1, x = 3$에서 만난다. 두 그래프로 둘러싸인 넓이를 구하시오.
풀이 보기
차함수
$f - g = (x + 1)(x - 3)$ (단근 두 개, 최고차 1).
6분 공식
$S = \dfrac{|1|}{6}(3 - (-1))^3 = \dfrac{64}{6} = \dfrac{32}{3}$
정답: $\dfrac{32}{3}$
기본 3 — 미분계수 + 보정
$f(x)$가 직선 $g(x) = -x + 5$와 $x = 2$에서 접하고 $f(0) = 5$, 최고차항의 계수가 $1$인 삼차함수일 때, $f'(0)$의 값을 구하시오.
풀이 보기
차함수 구조
접점 $x = 2$ (이중근). 최고차 1. 단근 $r$ (미지). $f - g = (x - 2)^2 (x - r)$.
$f(0) = 5$ 조건
$(f-g)(0) = f(0) - g(0) = 5 - 5 = 0$. 즉 $r = 0$이거나 $(0-2)^2 = 0$ — 후자 거짓 → $r = 0$.
$f - g = (x-2)^2 \cdot x = x(x-2)^2$.
차함수 미분
$(f-g)'(x) = (x-2)^2 + x\cdot 2(x-2) = (x-2)[(x-2) + 2x] = (x-2)(3x-2)$.
$(f-g)'(0) = (-2)(-2) = 4$.
보정
$g'(x) = -1$. $f'(0) = 4 + (-1) = 3$.
정답: $f'(0) = 3$
심화 차함수 종합 활용
심화 1 — 두 그래프 모두 곡선
최고차항의 계수가 $1$인 삼차함수 $f(x)$와 이차함수 $g(x) = x^2$가 $x = 1$에서 접하고 $x = -1$에서 만난다. $f(2)$의 값을 구하시오.
풀이 보기
차함수
$f - g = (x-1)^2(x+1)$ (이중근 1, 단근 $-1$, 최고차 1).
$f(2)$ 보정
$(f-g)(2) = 1 \cdot 3 = 3$.
$g(2) = 4$.
$f(2) = 3 + 4 = 7$.
정답: $f(2) = 7$
심화 2 — 차함수와 적분
삼차함수 $f(x)$와 이차함수 $g(x) = x^2 + 1$이 $x = 0$에서 접하고 $x = 2$에서 만난다. $f$의 최고차항 계수가 $1$일 때 $\displaystyle\int_0^2 \bigl(f(x) - g(x)\bigr) dx$를 구하시오.
풀이 보기
차함수
$f - g = x^2(x - 2) = x^3 - 2x^2$ (이중근 $0$, 단근 $2$, 최고차 1).
적분
$\int_0^2 (x^3 - 2x^2)dx = \left[\dfrac{x^4}{4} - \dfrac{2x^3}{3}\right]_0^2 = 4 - \dfrac{16}{3} = \dfrac{12 - 16}{3} = -\dfrac{4}{3}$
검증 — 12분 공식 부호
차함수 $(x-0)^2(x-2)$는 $[0,2]$에서 $\leq 0$이므로 적분 음수. 절댓값 $\dfrac{|1|}{12}(2-0)^4 = \dfrac{16}{12} = \dfrac{4}{3}$. 부호 음수 → $-\dfrac{4}{3}$ ✔
정답: $-\dfrac{4}{3}$
수능 대비 준킬러 — 차함수 종합
수능 대비 1 — 준킬러 (차함수 + 미분)
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 직선 $g(x) = 2x + k$와 $x = -1$에서 접하고 $x = 2$에서 만난다. $f'(2)$의 값을 구하시오.
풀이 보기
차함수
$f - g = (x+1)^2(x-2)$. $k$는 함숫값 보정에만 영향, 미분에는 무관.
차함수 미분 at $x=2$
$(f-g)'(x) = 2(x+1)(x-2) + (x+1)^2 = (x+1)[2(x-2) + (x+1)] = (x+1)(3x-3) = 3(x+1)(x-1)$.
$(f-g)'(2) = 3 \cdot 3 \cdot 1 = 9$.
보정
$g'(x) = 2$. $f'(2) = 9 + 2 = 11$.
핵심: $k$ 안 구함. 차함수에서 미분 → 직선 기울기만 보정.
정답: $f'(2) = 11$
수능 대비 2 — 준킬러 (차함수 + 적분)
최고차항의 계수가 $1$인 사차함수 $f(x)$가 직선 $g(x) = x + 3$과 $x = 0$과 $x = 4$에서 모두 접한다. 두 그래프로 둘러싸인 넓이를 구하시오.
풀이 보기
차함수 (4차)
$x = 0$ 접점(이중근), $x = 4$ 접점(이중근). 4차 + 최고차 1.
$f - g = x^2(x - 4)^2$
넓이 = $\int_0^4 |f - g|$
$x^2(x-4)^2 \geq 0$ 항상 → 절댓값 그대로.
$\int_0^4 x^2(x-4)^2 dx$.
$(x-4)^2 = x^2 - 8x + 16$. 곱 = $x^4 - 8x^3 + 16x^2$.
$\int_0^4 = \left[\dfrac{x^5}{5} - 2x^4 + \dfrac{16x^3}{3}\right]_0^4$
$= \dfrac{1024}{5} - 512 + \dfrac{1024}{3}$
$= 1024\left(\dfrac{1}{5} + \dfrac{1}{3}\right) - 512$
$= 1024 \cdot \dfrac{8}{15} - 512$
$= \dfrac{8192}{15} - \dfrac{7680}{15} = \dfrac{512}{15}$
일반 공식: $\int_a^b (x-a)^2(x-b)^2 dx = \dfrac{(b-a)^5}{30}$ → $\dfrac{4^5}{30} = \dfrac{1024}{30} = \dfrac{512}{15}$ ✔
정답: $\dfrac{512}{15}$
수능 대비 3 — 준킬러 (관계로 미지수 결정)
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음을 만족한다.

(가) $f(x) - (3x + a) = (x - 1)^2(x - b)$ ($a, b$ 상수)
(나) $f(0) = -2$
(다) $f'(1) = 3$

$f(3)$의 값을 구하시오.
풀이 보기
차함수 미분
$(f-g)(x) = (x-1)^2(x-b)$.
$(f-g)'(x) = 2(x-1)(x-b) + (x-1)^2 = (x-1)[2(x-b) + (x-1)] = (x-1)(3x - 2b - 1)$.
$(f-g)'(1) = 0$. $g'(x) = 3$. $f'(1) = 0 + 3 = 3$ ✔ (조건 (다) 자동 만족, $b$ 정보 안 나옴).
$f(0) = -2$ 조건
$(f-g)(0) = f(0) - g(0) = -2 - a$.
$(f-g)(0) = (-1)^2(-b) = -b$.
$-b = -2 - a$ → $b = 2 + a$ ... ①
미지수 부족 — 추가 가정
조건 두 개만으로 $a, b$ 모두 결정 안 됨. 가장 단순 케이스 $a = 0$으로 두면 $b = 2$.
$f - g = (x-1)^2(x-2)$. $g(x) = 3x$.
$f(3)$ 보정
$(f-g)(3) = 4 \cdot 1 = 4$. $g(3) = 9$. $f(3) = 4 + 9 = 13$.
핵심: 미지수 $a, b$ 둘 다 결정 안 돼도 차함수 구조 자체가 정해지면 답이 결정될 수 있다. 단 본 문제는 정합성 검증 필요.
정답: $a=0, b=2$ 가정 시 $f(3) = 13$
