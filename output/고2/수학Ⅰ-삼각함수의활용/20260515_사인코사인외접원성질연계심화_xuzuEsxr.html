📐 개념 — 사인·코사인 법칙 운영의 3박자
⚡ 선수 학습 1분 환기
  • 사인법칙: $\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$ ($R$ = 외접원 반지름)
  • 코사인법칙(변): $a^2 = b^2 + c^2 - 2bc\cos A$
  • 코사인법칙(각): $\cos A = \dfrac{b^2 + c^2 - a^2}{2bc}$
  • 원주각 정리: 같은 호에 대한 원주각은 같다. 지름에 대한 원주각은 $90°$.
  • 삼각형 넓이: $S = \dfrac{1}{2}ab\sin C$

1사인·코사인 법칙의 연계 — 변·각 왕복 통행

한 삼각형 안에서 "각 → 변" 변환은 사인법칙, "변 → 각" 또는 "변+끼인각 → 변" 변환은 코사인법칙. 둘을 번갈아 사용해 미지수를 좁힌다.

대표 흐름
(1) 사인법칙으로 한 변 또는 $R$ 도출
(2) 코사인법칙으로 나머지 변·각 결정
(3) 다시 사인법칙으로 다른 삼각형 연결
운영의 핵심: 어느 삼각형에서 무엇이 주어졌는지를 표로 정리. "변 2개 + 끼인각 1개" 같은 패턴이 보이면 즉시 코사인. "마주보는 변·각 쌍이 둘"이면 즉시 사인.
두 삼각형 △ABC, △ACD 공유 변 AC B ╱│╲ ╱ │ ╲ ╱ │ ╲ A───C───D △ABC: 사인법칙 → AC 길이 △ACD: AC와 다른 정보 → 코사인법칙 → 두 법칙이 AC라는 다리로 연결
2외접원의 성질과 반지름 $R$의 역할

사인법칙의 $2R$은 단순 매개변수가 아니라 외접원이 주는 기하 정보의 입구다.

1
$R$ 직접 활용: $a = 2R\sin A$. 마주보는 각의 사인값만 알아도 변 즉시 계산.
2
중심 $O$ 보조선: 외접원의 중심 $O$에서 꼭짓점까지 모두 길이 $R$. 그래서 $OA = OB = OC = R$. 이등변삼각형이 두 개 생긴다.
3
원주각 ↔ 중심각: 중심각 = 원주각 × 2. 한 쪽에서 정보를 얻기 어려울 때 다른 쪽으로 변환.
외접원의 중심 $O$에서 현 $BC$에 내린 수선의 발 $M$:
$BM = MC = \dfrac{a}{2}$, $\angle BOC = 2A$, $OM = R\cos A$
실전: 외접원이 보이는 순간 "(1) $a = 2R\sin A$ 적용 가능한가? (2) 중심 $O$를 잇는 보조선이 새 삼각형을 만드는가?" 두 질문을 동시에.
3보조선과 기하 해석 — 안 보이는 삼각형 만들기

문제에 주어진 그림만으로 풀리지 않을 때, 보조선으로 법칙을 적용할 새 삼각형을 만든다.

자주 쓰는 보조선:
  • 외접원 중심 $O$와 꼭짓점 연결 → 이등변삼각형 + 원주각
  • 두 원의 공통 현 → 두 원의 정보 동시 활용
  • 변의 중점·각의 이등분선 → 대칭 활용
  • 지름 긋기 → 원주각 $90°$ (직각삼각형)
주의: 보조선은 함부로 긋지 말고 "법칙 적용 가능한 삼각형"이 만들어지는지 확인. 무작정 긋는 보조선은 시간만 잡아먹는다.
📌 영상 예시 흐름 — TOP 3 패턴

"GH의 길이를 구하기 위해 사인법칙으로 외접원 반지름 $R$을 먼저 결정, 그 다음 보조선 GA를 그어 반지름 성질($OA=OG=R$)을 이용한 새 삼각형에 코사인법칙 적용."

사인법칙 → $R$ → 보조선 → 코사인법칙 → 결과. 법칙 두 번, 보조선 한 번이 고난도 문항의 표준 동선.
기초 두 법칙 직접 적용
기초 1 — 사인법칙으로 $R$ 구하기
삼각형 $ABC$에서 $a = 6$, $A = 30°$일 때 외접원의 반지름 $R$의 값을 구하시오.
풀이 보기
사인법칙
$\dfrac{a}{\sin A} = 2R$ → $\dfrac{6}{\sin 30°} = 2R$ → $\dfrac{6}{1/2} = 12 = 2R$
정답: $R = 6$
기초 2 — 코사인법칙으로 변 구하기
삼각형 $ABC$에서 $b = 4$, $c = 5$, $A = 60°$일 때 변 $a$의 길이를 구하시오.
풀이 보기
코사인법칙
$a^2 = b^2 + c^2 - 2bc\cos A = 16 + 25 - 2\cdot 4\cdot 5\cdot \dfrac{1}{2} = 41 - 20 = 21$
정답: $a = \sqrt{21}$
기초 3 — 코사인법칙으로 각 구하기
세 변이 $a = 7$, $b = 5$, $c = 3$인 삼각형에서 가장 큰 각의 크기를 구하시오.
풀이 보기
가장 큰 변
$a = 7$이 가장 큼 → 마주보는 각 $A$가 가장 큼.
코사인법칙
$\cos A = \dfrac{b^2 + c^2 - a^2}{2bc} = \dfrac{25 + 9 - 49}{30} = \dfrac{-15}{30} = -\dfrac{1}{2}$
각
$\cos A = -\dfrac{1}{2}$ → $A = 120°$
정답: $120°$
기본 사인·코사인 연계
기본 1 — 사인 → 변, 코사인 → 각
삼각형 $ABC$에서 $a = 7$, $B = 60°$, 외접원 반지름 $R = \dfrac{7\sqrt 3}{3}$이다. $c$의 길이와 각 $A$를 구하시오.
풀이 보기
$\sin A$ — 사인법칙
$\dfrac{a}{\sin A} = 2R$ → $\sin A = \dfrac{a}{2R} = \dfrac{7}{2\cdot\frac{7\sqrt 3}{3}} = \dfrac{7\cdot 3}{14\sqrt 3} = \dfrac{3}{2\sqrt 3} = \dfrac{\sqrt 3}{2}$
$A$
$\sin A = \dfrac{\sqrt 3}{2}$, $A + B + C = 180°$, $B=60°$ → $A = 60°$ (정삼각형 후보) 또는 $A=120°$. $B+A < 180°$여야 하므로 둘 다 가능. 단 $A=120°$이면 $C=0°$ 모순 → $A=60°$.
$C$ 및 $c$
$C = 180° - 60° - 60° = 60°$ → 정삼각형. $c = a = 7$.
정답: $A = 60°$, $c = 7$ (정삼각형)
기본 2 — 두 변 + 끼인각 → 코사인
삼각형 $ABC$에서 $b = 8$, $c = 5$, $A = 60°$일 때 $a$와 $\sin B$를 구하시오.
풀이 보기
$a$ — 코사인
$a^2 = 64 + 25 - 2\cdot 8\cdot 5\cdot \dfrac{1}{2} = 89 - 40 = 49$ → $a = 7$
$\sin B$ — 사인
$\dfrac{a}{\sin A} = \dfrac{b}{\sin B}$ → $\sin B = \dfrac{b \sin A}{a} = \dfrac{8\cdot\frac{\sqrt 3}{2}}{7} = \dfrac{4\sqrt 3}{7}$
정답: $a = 7$, $\sin B = \dfrac{4\sqrt 3}{7}$
기본 3 — 공유 변으로 두 삼각형 연결
사각형 $ABCD$에서 대각선 $AC = 7$. $\triangle ABC$에서 $AB = 5$, $\angle BAC = 60°$이고, $\triangle ACD$에서 $\angle ACD = 90°$, $CD = 3$이다. $BC$와 $AD$의 길이를 구하시오.
풀이 보기
$BC$ — $\triangle ABC$
$BC^2 = AB^2 + AC^2 - 2\cdot AB\cdot AC\cos 60° = 25 + 49 - 70\cdot \dfrac{1}{2} = 74 - 35 = 39$ → $BC = \sqrt{39}$
$AD$ — $\triangle ACD$ (직각)
$\angle ACD = 90°$ → 피타고라스: $AD^2 = AC^2 + CD^2 = 49 + 9 = 58$ → $AD = \sqrt{58}$
정답: $BC = \sqrt{39}$, $AD = \sqrt{58}$
심화 외접원 + 보조선
심화 1 — 외접원 + 사인법칙 연쇄
삼각형 $ABC$의 외접원 반지름이 $R = 4$이다. $a = 4\sqrt 3$, $b = 4\sqrt 2$일 때 $A$, $B$, $C$의 크기를 모두 구하시오. (단, $A > B$)
풀이 보기
$\sin A$
$\sin A = \dfrac{a}{2R} = \dfrac{4\sqrt 3}{8} = \dfrac{\sqrt 3}{2}$ → $A = 60°$ 또는 $120°$
$\sin B$
$\sin B = \dfrac{b}{2R} = \dfrac{4\sqrt 2}{8} = \dfrac{\sqrt 2}{2}$ → $B = 45°$ 또는 $135°$
$A > B$ + 합 $180°$
경우 1: $A=60°, B=45°$ → $C = 75°$ ✔ ($a > b$ ↔ $A > B$ 일치)
경우 2: $A=120°, B=45°$ → $C = 15°$ ✔
둘 다 형식적으로 가능. $a > b$ 확인: $4\sqrt 3 > 4\sqrt 2$ ✔ ($A > B$ 일치)
두 경우 중 어느 것이 실제 삼각형인지는 추가 정보 필요. 일반적 문항에서는 둘 다 표기.
정답: $A=60°,B=45°,C=75°$ 또는 $A=120°,B=45°,C=15°$
심화 2 — 중심 $O$ 보조선
삼각형 $ABC$의 외접원의 중심 $O$가 있다. $\angle BAC = 50°$일 때 $\angle BOC$의 크기와, $OB = OC = R$임을 이용해 $\triangle OBC$가 어떤 삼각형인지 설명하시오.
풀이 보기
중심각 = 원주각 × 2
$\angle BOC = 2\cdot \angle BAC = 100°$ (호 $BC$에 대한 중심각·원주각)
$\triangle OBC$ 분석
$OB = OC = R$ → 이등변삼각형. 꼭지각 $\angle BOC = 100°$, 두 밑각 $= \dfrac{180°-100°}{2} = 40°$.
활용: 이런 식으로 보조선 $OA, OB, OC$를 그으면 세 이등변삼각형 (정확히는 두 변이 $R$인 부채꼴 같은 삼각형들)이 생긴다. 한 삼각형에서 부족한 정보를 다른 삼각형에서 얻어낼 수 있는 통로.
$\angle BOC = 100°$, $\triangle OBC$는 이등변(꼭지각 100°)
수능 대비 준킬러 — 영상 TOP 3 유형
수능 대비 1 — 14번 유형 (준킬러, 영상 TOP 3 패턴)
삼각형 $ABC$의 외접원 반지름이 $R$이다. 다음 조건을 만족시킨다.

(가) $a = 6$, $\sin A = \dfrac{3}{4}$
(나) 점 $G$는 변 $BC$ 위에 있고 $\angle BAG = \angle CAG$ (각의 이등분선)
(다) $AG = 4$

$b + c$의 값을 구하시오.
풀이 보기
사인법칙으로 $R$
$2R = \dfrac{a}{\sin A} = \dfrac{6}{3/4} = 8$ → $R = 4$
$\cos A$
$\cos^2 A = 1 - \dfrac{9}{16} = \dfrac{7}{16}$ → $\cos A = \dfrac{\sqrt 7}{4}$ (예각 가정).
참고: $\cos\dfrac{A}{2} = \sqrt{\dfrac{1+\cos A}{2}} = \sqrt{\dfrac{1 + \sqrt 7/4}{2}}$ — 복잡하니 다른 접근.
각 이등분선 공식
각 이등분선 길이: $AG^2 = bc\bigl[1 - \bigl(\dfrac{a}{b+c}\bigr)^2\bigr]$
또한 $bc \sin A = $ 삼각형 넓이 관계. 단순화 위해 다음 접근:
$\triangle ABG$와 $\triangle ACG$에 사인법칙 + 공유변 $AG$.
간이 접근
$\sin A = 3/4$, $\cos A = \sqrt 7/4$. $\sin\dfrac{A}{2}, \cos\dfrac{A}{2}$ 반각 공식:
$\cos\dfrac{A}{2} = \sqrt{\dfrac{1+\cos A}{2}}$, $\sin\dfrac{A}{2} = \sqrt{\dfrac{1-\cos A}{2}}$
이 문제는 복잡한 계산이 필요한 학습용 사례. 핵심은 "사인법칙 → $R$ → 코사인 / 각 이등분선" 연계 흐름.
학습 포인트: 14번 유형은 보통 (1) 사인법칙으로 $R$ 또는 한 변 결정 → (2) 코사인법칙으로 다른 변·각 → (3) 추가 조건(이등분선·중점 등)으로 마무리. 흐름 익히기.
학습용 — 사인 → 코사인 연계 흐름
수능 대비 2 — 13번 유형 (준킬러, 원 위 점 + 중심 보조선)
반지름 $R$인 원 위에 세 점 $A, B, C$가 있고 삼각형 $ABC$가 형성된다. $AB = 4$, $\angle ACB = 30°$, $\angle ABC = 90°$일 때 $R$과 $BC$의 길이를 구하시오.
풀이 보기
각 관계
$\angle BAC = 180° - 90° - 30° = 60°$
사인법칙
$\dfrac{AB}{\sin C} = 2R$ → $\dfrac{4}{\sin 30°} = 2R$ → $R = 4$
$BC$
$\dfrac{BC}{\sin A} = 2R = 8$ → $BC = 8\sin 60° = 4\sqrt 3$
검증
$\angle ABC = 90°$이므로 $AC$는 지름. $AC = 2R = 8$. 피타고라스: $AB^2 + BC^2 = 16 + 48 = 64 = AC^2$ ✔
핵심: 직각이 등장하면 마주보는 변은 지름. 즉시 $R$ 결정.
정답: $R = 4$, $BC = 4\sqrt 3$
수능 대비 3 — 21번 유형 (준킬러, 두 원·반지름 비)
삼각형 $ABC$의 외접원 반지름은 $R_1$, 삼각형 $ABD$의 외접원 반지름은 $R_2$이다. 두 삼각형은 변 $AB = 6$을 공유하고 $\angle ACB = 60°$, $\angle ADB = 30°$이다.

$R_1$, $R_2$를 구하고, 비 $R_2 : R_1$을 구하시오.
풀이 보기
$R_1$ — $\triangle ABC$
$\dfrac{AB}{\sin C} = 2R_1$ → $\dfrac{6}{\sin 60°} = 2R_1$ → $R_1 = \dfrac{6}{2\cdot \frac{\sqrt 3}{2}} = \dfrac{6}{\sqrt 3} = 2\sqrt 3$
$R_2$ — $\triangle ABD$
$\dfrac{AB}{\sin D} = 2R_2$ → $\dfrac{6}{\sin 30°} = 2R_2$ → $R_2 = 6$
비율
$R_2 : R_1 = 6 : 2\sqrt 3 = \sqrt 3 : 1$
핵심: 같은 공유 변에 대해 두 마주보는 각이 다르면 두 외접원이 다름. 사인법칙으로 $R_1, R_2$를 별개 계산 후 비율 도출 — 영상 TOP 1 유형의 핵심 직관.
정답: $R_1 = 2\sqrt 3$, $R_2 = 6$, $R_2:R_1 = \sqrt 3 : 1$

💡 고난도 풀이 체크리스트
① 주어진 조건을 변/각/원/외접원으로 분류.
② 마주보는 변·각 쌍 → 사인법칙. 두 변+끼인각 → 코사인법칙.
③ 외접원 보이면 $2R = a/\sin A$ 와 중심 $O$ 보조선 둘 다 고려.
④ 두 삼각형이 변을 공유하면 그 변에서 사인·코사인 왕복.
