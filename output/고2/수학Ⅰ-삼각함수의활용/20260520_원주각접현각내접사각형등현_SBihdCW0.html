📐 개념 — 원 도형 4대 도구
⚡ 선수 학습 1분 환기
  • 원주각: 원 위의 한 점에서 호 $BC$를 바라본 각.
  • 중심각: 원의 중심 $O$에서 호 $BC$를 바라본 각.
  • 코사인법칙: $a^2 = b^2 + c^2 - 2bc\cos A$
  • 사인법칙: $\dfrac{a}{\sin A} = 2R$
  • 닮음: AA 닮음 — 두 각이 같으면 닮음. 대응변 비율 일정.
1원주각·중심각·접현각 — 같은 호에서 같은 각
① 원주각 정리: 같은 호 $BC$에 대한 원주각은 모두 같다.
② 중심각 = 원주각 × 2: 같은 호에 대한 중심각은 원주각의 두 배.
③ 접현각 정리: 접선과 현이 이루는 각 = 그 현의 반대쪽 원주각.
원주각 — 같은 호 같은 각 ●A ●A' ← 같은 호 BC 바라봄 ╲ ╱ ∠BAC = ∠BA'C ✔ ╲ ╱ ●B─────●C (호 BC) 중심각 = 원주각 × 2 A● ← 원주각 ∠BAC = α ╱╲ ╱ ╲ ╱ ╲ ╱ ●O ← 중심 ╱ ╱ ╲╲ 중심각 ∠BOC = 2α B●━━━━━━●C 접현각 — 접선과 현 T (접점에서 접선) ╱─→ ╱ ●A ╱ ╲ ●T───────●B ← 현 TB 접선과 현 TB 이루는 각 = ∠TAB (반대쪽 원주각)
실전 활용: 문제에 원이 보이고 같은 호 옆에 두 각이 표시되면, 즉시 "두 각 같다"고 인식. 접선이 보이면 접현각으로 현 반대쪽 각과 연결.
2원에 내접하는 사각형 — 대각의 합 = $\pi$
$ABCD$가 원에 내접 → $\angle A + \angle C = \angle B + \angle D = \pi$ ($180°$)
⚙
왜 그런가: $\angle A$는 호 $BCD$의 원주각, $\angle C$는 호 $BAD$의 원주각. 두 호 합 = 원 한 바퀴 $2\pi$. 따라서 원주각 합 = $\pi$.
⚙
코사인 활용: $\angle A + \angle C = \pi$ → $\cos C = -\cos A$. 두 삼각형에 코사인법칙을 양변 더하거나 빼면 미지수 한 번에 결정.
실전 신호: "원에 내접하는 사각형" 보이면 즉시 $\cos C = -\cos A$ 메모. 코사인법칙 두 번 + 연립.
원에 내접하는 사각형 ABCD A ╱╲╲ ╱ ╲ ╲ ╱ ╲ ╲ B──────●D ╲ ╱ ╱ ╲ ╱ ╱ ╲╱╱ C ∠A + ∠C = π ∠B + ∠D = π → cos C = -cos A → cos D = -cos B
3등현사각형 — 이웃한 두 변 같을 때

내접사각형에서 이웃한 두 변이 같은 경우 ($AB = AD$ 등) — 영상에서 "등현 공식"이라 불리는 패턴.

$ABCD$ 내접 + $AB = AD = m$, $BC = p$, $CD = q$일 때
대각선 $BD$를 $\triangle ABD$와 $\triangle CBD$ 양쪽에 코사인법칙:
$BD^2 = 2m^2 - 2m^2\cos A$ (← 이등변삼각형 $ABD$)
$BD^2 = p^2 + q^2 - 2pq\cos C$ (← $\triangle CBD$)
1
대각 관계 적용: $\cos C = -\cos A$. 위 두 식을 연립.
2
$2m^2 - 2m^2\cos A = p^2 + q^2 + 2pq\cos A$
$\cos A$에 대한 일차방정식 → 한 번에 결정.
3
$\cos A$ 알면 $BD$, $\sin A$, 외접원 반지름 $R = \dfrac{BD}{2\sin A}$ 등 줄줄이 결정.
📌 등현 공식 한 줄 정리

$AB = AD = m$, $BC = p$, $CD = q$인 내접사각형에서

$$\cos A = \dfrac{2m^2 - p^2 - q^2}{2(m^2 + pq)}$$
도출: 두 코사인식을 $\cos A$에 대해 정리하면 위 공식. 매번 도출 가능하지만 패턴을 알면 10초컷.
4나비 닮음 — 두 대각선이 만든 두 삼각형

내접사각형 $ABCD$의 두 대각선 $AC$, $BD$의 교점을 $P$라 할 때, $\triangle APB$와 $\triangle DPC$는 닮음 (나비 모양).

$\triangle APB \sim \triangle DPC$ (AA 닮음)
$\angle PAB = \angle PDC$ (호 $BC$의 원주각)
$\angle PBA = \angle PCD$ (호 $AD$의 원주각)
비례식: $\dfrac{PA}{PD} = \dfrac{PB}{PC} = \dfrac{AB}{DC}$
나비 닮음 (Butterfly) A● ●B ╲ ╱ ╲ ╱ ╲ ╱ ●P ← 대각선 교점 ╱ ╲ ╱ ╲ ╱ ╲ C● ●D △APB ∼ △DPC (마주보는 두 삼각형) ⇒ PA · PC = PB · PD (방멱 정리)
방멱 정리 (Power of Point): $PA \cdot PC = PB \cdot PD$. 한 점에서 원에 그은 두 현의 분할 비율로 확장.
기초 4대 도구 직접 적용
기초 1 — 원주각
원 위의 네 점 $A, B, C, D$에 대해 $\angle BAC = 40°$일 때, 같은 호 $BC$에 대한 원주각 $\angle BDC$의 크기를 구하시오.
풀이 보기
원주각
같은 호 → 같은 원주각. $\angle BDC = 40°$.
정답: $40°$
기초 2 — 중심각
원의 중심 $O$와 원 위의 두 점 $B, C$에 대해 호 $BC$의 중심각이 $120°$일 때, 호 $BC$에 대한 원주각의 크기를 구하시오.
풀이 보기
관계
중심각 = 원주각 × 2 → 원주각 = $\dfrac{120°}{2} = 60°$.
정답: $60°$
기초 3 — 내접사각형 대각합
원에 내접하는 사각형 $ABCD$에서 $\angle A = 75°$일 때 $\angle C$의 크기와 $\cos A + \cos C$의 값을 구하시오.
풀이 보기
대각합
$\angle A + \angle C = 180°$ → $\angle C = 105°$.
$\cos$ 관계
$\cos C = \cos(180° - 75°) = -\cos 75°$ → $\cos A + \cos C = 0$.
정답: $\angle C = 105°$, $\cos A + \cos C = 0$
기본 내접사각형 + 코사인 연립
기본 1 — 내접사각형 대각선
원에 내접하는 사각형 $ABCD$에서 $AB = 4$, $BC = 3$, $CD = 2$, $DA = 5$일 때 대각선 $AC$의 길이를 구하시오.
풀이 보기
$\triangle ABC$
$AC^2 = 16 + 9 - 2\cdot 4\cdot 3\cos B = 25 - 24\cos B$
$\triangle ACD$
$AC^2 = 4 + 25 - 2\cdot 2\cdot 5\cos D = 29 - 20\cos D$
대각합
$B + D = \pi$ → $\cos D = -\cos B$. 두 식 연립:
$25 - 24\cos B = 29 + 20\cos B$ → $-44\cos B = 4$ → $\cos B = -\dfrac{1}{11}$
$AC$
$AC^2 = 25 - 24\cdot(-1/11) = 25 + 24/11 = \dfrac{275 + 24}{11} = \dfrac{299}{11}$
$AC = \sqrt{299/11} = \dfrac{\sqrt{3289}}{11}$
핵심: 대각합 $\pi$로 $\cos D = -\cos B$ 만들면 두 식 연립 한 번에 $\cos B$ 결정.
정답: $AC = \dfrac{\sqrt{3289}}{11}$
기본 2 — 접현각
원 위의 두 점 $A, B$에서 접선과 현 $AB$가 만나는 각이 $\angle TAB = 35°$일 때, 호 $AB$의 반대쪽 점 $C$에서 $\angle ACB$의 크기를 구하시오.
풀이 보기
접현각
접선과 현이 이루는 각 = 현의 반대쪽 원주각. $\angle ACB = \angle TAB = 35°$.
정답: $35°$
기본 3 — 나비 닮음
원에 내접하는 사각형 $ABCD$의 대각선 $AC, BD$의 교점을 $P$라 하자. $PA = 2$, $PC = 6$, $PB = 3$일 때 $PD$의 길이를 구하시오.
풀이 보기
방멱 정리
$PA \cdot PC = PB \cdot PD$ → $2 \cdot 6 = 3 \cdot PD$ → $PD = 4$.
정답: 4
심화 등현사각형 공식
심화 1 — 등현사각형 cosA
원에 내접하는 사각형 $ABCD$에서 $AB = AD = 5$, $BC = 3$, $CD = 7$일 때 $\cos A$와 대각선 $BD$의 길이를 구하시오.
풀이 보기
등현 공식
$m = 5$, $p = 3$, $q = 7$.
$\cos A = \dfrac{2m^2 - p^2 - q^2}{2(m^2 + pq)} = \dfrac{50 - 9 - 49}{2(25 + 21)} = \dfrac{-8}{92} = -\dfrac{2}{23}$
$BD^2$ — $\triangle ABD$
$BD^2 = 2m^2(1 - \cos A) = 2\cdot 25\cdot(1 + 2/23) = 50\cdot\dfrac{25}{23} = \dfrac{1250}{23}$
검산 — $\triangle BCD$
$BD^2 = 9 + 49 - 2\cdot 3\cdot 7\cos C = 58 - 42\cos C$
$\cos C = -\cos A = 2/23$ → $BD^2 = 58 - 42\cdot 2/23 = 58 - 84/23 = (1334-84)/23 = 1250/23$ ✔
$BD$
$BD = \sqrt{1250/23} = \dfrac{25\sqrt 2}{\sqrt{23}} = \dfrac{25\sqrt{46}}{23}$
정답: $\cos A = -\dfrac{2}{23}$, $BD = \dfrac{25\sqrt{46}}{23}$
심화 2 — 외접원 반지름
위 심화 1의 등현사각형에서 외접원의 반지름 $R$을 구하시오.
풀이 보기
$\sin A$
$\cos A = -2/23$. $\sin^2 A = 1 - 4/529 = 525/529$ → $\sin A = \dfrac{\sqrt{525}}{23} = \dfrac{5\sqrt{21}}{23}$ ($A$는 둔각이지만 $\sin > 0$).
사인법칙
$\triangle ABD$에 사인법칙: $\dfrac{BD}{\sin A} = 2R$
$2R = \dfrac{25\sqrt{46}/23}{5\sqrt{21}/23} = \dfrac{25\sqrt{46}}{5\sqrt{21}} = \dfrac{5\sqrt{46}}{\sqrt{21}} = \dfrac{5\sqrt{966}}{21}$
$R = \dfrac{5\sqrt{966}}{42}$
흐름: 등현공식으로 $\cos A$ → $\sin A$ → 사인법칙으로 $R$. 모든 변·각이 줄줄이 결정.
정답: $R = \dfrac{5\sqrt{966}}{42}$
수능 대비 준킬러 — 4대 도구 종합
수능 대비 1 — 준킬러 (내접사각형 코사인)
원에 내접하는 사각형 $ABCD$에서 $AB = 2$, $BC = 2$, $CD = 3$, $DA = 5$일 때 $\cos B$의 값을 구하시오.
풀이 보기
$\triangle ABD$ (BD 대각선)
잠깐 — 대각선 $AC$로 가는 게 더 단순. $AB=BC=2$ 이웃변 같으니 $\triangle ABC$는 이등변.
$AC^2 = 4 + 4 - 2\cdot 2\cdot 2\cos B = 8(1 - \cos B)$
$\triangle ACD$
$AC^2 = 9 + 25 - 2\cdot 3\cdot 5\cos D = 34 - 30\cos D$
$\cos D = -\cos B$이므로 $AC^2 = 34 + 30\cos B$.
연립
$8 - 8\cos B = 34 + 30\cos B$ → $-38\cos B = 26$ → $\cos B = -\dfrac{13}{19}$
10초컷 흐름: 대각선 → 두 코사인식 → $\cos D = -\cos B$ → 연립.
정답: $\cos B = -\dfrac{13}{19}$
수능 대비 2 — 준킬러 (등현사각형 + 사인법칙)
원에 내접하는 사각형 $ABCD$가 $AB = AD = 4$, $BC = 2$, $CD = 6$이다. 외접원의 반지름 $R$을 구하시오.
풀이 보기
등현 공식
$m=4, p=2, q=6$. $\cos A = \dfrac{32 - 4 - 36}{2(16 + 12)} = \dfrac{-8}{56} = -\dfrac{1}{7}$.
$BD^2$
$BD^2 = 2\cdot 16\cdot(1 + 1/7) = 32\cdot 8/7 = 256/7$.
$\sin A$
$\sin^2 A = 1 - 1/49 = 48/49$ → $\sin A = \dfrac{\sqrt{48}}{7} = \dfrac{4\sqrt 3}{7}$.
사인법칙
$2R = \dfrac{BD}{\sin A} = \dfrac{\sqrt{256/7}}{4\sqrt 3/7} = \dfrac{16/\sqrt 7}{4\sqrt 3/7} = \dfrac{16 \cdot 7}{4\sqrt 3 \cdot \sqrt 7} = \dfrac{28}{\sqrt{21}} = \dfrac{28\sqrt{21}}{21} = \dfrac{4\sqrt{21}}{3}$
$R = \dfrac{2\sqrt{21}}{3}$
정답: $R = \dfrac{2\sqrt{21}}{3}$
수능 대비 3 — 준킬러 (방멱 + 사인법칙)
원 위의 네 점 $A, B, C, D$에서 두 현 $AC, BD$의 교점을 $P$라 한다. $AP = 3$, $PC = 8$, $BP = 4$일 때 $PD$와, $\angle APB = 60°$일 때 $AB^2 + CD^2$의 값을 구하시오.
풀이 보기
방멱 정리
$AP\cdot PC = BP\cdot PD$ → $3\cdot 8 = 4\cdot PD$ → $PD = 6$.
$AB^2$ — $\triangle APB$ ($\angle APB = 60°$)
$AB^2 = 9 + 16 - 2\cdot 3\cdot 4 \cos 60° = 25 - 12 = 13$.
$CD^2$ — $\triangle CPD$ ($\angle CPD = 60°$, 대각)
$\angle CPD = \angle APB = 60°$ (맞꼭지각).
$CD^2 = 64 + 36 - 2\cdot 8\cdot 6\cos 60° = 100 - 48 = 52$.
합
$AB^2 + CD^2 = 13 + 52 = \mathbf{65}$.
키: 방멱으로 $PD$ → 두 마주보는 삼각형 코사인법칙 → 합. 대각의 맞꼭지각이 같음을 활용.
정답: $PD = 6$, $AB^2 + CD^2 = 65$
