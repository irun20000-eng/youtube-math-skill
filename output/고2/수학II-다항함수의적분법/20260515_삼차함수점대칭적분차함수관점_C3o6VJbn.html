📐 개념 — 차함수 관점 + 변곡점 점대칭
⚡ 선수 학습 1분 환기
  • 삼차함수의 변곡점: $f(x)=ax^3+bx^2+cx+d$의 변곡점은 $f''(x)=0$인 점. $x = -\dfrac{b}{3a}$.
  • 점대칭(점 대칭): 모든 삼차함수는 변곡점에 대해 점대칭. 즉 $f(p+h) + f(p-h) = 2q$ (변곡점 $(p,q)$).
  • 정적분의 정의: $\displaystyle\int_a^b f(x)\,dx$ = "$x$축과 그래프 사이 부호 있는 넓이".
  • 평행이동: $g(x)=f(x-p)+q$ → $f$를 오른쪽 $p$, 위 $q$ 이동.
1차함수 관점 — "쪼개지 말고 묶어라"

꺾인 직선 $p(x)$와 곡선 $f(x)$가 동시에 등장하는 문제는, 흔히 두 부분으로 나누어 적분한다. 그러나 1등급의 풀이는 다르다.

$$\int_a^b p(x)\,dx - \int_a^b f(x)\,dx = \int_a^b \bigl[p(x)-f(x)\bigr]\,dx$$
핵심: 두 함수의 차함수 $g(x) = p(x) - f(x)$를 먼저 만들어, 전체 구간에서 한 번에 적분한다. 차함수의 모양·근·점대칭 구조가 보이면 계산은 자동으로 풀린다.
[일반적 풀이] [차함수 관점] ∫₋₃⁰ + ∫₀¹ ∫₋₃¹ (p - f) dx ↓ ↓ 구간 둘로 쪼개기 한 번에 (구조 분석 가능) 계산 두 번 → 변곡점·점대칭 발견 시 끝 시간 ↑↑ 시간 ↓↓
실전 팁: 직선 $p(x)$와 곡선 $f(x)$의 그래프를 머리에 그릴 때, 직선 자체보다 차이($p - f$)의 그래프를 떠올려라. 두 그래프 사이의 "갭"을 함수로 본 셈.
2삼차함수의 변곡점 점대칭 (핵심 무기)

모든 삼차함수 $f(x)$는 변곡점에 대해 점대칭이다. 변곡점을 $(p, q)$라 하면, 좌우 대칭 구간에서 함수값들이 짝지어 평균 $q$가 된다.

점대칭 성질:
$f(p + h) + f(p - h) = 2q$ (모든 $h$에 대해)
⚙
적분 단순화: 변곡점 $x=p$가 적분 구간 $[a, b]$의 중점이라면 $$\int_a^b f(x)\,dx = q\cdot(b - a)$$
$$\int_{p-L}^{p+L} f(x)\,dx = 2L \cdot q = (\text{구간 길이}) \times (\text{변곡점 } y\text{값})$$
왜 성립하나: 점대칭으로 $f(p+h) + f(p-h) = 2q$가 모든 $h$에서 성립 → 좌우 대칭 구간을 짝지어 평균을 내면 모두 $q$ → 적분(넓이의 합)은 $q$와 구간 길이의 곱.
변곡점 (p, q) — 점대칭 비유 곡선 ╱╲ 평균 높이 q ╱ ╲ ─────●────── ← 변곡점 (p, q) ╱ ╲ → │ │ p-L p+L p-L p+L ◀─── 2L ────▶ (튀어나온 + 들어간 = 평평한 평균) 적분값 = q × 2L
📌 영상 예시 — 6평 유형 적용

변곡점 $(-1,\;2)$인 삼차함수 $f(x)$를 구간 $[-3,\;1]$에서 적분.

중점 확인
$[-3, 1]$의 중점 = $-1$ = 변곡점 $x$좌표 ✔
변곡점 $y$값
$q = 2$
즉시 적분
$\displaystyle\int_{-3}^{1} f(x)\,dx = q\cdot(b-a) = 2\cdot 4 = \mathbf{8}$
식조차 안 씀. 변곡점 좌표 두 수만 알면 끝. 영상에서 강사가 "복잡한 식에 대입할 필요 없이"라고 한 것이 이 의미.
3"울퉁불퉁한 길의 평균 높이" 비유

아무리 굴곡이 심한 길이라도, 한 점에 대해 좌우가 대칭(높낮이 짝지어 평평)이라면 그 길의 평균 높이 = 대칭 중심의 높이.

╱╲ ╱ ╲ ╱╲ ╱ ╲ ╱╲ 삼차함수 그래프 ╱ ╲ ╲ ╲ ↓ ● ← 변곡점 이 길의 평균 높이 = q "넓이 = 평균 × 길이" 라는 초등 개념으로 회귀
기하학적 의미: 점대칭 구간에서 정적분은 직사각형(가로 = 길이, 세로 = 평균 = $q$)의 넓이로 환원된다. 곡선이지만 답은 직사각형 넓이.
주의 조건: 이 공식은 변곡점이 구간의 중점일 때만 그대로 통한다. 변곡점이 한쪽에 치우치면 좌우가 짝이 안 맞아 평균 $q$가 안 된다 (별도 처리 필요).
기초 변곡점·점대칭 공식 직접 적용
기초 1 — 변곡점 찾기
$f(x) = x^3 - 6x^2 + 5$의 변곡점의 좌표를 구하시오.
풀이 보기
$f''=0$
$f'(x) = 3x^2 - 12x$, $f''(x) = 6x - 12 = 0$ → $x = 2$
$y$값
$f(2) = 8 - 24 + 5 = -11$
정답: $(2, -11)$
기초 2 — 점대칭 적분 (변곡점 = 구간 중점)
변곡점이 $(3, 5)$인 삼차함수 $f(x)$에 대해 $\displaystyle\int_1^5 f(x)\,dx$의 값을 구하시오.
풀이 보기
중점 확인
$[1, 5]$의 중점 = $3$ = 변곡점 $x$좌표 ✔
계산
적분 = $q \times (b-a) = 5 \times 4 = \mathbf{20}$
정답: 20
기초 3 — $f(p+h)+f(p-h)=2q$ 확인
$f(x) = x^3 - 3x^2 + 4$의 변곡점 좌표를 구하고, $f(0) + f(2)$의 값을 구하시오.
풀이 보기
변곡점
$f''(x) = 6x - 6 = 0$ → $x = 1$, $f(1) = 1-3+4 = 2$. 변곡점 $(1, 2)$.
점대칭
$0$과 $2$는 $x=1$ 대칭 ($h=1$). 점대칭: $f(0) + f(2) = 2q = 4$
검산
$f(0) = 4$, $f(2) = 8-12+4 = 0$. $4+0 = 4$ ✔
정답: 변곡점 $(1,2)$, $f(0)+f(2) = 4$
기본 차함수 + 점대칭 결합
기본 1 — 차함수 만들기
$f(x) = x^3$, $p(x) = 3x - 2$일 때 $\displaystyle\int_{-2}^{2} \bigl[p(x) - f(x)\bigr]\,dx$의 값을 구하시오.
풀이 보기
차함수
$g(x) = p(x) - f(x) = -x^3 + 3x - 2$
변곡점
$g'(x) = -3x^2 + 3$, $g''(x) = -6x = 0$ → $x=0$. $g(0) = -2$. 변곡점 $(0, -2)$.
중점 확인
$[-2, 2]$의 중점 = $0$ = 변곡점 $x$ ✔
적분
$q \times (b-a) = (-2) \times 4 = \mathbf{-8}$
정답: $-8$
기본 2 — 변곡점 위치 결정
$f(x) = x^3 - 3x^2 + ax + b$의 변곡점이 $(1, 0)$일 때, 상수 $a, b$의 값을 구하고 $\displaystyle\int_{-1}^{3} f(x)\,dx$를 구하시오.
풀이 보기
$x$좌표 조건
$f''(x) = 6x - 6 = 0$ → $x=1$ (이미 조건 만족, $a$와 무관). ✔
$y$좌표 조건
$f(1) = 1 - 3 + a + b = a + b - 2 = 0$ → $a + b = 2$
정보 부족 → 추가 가정
$a, b$ 단일값이 결정되려면 한 조건 더 필요. 예: $a=0$이라 하면 $b=2$.
적분만 보면 $a, b$ 값 무관: $[-1, 3]$의 중점 = $1$ = 변곡점, $q=0$ → 적분 = $0 \cdot 4 = \mathbf{0}$
핵심: 적분 계산은 변곡점 좌표만 알면 되므로 $a, b$ 값을 모두 구할 필요 없음. 이것이 점대칭 적분의 위력.
정답: $a+b=2$, $\int = 0$
기본 3 — 꺾인 함수 + 곡선
변곡점이 $(0, 1)$인 삼차함수 $f(x)$가 있다. 직선 $p(x) = 2x + 3$일 때 $\displaystyle\int_{-2}^{2} \bigl[p(x) - f(x)\bigr]\,dx$의 값을 구하시오.
풀이 보기
분리 적분
$\int_{-2}^{2} p(x)\,dx - \int_{-2}^{2} f(x)\,dx$
$\int p$
$p(x) = 2x + 3$. $[-2, 2]$의 중점 $0$에서 $p(0)=3$. 직선은 모든 점에서 점대칭이므로 평균 $\times$ 길이 = $3 \cdot 4 = 12$
(또는 $\int_{-2}^{2}(2x+3)dx = [x^2+3x]_{-2}^{2} = 10-(-2)=12$ ✔)
$\int f$
$f$의 변곡점 $(0, 1)$, 구간 중점 $0$ ✔. 적분 = $1 \cdot 4 = 4$
차
$12 - 4 = \mathbf{8}$
직선·곡선 모두 변곡점 평균 방식으로 즉시 처리. $f$의 식 자체를 몰라도 풀린다.
정답: 8
심화 변곡점 ≠ 구간 중점 처리
심화 1 — 구간 비대칭 분할
변곡점이 $(1, 3)$인 삼차함수 $f(x)$에 대해 $\displaystyle\int_{-2}^{4} f(x)\,dx$를 구하시오.
풀이 보기
중점 확인
$[-2, 4]$의 중점 = $1$ = 변곡점 $x$ ✔ (운 좋게 정확히 매치)
즉시
적분 = $3 \times 6 = \mathbf{18}$
학습 포인트: 변곡점이 정확히 구간 중점일 때만 직격으로 통하는 것을 확인. 출제자는 일부러 매치되도록 좌표를 잡는다.
정답: 18
심화 2 — 변곡점 비매치 시 우회
$f(x) = x^3 - 6x^2 + 9x$의 변곡점을 구하고, $\displaystyle\int_0^4 f(x)\,dx$를 구하시오.
풀이 보기
변곡점
$f''(x) = 6x - 12 = 0$ → $x = 2$. $f(2) = 8 - 24 + 18 = 2$. 변곡점 $(2, 2)$.
중점 매치
$[0, 4]$의 중점 = $2$ ✔. 변곡점 $x$와 일치.
즉시 계산
$q \times (b-a) = 2 \times 4 = \mathbf{8}$
검산 (직접 적분)
$\int_0^4 (x^3-6x^2+9x)dx = \left[\dfrac{x^4}{4} - 2x^3 + \dfrac{9x^2}{2}\right]_0^4 = 64 - 128 + 72 = 8$ ✔
정답: 변곡점 $(2,2)$, 적분 = 8
수능 대비 준킬러 — 6평 유형
수능 대비 1 — 준킬러 (영상 6평 유형 변형)
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음을 만족시킨다.

(가) $f(-1) = 2$, $f'(-1) = -3$
(나) $f''(-1) = 0$

$\displaystyle\int_{-3}^{1} f(x)\,dx$의 값을 구하시오.
풀이 보기
조건 (나)
$f''(-1) = 0$ → $x = -1$이 변곡점의 $x$좌표.
조건 (가)
$f(-1) = 2$ → 변곡점 $y$값 $q = 2$.
구간 중점 매치
$[-3, 1]$의 중점 = $\dfrac{-3+1}{2} = -1$ = 변곡점 $x$ ✔
즉시 적분
$q \times (b-a) = 2 \times 4 = \mathbf{8}$
핵심: 조건 (가)의 $f'(-1) = -3$은 적분에 불필요한 미끼! 함수 식을 복원해서 적분하려는 풀이는 시간 낭비. 변곡점 좌표 두 정보만으로 끝.
정답: 8
수능 대비 2 — 준킬러 (꺾은선 + 삼차)
최고차항의 계수가 $1$인 삼차함수 $f(x)$의 변곡점이 $(2, 4)$이다. 함수 $p(x) = |x - 2| + 4$일 때 $\displaystyle\int_0^4 \bigl[p(x) - f(x)\bigr]\,dx$의 값을 구하시오.
풀이 보기
분리
$\int_0^4 p\,dx - \int_0^4 f\,dx$
$\int p$ — 절댓값 함수
$p(x) = |x-2| + 4$. V자 모양, 꼭짓점 $(2, 4)$.
$[0,4]$에서 도형은 밑변 4, 높이 (꼭짓점에서 변까지) 2인 사다리꼴 + 직사각형. 또는 직접 계산:
$\int_0^2 (-(x-2)+4)dx + \int_2^4 ((x-2)+4)dx$
$= \int_0^2 (6-x)dx + \int_2^4 (x+2)dx$
$= [6x-\tfrac{x^2}{2}]_0^2 + [\tfrac{x^2}{2}+2x]_2^4$
$= (12-2) + (16 - 6) = 10 + 10 = 20$
$\int f$ — 점대칭 활용
$[0,4]$의 중점 = $2$ = 변곡점 $x$ ✔. $q = 4$. 적분 = $4 \times 4 = 16$
차
$20 - 16 = \mathbf{4}$
꺾은선($p$)은 직접 계산, 곡선($f$)은 점대칭 공식. 두 가지 도구를 결합.
정답: 4
수능 대비 3 — 준킬러 (변곡점·근 동시 활용)
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 $f(0) = 0$, $f(4) = 0$이고 변곡점의 $x$좌표가 $2$이다. $\displaystyle\int_0^4 f(x)\,dx$의 값을 구하시오.
풀이 보기
점대칭 $\Rightarrow$ $f(2)$
변곡점 $x=2$에 점대칭. $f(0)+f(4) = 2 \cdot f(2)$
$0 + 0 = 2f(2)$ → $f(2) = 0$. 변곡점은 $(2, 0)$.
중점 매치
$[0,4]$의 중점 = $2$ = 변곡점 $x$ ✔. $q=0$.
적분
$0 \times 4 = \mathbf{0}$
기하적 해석
$f(x) = x(x-2)(x-4)$ (세 근이 $0, 2, 4$).
$x=2$에 점대칭이라 $x=0, 4$ 사이의 양·음 넓이가 정확히 상쇄.
정답: 0
📝 한눈에 보는 정리
도구핵심 식활용 조건
점대칭 평균 $f(p+h)+f(p-h)=2q$ 모든 $h$ (변곡점 $(p,q)$인 삼차)
점대칭 적분 $\int_{p-L}^{p+L} f\,dx = 2qL$ 변곡점 = 구간 중점
차함수 묶기 $\int(p-f) = \int p - \int f$ 곡선·꺾은선 동시 등장 시
변곡점 위치 $f''(x)=0$ 풀면 $x=-\tfrac{b}{3a}$ $f(x)=ax^3+bx^2+\cdots$
💡 시험장 체크리스트
① 적분 구간의 중점이 무엇인지 즉시 계산.
② 삼차함수면 변곡점 $x$좌표를 즉시 ($-\tfrac{b}{3a}$).
③ 둘이 일치하면 → "$q \times $ 구간 길이"로 1초 내 답.
④ 꺾은선·직선과 곡선이 같이 있으면 차함수로 묶을지 보기.
⑤ 조건의 $f'(\cdots)$ 값은 적분에 종종 불필요한 미끼.
