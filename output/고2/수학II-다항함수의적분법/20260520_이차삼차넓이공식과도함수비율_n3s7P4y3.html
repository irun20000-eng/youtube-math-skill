📐 개념 — 3가지 핵심 공식
⚡ 선수 학습 1분 환기
  • 정적분의 기본정리: $\int_a^b f(x)\,dx = F(b) - F(a)$, $F'(x) = f(x)$
  • 두 곡선 사이 넓이: $S = \int_\alpha^\beta |f(x) - g(x)|\,dx$
  • 이차함수 인수분해: $ax^2+bx+c = a(x-\alpha)(x-\beta)$ ($\alpha, \beta$ 근)
  • 증감과 극값: $F'(x) = f(x) = 0$인 점에서 $F$의 극값.
1$\dfrac{1}{6}$ 공식 — 이차함수 넓이

이차함수 $f(x) = a(x-\alpha)(x-\beta)$와 $x$축으로 둘러싸인 넓이 (또는 두 곡선 차이가 이차함수일 때).

$$S = \int_\alpha^\beta |a(x-\alpha)(x-\beta)|\,dx = \dfrac{|a|}{6}(\beta - \alpha)^3$$
⚙
유도: $\int_\alpha^\beta (x-\alpha)(x-\beta)\,dx = -\dfrac{(\beta-\alpha)^3}{6}$ (직접 계산). 부호는 그래프 위치에 따라 ±.
적용 패턴:
  • 이차함수 $y = ax^2 + \cdots$ 와 $x$축 사이
  • 이차함수 $y = ax^2 + \cdots$ 와 일차함수 $y = mx + n$ 사이 (두 곡선 차이가 이차)
  • 두 이차함수 $y = a_1 x^2 + \cdots$, $y = a_2 x^2 + \cdots$ 사이 (차이가 이차, 최고차 계수 $a_1 - a_2$)
y = a(x-α)(x-β), a > 0 인 경우 y │ │ ╲ ╱ (포물선) │ ╲ ╱ │ ╲ ╱ │ ╲ ╱ ●─────●───── x축 α β ↑넓이 S = |a|/6 · (β-α)³
📌 예시 — 직접 계산

$y = -x^2 + 4x$ ($a = -1$)와 $x$축으로 둘러싸인 넓이.

근
$-x(x-4) = 0$ → $x = 0, 4$. $\alpha = 0, \beta = 4$.
공식
$S = \dfrac{|-1|}{6}(4-0)^3 = \dfrac{64}{6} = \dfrac{32}{3}$
검산
$\int_0^4 (-x^2+4x)\,dx = \left[-\dfrac{x^3}{3} + 2x^2\right]_0^4 = -\dfrac{64}{3} + 32 = \dfrac{-64+96}{3} = \dfrac{32}{3}$ ✔
강사 인용: "이게 90% 이상 이것만 쓰는 게 90% 이상." — 이차함수 넓이 문제는 사실상 이 공식만 안다면 끝.
2$\dfrac{1}{12}$ 공식 — 삼차함수와 접선

삼차함수 $f(x) = a(x-\alpha)(x-\beta)(x-\gamma)$ 형태의 그래프와 그 위 한 점에서의 접선이 다시 만나는 점까지의 넓이.

삼차함수 $f(x)$와 접선이 $x = \alpha$ (접점), $x = \beta$ (교점)을 가질 때
$f(x) - (\text{접선}) = a(x - \alpha)^2 (x - \beta)$  (접점은 이중근)
$$S = \int_\alpha^\beta |a(x-\alpha)^2(x-\beta)|\,dx = \dfrac{|a|}{12}(\beta - \alpha)^4$$
⚙
왜 $(x-\alpha)^2$ 인가: 접선이라는 것은 $f(x) - \text{접선}$의 그래프가 $x=\alpha$에서 $x$축에 접한다는 뜻 → $(x-\alpha)$가 이중근.
⚙
유도: $\int_\alpha^\beta (x-\alpha)^2(x-\beta)\,dx = -\dfrac{(\beta-\alpha)^4}{12}$ (직접 계산).
삼차함수 + 접선 y ● (교점 β) │ ╱╲ │ ╱ ╲ ← 접선 │ ╱ ╲ │ ●────●─ ╲ │ 접점 α ╲ ●─────────────────●─ x축 α β S = |a|/12 · (β-α)⁴
📌 예시 — 삼차+접선 넓이

$f(x) = x^3 - 3x$의 $x = 1$에서의 접선과 곡선으로 둘러싸인 넓이.

접선 식
$f'(x) = 3x^2 - 3$, $f'(1) = 0$ (접선 기울기 $0$). $f(1) = -2$. 접선 $y = -2$.
교점
$x^3 - 3x = -2$ → $x^3 - 3x + 2 = 0$.
$x = 1$이 이중근이므로 $(x-1)^2(x-r) = 0$. 전개해서 계수 비교 → $(x-1)^2(x+2) = 0$. $\beta = -2$.
공식
$\alpha = 1$, $\beta = -2$. $|\beta - \alpha| = 3$.
$S = \dfrac{|1|}{12}\cdot 3^4 = \dfrac{81}{12} = \dfrac{27}{4}$
주의: $\alpha < \beta$가 아니어도 $|\beta - \alpha|$ 자리에 절댓값 차이를 넣음. 부호는 그래프 위치로 결정.
3도함수 넓이 $=$ 원함수 극값 차

$F'(x) = f(x)$의 관계에서, $f(x)$의 그래프가 $x$축과 만드는 넓이는 $F(x)$의 극값 변화량과 일치.

$$\int_a^b f(x)\,dx = F(b) - F(a)$$
$f$의 부호 있는 넓이 = $F$의 함숫값 변화량
핵심 해석: $f(x)$의 그래프가 만드는 양·음 부호 넓이는 $F(x)$의 극대·극소 사이 높이 차이와 같다. 두 그래프를 하나의 정보로 바꿔 본다.
F(x) (원함수) f(x) = F'(x) (도함수) y y │ 극대 │ │ ╱╲ │ ╱──╲ (+) 양의 넓이 │ ╱ ╲ │ ╱ ╲ ●╱ ╲ ●──────●─── x ╲ α β \ ╲ ╲ (-) 음의 넓이 극소 ●╲ ╲ F(β) - F(α) = [α, β] 구간 f의 부호 있는 넓이 극값 차 = 양음 넓이 차
⚙
실전 활용 — 정적분 = 0: $\int_a^b f(x)\,dx = 0$이면 $F(b) = F(a)$. 두 점에서 같은 함숫값.
⚙
비율 관계 — 2:1: 삼차함수 $F$의 극대점·극소점 $x$좌표가 $p, q$이면, $F(p) - F(q)$를 $\int_p^q f \,dx$로 환산. 도함수 $f$ (이차)가 $x$축과 만드는 두 영역 (양수부·음수부) 넓이의 비가 보통 2:1 또는 정수비.
📌 예시 — 도함수 넓이로 극값 차 계산

$F(x) = x^3 - 3x^2$의 극값 차이를 도함수로 구하라.

$F'(x)$
$F'(x) = 3x^2 - 6x = 3x(x-2)$. 극값 $x = 0, 2$.
$\int_0^2 F'(x)\,dx$
$F'$가 $[0,2]$에서 부호 음수 → 음의 넓이.
$|S| = \dfrac{|3|}{6}(2-0)^3 = \dfrac{8}{2} = 4$. 부호 음수이므로 $\int_0^2 = -4$.
극값 차
$F(2) - F(0) = -4 - 0 = -4$ ✔ 정확히 일치.
기초 6분 공식·12분 공식 직접 적용
기초 1 — 6분 공식
$y = -x^2 + 6x$와 $x$축으로 둘러싸인 부분의 넓이를 구하시오.
풀이 보기
근
$-x(x-6) = 0$ → $\alpha = 0, \beta = 6$. $|a| = 1$.
공식
$S = \dfrac{1}{6}(6)^3 = \dfrac{216}{6} = 36$
정답: 36
기초 2 — 두 곡선 사이
$y = x^2 - 2x$와 $y = 2x - 3$로 둘러싸인 부분의 넓이를 구하시오.
풀이 보기
차이
$g(x) = (x^2 - 2x) - (2x - 3) = x^2 - 4x + 3 = (x-1)(x-3)$. $a = 1$.
공식
$S = \dfrac{1}{6}(3-1)^3 = \dfrac{8}{6} = \dfrac{4}{3}$
정답: $\dfrac{4}{3}$
기초 3 — 12분 공식 인지
$f(x) = x^3 - 3x^2 + 4$의 그래프와 $x = 2$에서의 접선으로 둘러싸인 부분의 넓이를 구하시오.
풀이 보기
접선
$f'(x) = 3x^2 - 6x$, $f'(2) = 0$. $f(2) = 0$. 접선 $y = 0$ ($x$축).
교점
$x^3 - 3x^2 + 4 = 0$. $x=2$ 이중근: $(x-2)^2(x+1) = x^3 - 3x^2 + 4$ ✔. $\beta = -1$.
공식
$\alpha = 2$, $\beta = -1$. $|\beta - \alpha| = 3$.
$S = \dfrac{|1|}{12}\cdot 3^4 = \dfrac{81}{12} = \dfrac{27}{4}$
정답: $\dfrac{27}{4}$
기본 두 곡선·접선 결합
기본 1 — 두 이차함수 사이
$y = x^2$와 $y = -x^2 + 4$로 둘러싸인 부분의 넓이를 구하시오.
풀이 보기
차이
$g(x) = x^2 - (-x^2 + 4) = 2x^2 - 4 = 2(x^2 - 2) = 2(x-\sqrt 2)(x+\sqrt 2)$. $a = 2$.
공식
$\alpha = -\sqrt 2, \beta = \sqrt 2$. $\beta - \alpha = 2\sqrt 2$.
$S = \dfrac{|2|}{6}\cdot(2\sqrt 2)^3 = \dfrac{1}{3}\cdot 16\sqrt 2 = \dfrac{16\sqrt 2}{3}$
정답: $\dfrac{16\sqrt 2}{3}$
기본 2 — 도함수 넓이로 극값
$F(x) = x^3 - 6x^2 + 9x$의 극댓값과 극솟값의 차이를 도함수의 넓이를 이용하여 구하시오.
풀이 보기
$F'(x)$
$F'(x) = 3x^2 - 12x + 9 = 3(x-1)(x-3)$. 극값 $x = 1, 3$.
차이 = $|\int_1^3 F' dx|$
$\int_1^3 3(x-1)(x-3) dx$. 부호: $(x-1)(x-3) < 0$ in (1,3) → 음수.
$|\int| = \dfrac{|3|}{6}(3-1)^3 = \dfrac{1}{2}\cdot 8 = 4$. 극값 차 = $4$.
검산
$F(1) = 1 - 6 + 9 = 4$ (극대), $F(3) = 27 - 54 + 27 = 0$ (극소). 차 = $4$ ✔
정답: 4
기본 3 — 곡선 + 접선
$f(x) = x^3$의 그래프와 $x = -1$에서의 접선으로 둘러싸인 부분의 넓이를 구하시오.
풀이 보기
접선
$f'(x) = 3x^2$, $f'(-1) = 3$. $f(-1) = -1$. 접선 $y = 3(x+1) - 1 = 3x + 2$.
교점
$x^3 = 3x + 2$ → $x^3 - 3x - 2 = 0$. $x = -1$ 이중근: $(x+1)^2(x-2) = 0$. $\beta = 2$.
공식
$\alpha = -1, \beta = 2$. $|\beta - \alpha| = 3$.
$S = \dfrac{|1|}{12}\cdot 3^4 = \dfrac{81}{12} = \dfrac{27}{4}$
정답: $\dfrac{27}{4}$
심화 비율 관계 + 정적분 = 0
심화 1 — 정적분 = 0 (넓이 같음)
$f(x) = x^2 - 2x$일 때 $\displaystyle\int_0^a f(x)\,dx = 0$을 만족하는 양수 $a$를 구하시오.
풀이 보기
기하 해석
$\int_0^a f = 0$ → $[0, a]$에서 양·음 넓이 같음. $f$의 근: $x = 0, 2$.
$[0,2]$에서 음수, $[2,a]$에서 양수 → 같은 절댓값이어야.
직접 계산
$\int_0^a (x^2 - 2x)dx = \left[\dfrac{x^3}{3} - x^2\right]_0^a = \dfrac{a^3}{3} - a^2 = \dfrac{a^2(a - 3)}{3} = 0$
$a = 3$ ($a > 0$)
비율 관계: 양·음 넓이가 같은 경우, $a = \dfrac{3}{2}\cdot 2 = 3$이 종종 등장 ($x = 0, 2$의 위치를 1:2로 분할).
정답: $a = 3$
심화 2 — 도함수 넓이 비율
삼차함수 $F(x)$의 도함수 $f(x) = F'(x)$가 $f(x) = 3(x-1)(x-4)$일 때, $F$의 극댓값과 극솟값의 차이를 구하시오.
풀이 보기
극값 위치
$f(x) = 0$ → $x = 1, 4$. $f$가 $[1,4]$에서 음수 → $F$는 감소 → $x=1$ 극대, $x=4$ 극소.
$\int_1^4 f\,dx$ 절댓값
$\left|\int_1^4 3(x-1)(x-4)dx\right| = \dfrac{3}{6}(4-1)^3 = \dfrac{27}{2}$
극값 차
$F(1) - F(4) = -\int_1^4 f dx = \dfrac{27}{2}$ (부호 양수, $F$가 감소했으므로).
정답: $\dfrac{27}{2}$
수능 대비 준킬러 — 공식·비율 종합
수능 대비 1 — 준킬러 (이차 + 비율)
이차함수 $f(x) = x^2 - 4x$와 직선 $y = mx$로 둘러싸인 부분의 넓이가 $\dfrac{32}{3}$일 때, 양수 $m$의 값을 구하시오.
풀이 보기
차이
$g(x) = x^2 - 4x - mx = x^2 - (m+4)x = x(x - (m+4))$. $a = 1$.
근 $\alpha = 0, \beta = m + 4$.
6분 공식
$S = \dfrac{1}{6}(m+4)^3 = \dfrac{32}{3}$
$(m+4)^3 = 64$ → $m + 4 = 4$ → $m = 0$? 아닌데 양수 조건. → $(m+4)^3 = 64$ → $m+4 = \sqrt[3]{64} = 4$ → $m = 0$
$m = 0$이면 직선이 $x$축. $y = mx = 0$ → 둘러싸인 넓이 $= \int_0^4 |x^2-4x|dx = \dfrac{32}{3}$ ✔ (기초 1과 동일).
그러나 양수 조건 → 답 없음? 문제 재검토. 사실 $m > 0$이면 두 곡선 교점이 $x = 0, m+4$이며 $m+4 > 4$. 새 계산.
$\dfrac{1}{6}(m+4)^3 = \dfrac{32}{3}$ → $(m+4)^3 = 64$ → $m+4 = 4$ → $m=0$ (모순).
$\dfrac{1}{6}(m+4)^3 = \dfrac{32}{3}$가 답이 $m+4=4$ 정확. 양수 해 없음. 조건 보정 필요.

문제 조정: $S = \dfrac{125}{6}$이라면 $(m+4)^3 = 125$ → $m+4 = 5$ → $m = 1$ (양수 ✔).
본 문제 학습 포인트: 6분 공식의 역방향 — 넓이가 주어졌을 때 미지수를 찾는 흐름. 양수 해 검증 필수.
학습용 — 공식 역방향 흐름 + 정합 검증
수능 대비 2 — 준킬러 (삼차 + 접선 + 비율)
삼차함수 $f(x) = x^3 - 3x^2$ 와 $x = t$ ($t > 0$)에서의 접선으로 둘러싸인 부분의 넓이가 $\dfrac{1}{12}$일 때, $t$의 값을 구하시오.
풀이 보기
접선
$f'(x) = 3x^2 - 6x$. 접점 $x = t$에서 접선:
$y - f(t) = f'(t)(x - t)$
$f(x) - $ 접선 $= (x - t)^2 (x - r)$ 형태 (이중근). 12분 공식의 $\beta = r$.
$r$ 결정
$f(x) - $ 접선의 최고차 계수 $1$. $(x-t)^2(x-r) = x^3 - (2t+r)x^2 + (t^2+2tr)x - t^2 r$.
$f(x) - $ 접선의 $x^2$ 계수는 $-3$ (원래 $-3x^2$, 접선은 일차).
$-(2t + r) = -3$ → $r = 3 - 2t$.
12분 공식
$|\beta - \alpha| = |r - t| = |3 - 3t|$.
$S = \dfrac{1}{12}(3 - 3t)^4 = \dfrac{1}{12} \cdot 81(1-t)^4 = \dfrac{27(1-t)^4}{4}$
$= \dfrac{1}{12}$ → $(1-t)^4 = \dfrac{1}{81}$ → $|1-t| = \dfrac{1}{3}$
$t = \dfrac{2}{3}$ 또는 $t = \dfrac{4}{3}$ (양수 모두).
핵심: 12분 공식 + 계수 비교로 교점 $r$ 결정 + 양수 조건으로 후보 좁힘.
정답: $t = \dfrac{2}{3}$ 또는 $\dfrac{4}{3}$
수능 대비 3 — 준킬러 (도함수 넓이 비율)
삼차함수 $F(x)$의 도함수 $f(x) = F'(x)$의 그래프가 $x$축과 점 $A(\alpha, 0), B(\beta, 0)$ ($\alpha < \beta$)에서 만나며 최고차항의 계수가 $1$이다. $[\alpha, \beta]$에서 $f(x)$의 그래프와 $x$축이 둘러싼 넓이가 $\dfrac{4}{3}$이고 $F(\alpha) = 5$일 때, $F(\beta)$의 값을 구하시오.
풀이 보기
$f$ 형태
$f(x) = (x - \alpha)(x - \beta)$ (최고차항 1). $[\alpha, \beta]$에서 음수.
넓이
$S = \dfrac{|1|}{6}(\beta - \alpha)^3 = \dfrac{4}{3}$ → $(\beta - \alpha)^3 = 8$ → $\beta - \alpha = 2$.
$\int_\alpha^\beta f\,dx$
$f < 0$이므로 $\int_\alpha^\beta f\,dx = -\dfrac{4}{3}$.
$F(\beta) - F(\alpha)$
$F(\beta) - F(\alpha) = \int_\alpha^\beta f\,dx = -\dfrac{4}{3}$
$F(\beta) = 5 - \dfrac{4}{3} = \dfrac{15 - 4}{3} = \dfrac{11}{3}$
핵심: 도함수 넓이 ↔ 원함수 함숫값 변화. 부호도 함께 추적.
정답: $F(\beta) = \dfrac{11}{3}$
