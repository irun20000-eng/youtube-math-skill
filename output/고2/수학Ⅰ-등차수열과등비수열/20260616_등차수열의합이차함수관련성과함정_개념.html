등차수열의 합 $S_n$ 과 이차함수 — 관련성과 함정

$S_n = An^2 + Bn$ 이라는 $n$의 이차함수 관점으로 $S_n$ 최대·최소, 대칭축, 두 합이 같은 경우를 빠르게. 그러나 함정도 안다.

📘 개념 학습자료 (영상 없음 · 자연어 요청 생성)

🎯 학습 목표: ① $S_n$ 이 왜 $n$ 의 이차함수인지 유도. ② 이차함수 도구(대칭축·꼭짓점·근)로 쉽게 풀리는 패턴 3가지. ③ 그러나 $n$ 은 자연수·상수항 $0$·$d=0$ 같은 함정 4가지를 정확히 안다. · 대상: 중수준(하위 40%)도 따라갈 수 있는 호흡

📌 도입 — 이차함수처럼 보이는 등차수열의 합
문제: 등차수열 $\{a_n\}$ 의 합 $S_n$ 에 대해 $S_5 = S_{12}$ 가 성립한다. $S_{17}$ 의 값을 구하시오.
풀이 보기 (이차함수 대칭축으로 5초)
STEP 1
등차수열의 합은 $S_n = \dfrac{d}{2}n^2 + \left(a_1 - \dfrac{d}{2}\right)n$ — $n$ 에 대한 이차함수 모양($An^2+Bn$ 형, 상수항 $0$).
$n=0$ 일 때 $S_0 = 0$(아무 것도 안 더함). 즉 $n$ 의 이차함수로 보면 그래프가 항상 원점을 지난다.
STEP 2
$S_n$ 그래프는 두 점 $(5, S_5)$ 와 $(12, S_5)$ 에서 같은 $y$값. 이차함수에서 같은 $y$값 두 점 → 대칭축이 두 점의 중점: $x = \dfrac{5+12}{2} = \dfrac{17}{2}$.
STEP 3
이차함수가 원점 $(0,0)$ 을 지난다 → 또 다른 근은 대칭축에서 반대쪽으로 같은 거리: $x = 17$. 즉 $S_{17} = 0$.
두 근의 평균이 대칭축. 한 근이 $0$ 이고 대칭축이 $\tfrac{17}{2}$ → 다른 근은 $17$.
검산
일반화: $S_p = S_q$ ($p\ne q$) → $S_{p+q} = 0$.
여기서 $p+q = 5+12 = 17$, $S_{17} = 0$ ✓.
정답: $S_{17}=0$
이 한 줄 — $S_p = S_q$ 이면 $S_{p+q} = 0$ — 이 본 자료의 핵심이다. 이차함수의 대칭축이 모든 비밀.
💡 개념 설명
1$S_n$ 은 왜 $n$의 이차함수인가? (유도)

등차수열 $\{a_n\}$ 의 첫째항 $a_1$, 공차 $d$. 첫 $n$ 항의 합:

$S_n = \dfrac{n\bigl(2a_1+(n-1)d\bigr)}{2} = \dfrac{d}{2}n^2 + \left(a_1-\dfrac{d}{2}\right)n$
A
$n^2$ 계수 $= \dfrac{d}{2}$. 공차 $d$ 의 절반.
B
$n$ 계수 $= a_1 - \dfrac{d}{2}$. 첫째항에서 공차 절반 뺀 값.
C
상수항 $= 0$ (반드시!). $S_0 = 0$ 이라는 자연스러운 사실.
⭐ $S_n = An^2 + Bn$ (상수항 $0$). 거꾸로 $S_n = An^2 + Bn$ 이면 그 수열은 등차수열, 공차 $d = 2A$.
2이차함수 도구 — 쉬운 경우 3가지

$S_n = An^2 + Bn$ 의 그래프(연속한 포물선)를 그려보면, 자연수 $n$ 의 값만 점으로 찍힌 형태가 된다. 대칭축·꼭짓점·근이 그대로 무기.

①
두 합이 같을 때 — 대칭축: $S_p = S_q$ ($p \ne q$) 이면 그 두 점은 대칭축에 대해 대칭 → 축의 위치 $n = \dfrac{p+q}{2}$.
한 근이 $0$ 이므로 다른 근은 $p+q$ → $S_{p+q} = 0$.
②
$S_n$ 의 최대·최소: 그래프의 꼭짓점 위치 $n^* = -\dfrac{B}{2A} = \dfrac{2a_1 - d}{2d}$ 근처에서 발생. (그러나 $n$ 은 자연수 → ⑤ 함정 주의)
③
두 합이 0일 때: $S_p = 0$ ($p \ne 0$) → $An^2 + Bn = An(n-p) = 0$. 즉 다른 근은 $p$ 자기 자신만 알려준 셈.
3꼭짓점 = 부호 바뀌는 항 — 항의 부호로도 알 수 있다

$S_n$ 이 최대가 되는 $n$ = $a_n > 0$ 인 마지막 $n$. (감소 등차수열에서 양수 항만 더할 때까지가 최대.)

$a_n > 0,\ a_{n+1} \le 0$ 인 $n$ 에서 $S_n$ 최대  (단, $d < 0$)

마찬가지로 증가 수열($d > 0$)에서 처음에 음수였다가 양수로 바뀌면 그 직전이 $S_n$ 최소.

이차함수 꼭짓점 ↔ 항의 부호: 두 방법이 같은 결과. 어느 쪽으로 풀어도 OK. 단 꼭짓점은 실수이고 항은 자연수라는 차이를 다음 ⑤에서 본다.
4역방향 — $S_n$ 식이 주어졌을 때 $a_n, d$ 끌어내기

$S_n = An^2 + Bn$ 의 모양으로 주어지면 다음이 즉시:

d
공차 $d = 2A$ (꼭 검산).
$a_1$
첫째항 $a_1 = S_1 = A + B$.
$a_n$
$n \ge 2$ 에서 $a_n = S_n - S_{n-1} = 2An - A + B$. ($n=1$ 도 식에 대입해 보고 일치하는지 검산)
⚠️ 만약 $S_n = An^2 + Bn + C$ ($C \ne 0$) 형이면 — 이 수열은 등차수열이 아니다. (또는 $n \ge 2$ 부터만 등차이고 $a_1$ 이 따로). $C$ 의 존재가 등차수열 가정의 위반 신호.
5⚠️ 함정 — 이차함수처럼 풀면 망하는 4가지

도구의 한계. 시험에서 정확히 이 지점들을 노린다.

①
$n$ 은 자연수만 가능. 꼭짓점 $n^* = -\dfrac{B}{2A}$ 가 실수로 나와도 $S_n$ 의 실제 최대·최소는 그 양옆 가장 가까운 자연수($\lfloor n^* \rfloor$ 또는 $\lceil n^* \rceil$) 중 큰 쪽에서.
예: $n^* = 5.7$ → $S_5$ 와 $S_6$ 모두 계산해 비교.
②
상수항이 0이라는 사실을 잊지 말 것. 문제가 "$S_n$ 은 $n$ 의 이차식"이라고 줘도 상수항이 있을 수 있는지(= 등차수열이 아닐 수 있는지) 먼저 점검.
$S_n = n^2 + 2n + 1$ 같이 상수항이 있으면 등차수열이 아니다 → $S_p = S_q \Rightarrow S_{p+q}=0$ 같은 공식 ❌.
③
공차 $d = 0$ 인 케이스: 모든 항이 같은 상수수열. $S_n = a_1 n$ 으로 일차. 이차함수 도구(대칭축·꼭짓점) ❌.
$d=0$ 이면 $S_p = S_q$ 가 성립하려면 $a_1=0$ 뿐. 그러면 모든 $S_n=0$ → 자명한 경우만.
④
$S_p = S_q \Rightarrow S_{p+q} = 0$ 의 조건 = $p \ne q$. 같은 두 점은 대칭축이 결정 안 됨. 또 두 점이 모두 꼭짓점 자체와 같은 위치인 경우(= 그 자리에서 $S_n$ 이 최대/최소)와 헷갈리지 않게.
⑤ $p$ 와 $q$ 의 평균이 자연수가 아닐 때: $S_p = S_q$, $p+q$ 가 홀수면 대칭축이 분수 → "꼭짓점에서 정수 $n$" 같이 잘못된 추론 ❌. 답이 되는 $S_{p+q} = 0$ 만 성립.
📝 수준별 문항 (기초 2 / 기본 2 / 심화 2 / 수능 대비 2)
기초 — 쉽게 풀리는 경우
기초 1 — 대칭축 즉시 (두 합이 같다)
문제: 등차수열 $\{a_n\}$ 의 합 $S_n$ 에 대해 $S_4 = S_9$ 가 성립한다. $S_{13}$ 의 값을 구하시오.
풀이 보기

이차함수 핵심: $S_p = S_q \Rightarrow S_{p+q} = 0$. ($S_n$ 이 $n$ 의 이차함수이고 상수항 $0$, 즉 그래프가 원점·$p+q$ 두 점에서 $0$.)

$p+q = 4+9 = 13$ → $S_{13} = 0$.

검산: 그래프 대칭축 $\dfrac{4+9}{2} = \dfrac{13}{2}$. 원점 반대편 근이 $13$. ✓

정답: $S_{13} = 0$
기초 2 — $S_n$ 식에서 $a_n$, $d$ 추출
문제: 등차수열 $\{a_n\}$ 의 합 $S_n = 2n^2 - 5n$ 일 때 공차 $d$ 와 $a_1$ 을 구하시오.
풀이 보기

$An^2 + Bn$ 꼴이고 상수항 $0$ → 등차수열 OK. $A=2, B=-5$.

$d = 2A = 4$, $a_1 = S_1 = A+B = 2-5 = -3$.

검산: $a_n = a_1 + (n-1)d = -3 + 4(n-1) = 4n - 7$. $S_n = \sum_{k=1}^n (4k-7) = 4\cdot\tfrac{n(n+1)}{2} - 7n = 2n^2 + 2n - 7n = 2n^2 - 5n$ ✓

정답: $d=4,\ a_1=-3$
기본 — 꼭짓점·항의 부호로 최대 찾기
기본 1 — $S_n$ 의 최댓값 (감소 등차)
문제: 등차수열 $a_1 = 30,\ d = -4$ 일 때 $S_n$ 이 최대가 되는 $n$ 의 값과 최댓값 $S_n$ 을 구하시오.
풀이 보기

방법 1 (항의 부호): $a_n = 30 + (n-1)(-4) = 34 - 4n$. $a_n > 0 \Leftrightarrow n < 8.5$ → 양수 항은 $a_1, \dots, a_8$ 까지(8개). $n=8$ 에서 최대.

$S_8 = \dfrac{8(2\cdot 30 + 7\cdot(-4))}{2} = \dfrac{8(60-28)}{2} = \dfrac{8\cdot 32}{2} = 128$.

방법 2 (이차함수 꼭짓점): $S_n = -2n^2 + 32n$ ($A=-2, B=32$). 꼭짓점 $n^* = -\dfrac{B}{2A} = -\dfrac{32}{-4} = 8$. 정확히 자연수 → 그 자리에서 최대. $S_8 = -2(64)+256 = 128$ ✓

정답: $n=8$, $S_8=128$
기본 2 — 꼭짓점이 자연수가 아닐 때 (함정 ①)
문제: 등차수열 $a_1 = 25,\ d = -4$ 일 때 $S_n$ 이 최대가 되는 $n$ 의 값을 구하시오.
풀이 보기

꼭짓점 계산: $S_n = -2n^2 + 27n$. 꼭짓점 $n^* = \dfrac{27}{4} = 6.75$ → 자연수가 아님!

주변 자연수 비교: $S_6, S_7$ 둘 다 계산.

$a_n = 29 - 4n$. $a_6 = 5 > 0$, $a_7 = 1 > 0$, $a_8 = -3 < 0$.
$\Rightarrow$ 양수 항은 $a_7$ 까지 → $n=7$ 에서 최대.

검산: $S_7 = \dfrac{7(50 + 6\cdot(-4))}{2} = \dfrac{7\cdot 26}{2} = 91$. $S_6 = \dfrac{6(50-20)}{2} = 90$. $S_8 = \dfrac{8(50-28)}{2} = 88$. $S_7 = 91$ 이 가장 큼 ✓

정답: $n=7$ (최댓값 $S_7=91$)

⚠️ 오답 주의: 꼭짓점 $6.75$ 만 보고 "$n=6$ 이나 반올림해서 $7$"이라고 어림짐작 ❌. 꼭 두 자연수에서 직접 계산해 비교. (이 문제는 $a_7 > 0$ 이므로 한 항 더 더하는 게 이득.)

심화 — 함정 인식
심화 1 — 상수항이 있는 경우 (함정 ②)
문제: 수열 $\{a_n\}$ 의 첫 $n$ 항 합 $S_n$ 이 $S_n = n^2 + 2n + 3$ 으로 주어졌다. ⑴ $\{a_n\}$ 이 등차수열인지 판정하고, ⑵ $a_n$ 의 일반항을 구하시오.
풀이 보기

⑴ 등차수열인가? $S_n$ 에 상수항 $3 \ne 0$ 이 있다. 등차수열이면 $S_0 = 0$ 이어야 하는데 $S_0 = 3$ → 등차수열 아님.

⑵ $a_n$ 추출: $S_n - S_{n-1}$ 로 $n \ge 2$ 부분만 일단:
$a_n = (n^2+2n+3) - ((n-1)^2 + 2(n-1) + 3) = 2n + 1$ ($n \ge 2$).
$a_1 = S_1 = 1 + 2 + 3 = 6$.

검산: $a_2 = 5$ (공식), $a_1 = 6$. $a_2 - a_1 = -1 \ne d$ ($a_3 - a_2 = 7-5 = 2 = d$). 즉 $n \ge 2$ 부터만 공차 2 인 등차이고 $a_1$ 은 별도. → 등차수열 ❌, 단지 "이차식으로 $S_n$ 이 표현된" 수열.

정답: 등차수열 아님 / $a_1=6,\ a_n=2n+1$ ($n\ge 2$)

⚠️ $S_n = An^2+Bn+C$ 에서 $C \ne 0$ 인 순간 "등차수열" 가정 무너짐. 그래도 "$S_p = S_q \Rightarrow S_{p+q}=0$"같은 도구를 그대로 쓰면 큰 오답.

심화 2 — 공차 $d=0$ 인 경우 (함정 ③)
문제: 등차수열 $\{a_n\}$ ($a_1 = 5$) 의 합 $S_n$ 이 $S_4 = S_9$ 를 만족한다. ⑴ 공차 $d$ 와 ⑵ $S_{13}$ 을 구하시오. (함정 주의)
풀이 보기

⑴ $d$ 결정: $S_n = \dfrac{d}{2}n^2 + (a_1-\dfrac{d}{2})n$.
$S_4 = S_9 \Rightarrow \dfrac{d}{2}(16-81) + (a_1-\dfrac{d}{2})(4-9) = 0$
$\Rightarrow -\dfrac{65d}{2} - 5(a_1-\dfrac{d}{2}) = 0$
$\Rightarrow -\dfrac{65d}{2} + \dfrac{5d}{2} - 5a_1 = 0$
$\Rightarrow -30d - 5a_1 = 0 \Rightarrow d = -\dfrac{a_1}{6} = -\dfrac{5}{6}$.

$d = -\dfrac{5}{6} \ne 0$ — 이차함수 도구 적용 가능. $S_{4+9} = S_{13} = 0$.

검산: $S_{13} = \dfrac{-5/6}{2}\cdot 169 + (5+\dfrac{5}{12})\cdot 13 = -\dfrac{5\cdot 169}{12} + \dfrac{65\cdot 13}{12} = \dfrac{-845 + 845}{12} = 0$ ✓

정답: $d=-\dfrac{5}{6}$, $S_{13}=0$

⚠️ 만약 문제에서 "$a_1 = 0$" 이라는 조건이 함께 주어졌다면 $d=0$ 도 가능: 모든 항이 $0$, 모든 $S_n=0$ 자명. 그 경우 "$S_{p+q}=0$" 은 새로운 정보 ❌. 항상 $d \ne 0$ 인지 확인.

수능 대비 — 함정 결합
수능 대비 1 — 단답형 (대칭축 활용)
문제: 등차수열의 합 $S_n$ 에 대해 $S_7 = S_{11}$ 이 성립할 때 $S_{18}$ 의 값을 구하시오.
풀이 보기

$S_p = S_q \Rightarrow S_{p+q} = 0$. $7+11 = 18$ → $S_{18} = 0$.

검산: $S_n = An^2 + Bn$. $S_7 = S_{11}$ → 대칭축 $n = 9$. 원점·$18$ 두 근. ✓

정답: $0$
수능 대비 2 — 함정 결합 (꼭짓점 분수 + 음수)
문제: 등차수열 $\{a_n\}$ 의 첫째항이 $20$, 공차가 $-3$ 이다. 합 $S_n$ 이 양수가 되는 자연수 $n$ 의 최댓값을 구하시오.
풀이 보기

$S_n$ 식: $S_n = -\dfrac{3}{2}n^2 + \left(20 + \dfrac{3}{2}\right)n = -\dfrac{3}{2}n^2 + \dfrac{43}{2}n$.

$S_n > 0$ 조건: $n\left(-\dfrac{3}{2}n + \dfrac{43}{2}\right) > 0$. $n > 0$ 이므로 $-\dfrac{3}{2}n + \dfrac{43}{2} > 0 \Rightarrow n < \dfrac{43}{3} = 14.33\ldots$

자연수 $n$ 최대: $n = 14$ (이때 $S_{14}$ 양수 확인 필요).
$S_{14} = -\dfrac{3}{2}\cdot 196 + \dfrac{43}{2}\cdot 14 = -294 + 301 = 7 > 0$ ✓
$S_{15} = -\dfrac{3}{2}\cdot 225 + \dfrac{43}{2}\cdot 15 = -337.5 + 322.5 = -15 < 0$ → 안 됨.

$S_n$ 그래프 근: $n=0$ 과 $n = \dfrac{43}{3}$. 자연수 근이 아니지만 양수 구간은 그 사이. $n=14$ 까지만 $S_n > 0$.

정답: $n=14$

⚠️ 오답 주의: ① "근 $\dfrac{43}{3}$ 을 반올림해 $14$" 같이 어림짐작 ❌, 반드시 $S_{14}, S_{15}$ 부호 둘 다 확인. ② 또한 "$a_n > 0$ 인 $n$"(여기서는 $n \le 7$)과 "$S_n > 0$ 인 $n$"(여기서는 $n \le 14$) 은 다르다. $a_n$ 이 음수여도 누적합이 아직 양수일 수 있음.
