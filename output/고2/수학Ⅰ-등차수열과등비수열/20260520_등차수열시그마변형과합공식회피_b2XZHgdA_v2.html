📐 개념 — 시그마 변형 3가지 무기
⚡ 선수 학습 1분 환기
  • 등차수열 일반항: $a_n = a_1 + (n-1)d$
  • 항의 차 ★ (v1 복습): $a_n - a_m = (n-m)d$
  • 등차중항 ★ (v1 복습): 합 = 중앙값 × 개수
  • $\Sigma$ 기본: $\displaystyle\sum_{k=1}^{n} a_k = a_1 + a_2 + \cdots + a_n$
  • $\Sigma$ 선형성: $\sum(a_k + b_k) = \sum a_k + \sum b_k$, $\sum c\cdot a_k = c\sum a_k$
1$\Sigma$ 분리 — 합의 차를 항으로 환원

$\sum_{k=m+1}^{n} a_k$ 같은 부분 합은 두 누적합의 차로 분해한다.

$$\sum_{k=m+1}^{n} a_k = \sum_{k=1}^{n} a_k - \sum_{k=1}^{m} a_k = S_n - S_m$$
활용: $S_n - S_m$ 차이가 주어지면 그것이 곧 $a_{m+1} + \cdots + a_n$ 의 부분합. 그 다음 (v1의) 등차중항으로 즉시 처리.
📌 시그마 분리 → 등차중항 흐름

$\displaystyle\sum_{k=4}^{10} a_k$ 를 등차중항으로 변환하라.

항 개수
$10 - 4 + 1 = 7$ 개.
중앙항
$a_4, a_5, \ldots, a_{10}$ 중앙 = $a_7$ (4번째).
결과
$\sum_{k=4}^{10} a_k = 7 \cdot a_7$
한 줄 공식: $\sum_{k=p}^{q} a_k = (q-p+1) \cdot a_{\text{중앙}}$ — 단 등차수열에서만.
2$\Sigma$ 인덱스 이동 — 항 번호 일치시키기

$\sum a_k$와 $\sum a_{k+c}$를 비교할 때, 인덱스를 평행이동해서 같은 형태로 만든다.

$$\sum_{k=1}^{n} a_{k+c} = \sum_{j=c+1}^{n+c} a_j$$
($j = k+c$ 치환)
⚙
$\sum a_{k+1} - \sum a_k$ 패턴:
$\displaystyle\sum_{k=1}^{n}(a_{k+1} - a_k)$ = 항의 차이의 합 = $(a_2 - a_1) + (a_3 - a_2) + \cdots + (a_{n+1} - a_n) = a_{n+1} - a_1$ (망원합)
망원합 (Telescoping): 등차수열에서 인접 항 차이는 모두 공차 $d$. 따라서 $\sum_{k=1}^{n}(a_{k+1} - a_k) = nd$.
📌 시그마 차이 패턴

등차수열 $\{a_n\}$의 공차 $d=3$일 때 $\displaystyle\sum_{k=1}^{20}(a_{k+2} - a_k)$를 구하라.

항 차이
$a_{k+2} - a_k = 2d = 6$ (항 번호 차 2).
시그마
$\sum_{k=1}^{20} 6 = 20 \cdot 6 = 120$
3합 공식 회피 — 등차중항 우선, $S_n$ 공식은 최후

학교에서 배우는 합 공식 $S_n = \dfrac{n(a_1+a_n)}{2}$는 강력하지만 $a_1$을 알아야 한다는 약점이 있다. v1의 "기준항 이동"과 결합하면 합 공식 없이 처리 가능.

$$S_n = \dfrac{n(a_1 + a_n)}{2} = n \cdot \dfrac{a_1 + a_n}{2}$$
= 항 개수 × (양 끝 항의 평균) = 항 개수 × 중앙값
핵심 통찰: $\dfrac{a_1 + a_n}{2}$ 자체가 등차중항($a_{(n+1)/2}$, 홀수개 항 시)이다. 따라서 $S_n$ 공식 = (등차중항으로 합) 공식. 두 공식 = 한 공식.
상황공식필요한 정보
$a_1, a_n$ 둘 다 앎$S_n = \dfrac{n(a_1+a_n)}{2}$$a_1, a_n$
중앙값 앎$S_n = n \cdot a_{\text{중앙}}$중앙항 ★
$a_1, d$ 앎$S_n = \dfrac{n(2a_1+(n-1)d)}{2}$$a_1, d$
임의 항 $a_m, d$ 앎$S_n = n \cdot \bigl(a_m + (\tfrac{n+1}{2}-m)d\bigr)$$a_m, d$ ★
회피 전략: $a_1$를 모를 때는 알려진 임의 항 + 공차로 중앙값을 우회 계산. 첫째병 탈피!
기초 시그마 분리·인덱스 이동
기초 1 — $\Sigma$ 분리
등차수열 $\{a_n\}$의 $S_n = \sum_{k=1}^{n} a_k$가 $S_{10} = 100$, $S_5 = 30$일 때, $\sum_{k=6}^{10} a_k$의 값을 구하시오.
풀이 보기
분리
$\sum_{k=6}^{10} a_k = S_{10} - S_5 = 100 - 30 = 70$
정답: 70
기초 2 — 망원합
등차수열 $\{a_n\}$의 공차가 $d=4$일 때, $\displaystyle\sum_{k=1}^{15}(a_{k+1} - a_k)$의 값을 구하시오.
풀이 보기
항 차이
$a_{k+1} - a_k = d = 4$ (모두 동일).
합
$15 \cdot 4 = 60$
정답: 60
기초 3 — 합 공식 우회
등차수열 $\{a_n\}$에서 $a_{10} = 15$이고 공차 $d=2$일 때, $a_1 + a_2 + \cdots + a_{19}$의 값을 구하시오.
풀이 보기
중앙항
$1$부터 $19$까지 19개 항. 중앙 = $a_{10} = 15$ (이미 알고 있음!).
합
$S = 19 \cdot 15 = 285$. $a_1$ 안 구함.
v1의 첫째병 탈피 + v2의 합 공식 회피가 결합한 깔끔한 풀이.
정답: 285
기본 시그마 + 등차중항 결합
기본 1 — $\Sigma$ 차이로 $d$ 결정
등차수열 $\{a_n\}$의 $S_{20} - S_{10} = 150$일 때, $a_{15} + a_{16}$의 값을 구하시오.
풀이 보기
분리
$S_{20} - S_{10} = \sum_{k=11}^{20} a_k$ (10개 항).
짝수 개
10개 항의 중앙 두 항 = $a_{15}, a_{16}$. 합 = $10 \cdot \dfrac{a_{15}+a_{16}}{2} = 5(a_{15}+a_{16}) = 150$.
결과
$a_{15} + a_{16} = 30$
정답: 30
기본 2 — 인덱스 이동
공차 $d = 5$인 등차수열 $\{a_n\}$에서 $\displaystyle\sum_{k=1}^{10}(a_{k+3} - a_{k+1})$의 값을 구하시오.
풀이 보기
항 차이
$a_{k+3} - a_{k+1} = 2d = 10$ (항 번호 차 2).
합
$\sum_{k=1}^{10} 10 = 100$
$a_{k+3}$ 안의 $a$ 자체 식은 신경 X. 항 번호 차이만 보면 즉시 $2d$.
정답: 100
기본 3 — 합 공식 회피 종합
등차수열 $\{a_n\}$에서 $a_7 = 10$, $a_{13} = 22$일 때 $\displaystyle\sum_{k=4}^{16} a_k$의 값을 구하시오.
풀이 보기
$d$
$(a_{13} - a_7) = 6d = 12$ → $d = 2$.
$\Sigma$ 항 개수
$16 - 4 + 1 = 13$개.
중앙항
$a_4, a_5, \ldots, a_{16}$ 중앙 = $a_{10}$.
$a_{10}$ — 기준항 이동
$a_7$ 기준: $a_{10} = a_7 + 3d = 10 + 6 = 16$.
합
$13 \cdot 16 = 208$
$a_1$ 한 번도 안 등장. v1(항의 차 + 기준항 이동) × v2(중앙항으로 합) 콤보.
정답: 208
심화 시그마 변형 종합
심화 1 — $\Sigma$ 분리 + $d$ 추출
등차수열 $\{a_n\}$의 합 $S_n$이 $S_{12} = 84$, $S_8 = 36$일 때, 공차 $d$를 구하시오.
풀이 보기
$\Sigma$ 분리
$S_{12} - S_8 = \sum_{k=9}^{12} a_k = 4$개 항 = 84 - 36 = 48.
중앙 두 항 평균 × 4: $4 \cdot \dfrac{a_{10}+a_{11}}{2} = 48$ → $a_{10} + a_{11} = 24$.
$S_8$ 활용
$S_8 = 8 \cdot \dfrac{a_4 + a_5}{2} = 4(a_4 + a_5) = 36$ → $a_4 + a_5 = 9$.
차이로 $d$
$(a_{10} - a_4) + (a_{11} - a_5) = 6d + 6d = 12d$
$24 - 9 = 15 = 12d$ → $d = \dfrac{15}{12} = \dfrac{5}{4}$
정답: $d = \dfrac{5}{4}$
심화 2 — 인덱스 평행이동
등차수열 $\{a_n\}$의 공차 $d=3$이고 $a_1 = 4$일 때, $\displaystyle\sum_{k=1}^{10} a_{2k}$ (즉 $a_2, a_4, \ldots, a_{20}$의 합)을 구하시오.
풀이 보기
$b_k = a_{2k}$ 정의
$\{b_k\}$도 등차수열. 공차 $= a_{2k+2} - a_{2k} = 2d = 6$.
첫째항·끝항
$b_1 = a_2 = 4 + 3 = 7$. $b_{10} = a_{20} = 4 + 19\cdot 3 = 61$.
중앙값
10개 항 짝수. 중앙 두 항 평균 $= \dfrac{b_5 + b_6}{2}$.
$b_5 = a_{10} = 4 + 27 = 31$, $b_6 = a_{12} = 4 + 33 = 37$.
평균 = $34$.
합
$10 \cdot 34 = 340$
정답: 340
수능 대비 준킬러 — 21번 유형
수능 대비 1 — 21번 유형 (시그마 변형 + 등차중항)
등차수열 $\{a_n\}$이 다음 조건을 만족시킨다.

(가) $\displaystyle\sum_{k=1}^{9} a_k = 36$
(나) $\displaystyle\sum_{k=10}^{18} a_k = 117$

$a_5 + a_{14}$의 값을 구하시오.
풀이 보기
(가) — 9개 항
중앙항 $a_5$. $9 a_5 = 36$ → $a_5 = 4$.
(나) — 9개 항
$a_{10}, \ldots, a_{18}$의 중앙 = $a_{14}$. $9 a_{14} = 117$ → $a_{14} = 13$.
합
$a_5 + a_{14} = 4 + 13 = \mathbf{17}$
$d$ 추출도 가능: $a_{14} - a_5 = 9d = 13 - 4 = 9$ → $d = 1$. 그러나 답은 $d$ 안 써도 즉시.
정답: 17
수능 대비 2 — 준킬러 (망원합 응용)
등차수열 $\{a_n\}$에서 $a_1 = 2$, 공차 $d = 4$일 때 $\displaystyle\sum_{k=1}^{n}\dfrac{1}{a_k\cdot a_{k+1}}$를 $n$의 식으로 나타내시오.
풀이 보기
부분분수 분해
$\dfrac{1}{a_k a_{k+1}} = \dfrac{1}{d}\bigl(\dfrac{1}{a_k} - \dfrac{1}{a_{k+1}}\bigr)$ (등차수열의 망원 패턴)
검산
$\dfrac{1}{d}\bigl(\dfrac{1}{a_k} - \dfrac{1}{a_{k+1}}\bigr) = \dfrac{a_{k+1} - a_k}{d \cdot a_k a_{k+1}} = \dfrac{d}{d \cdot a_k a_{k+1}} = \dfrac{1}{a_k a_{k+1}}$ ✔
망원합
$\dfrac{1}{4}\displaystyle\sum_{k=1}^{n}\bigl(\dfrac{1}{a_k} - \dfrac{1}{a_{k+1}}\bigr) = \dfrac{1}{4}\bigl(\dfrac{1}{a_1} - \dfrac{1}{a_{n+1}}\bigr)$
대입
$a_1 = 2$, $a_{n+1} = 2 + 4n$.
$= \dfrac{1}{4}\bigl(\dfrac{1}{2} - \dfrac{1}{2+4n}\bigr) = \dfrac{1}{4}\cdot\dfrac{4n}{2(2+4n)} = \dfrac{n}{2(2+4n)} = \dfrac{n}{4(1+2n)}$
핵심: 등차수열의 항 곱의 역수 합은 항상 망원합으로 풀린다. 공차 $d$가 분모에 등장.
정답: $\dfrac{n}{4(2n+1)}$
수능 대비 3 — 준킬러 (조건 → 미지수)
등차수열 $\{a_n\}$이 다음 조건을 만족시킨다.

(가) $a_3 + a_{17} = 24$
(나) $\displaystyle\sum_{k=5}^{15} a_k = 132$

공차 $d$와 $a_1$의 값을 구하시오.
풀이 보기
(가) → $a_{10}$
$a_3 + a_{17} = 2 a_{10}$ (등차중항). $a_{10} = 12$.
(나) → $a_{10}$ 재확인
$a_5 \sim a_{15}$의 11개 항 중앙 = $a_{10}$. $11 a_{10} = 132$ → $a_{10} = 12$ ✔ (일관).
$d$ 결정 — 정보 부족?
두 조건 모두 $a_{10}$만 결정. $d$는 단일 결정 안 됨 → 추가 조건 필요.
실전 풀이: 문제가 $d$ 유일성 보장한다면 추가 조건이 있어야. 학습용으로 "$d$ 미결정" 결론.
$a_1$ — $d$ 가정 시
$d=1$ 가정 시 $a_1 = a_{10} - 9d = 12 - 9 = 3$.
$d=2$ 가정 시 $a_1 = 12 - 18 = -6$.
학습 포인트: 두 조건이 같은 정보(중앙항 $a_{10}$)를 가리키면 새 정보 없음. 시그마 변형 시 중복 정보 검출이 1등급의 안목.
학습용 — $a_{10}=12$ 확정, $d$ 추가 조건 필요

① $\Sigma$ 보이면 즉시 항 개수·중앙항 표기.

③ 항 차이 시그마 → 망원합 후보.
