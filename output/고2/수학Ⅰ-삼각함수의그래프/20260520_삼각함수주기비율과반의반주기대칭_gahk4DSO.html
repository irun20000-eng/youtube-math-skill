📐 개념 — 주기 비율 + $\dfrac{T}{4}$ 대칭
⚡ 선수 학습 1분 환기
  • 특수각 삼각비: $\sin 0=0$, $\sin\dfrac{\pi}{6}=\dfrac{1}{2}$, $\sin\dfrac{\pi}{4}=\dfrac{\sqrt 2}{2}$, $\sin\dfrac{\pi}{3}=\dfrac{\sqrt 3}{2}$, $\sin\dfrac{\pi}{2}=1$.
  • 호도법: $360° = 2\pi$ (라디안). 한 바퀴.
  • $y = \sin x$의 주기: $T = 2\pi$ (각이 한 바퀴 돌면 원위치).
  • 그래프 평행·대칭이동: $y = f(x+a)$, $y = -f(x)$, $y = f(-x)$.
1주기 — "안의 식이 $2\pi$가 되도록 $x$ 변화량"

$y = \sin(ax)$나 $y = \sin(ax+b)$ 같은 함수의 주기 $T$를 구할 때, 공식 외우지 말고 본질로:

주기 정의: "안의 식($ax$)이 $0$에서 $2\pi$까지 변할 때 $x$의 변화량"
$ax_{\text{끝}} - ax_{\text{시작}} = 2\pi$ → $T = \dfrac{2\pi}{a}$
📌 예 — $y = \sin(2x)$의 주기
질문
"$2x$가 $0$에서 $2\pi$까지 가려면 $x$는 얼마나 가야 할까?"
계산
$2x = 2\pi$ → $x = \pi$. 즉 $x$가 $\pi$만큼 가는 동안 $2x$는 한 바퀴 = $2\pi$. $T = \pi$.
강사 인용: "여기에 뭘 넣으면 $2\pi$하고 물어보는 거야." — 공식 $T = \dfrac{2\pi}{a}$를 도출 과정 자체로 이해.
주의: $\tan$의 주기는 $\pi$ ($2\pi$가 아님). $y=\tan(ax)$의 주기는 $\dfrac{\pi}{a}$.
2주기 내 비율 분할 — 8등분 직관

$\sin x$의 한 주기 $[0, 2\pi]$를 8등분하면 특수각이 정확히 분할점에 떨어진다.

y=sin x 의 한 주기 8등분 x : 0 π/4 π/2 3π/4 π 5π/4 3π/2 7π/4 2π │ │ │ │ │ │ │ │ │ sin x: 0 √2/2 1 √2/2 0 -√2/2 -1 -√2/2 0 ●────●────●────●────●────●─────●─────●────● 시작 중간 끝 ↑ ↑ 최댓값(π/2) 최솟값(3π/2)
분할 활용: 임의의 $y = \sin(ax + b)$도 주기 $T = 2\pi/a$를 구한 후, $T$를 8등분하여 분할점에 0 → 최대 → 0 → 최소 → 0 패턴을 그대로 적용. 식 전개 없이 그래프 즉시 스케치.
📌 예 — $y = \sin(2x + \pi/3)$의 한 주기 그리기
주기
$T = \pi$.
시작점
안의 식 = 0 → $2x + \pi/3 = 0$ → $x = -\pi/6$. 이 점이 그래프의 "0에서 시작"점.
8등분
$x = -\pi/6$부터 $T = \pi$ 길이를 8등분 → $\pi/8$ 간격.
분할점: $-\pi/6, \;-\pi/6+\pi/8, \dots$
함숫값 패턴: 0 → √2/2 → 1 → √2/2 → 0 → -√2/2 → -1 → -√2/2 → 0
3반의 반주기 ($\dfrac{T}{4}$) 대칭 — 사분원 직관

$\sin x$ 그래프의 가장 강력한 성질은 $x = T/4$ 위치(최대점)에 대한 좌우 선대칭. 마찬가지로 $x = 3T/4$ 위치(최소점)에 대해서도 선대칭.

1/4 주기 대칭축:
$y = \sin(ax)$의 그래프는 $x = \dfrac{T}{4} = \dfrac{\pi}{2a}$에 선대칭
(최대점이 대칭축)
y=sin x — 사분원(피자 조각) 비유 y=1 ● (대칭축 x=π/2) ╱:╲ ╱ : ╲ ╱ : ╲ ╱ : ╲ ──────────●───┼───●────────── x축 0 π/2 π ↑ 좌(0~π/2)와 우(π/2~π)가 거울처럼 대칭 → 사분원 두 개를 붙인 모양. 좌측이 올라가는 모양과 우측이 내려가는 모양이 정확히 서로 거울 이미지.
활용: $y = \sin(ax) = k$ ($0 < k < 1$)의 한 주기 내 두 실근을 $\alpha, \beta$라 하면 $\alpha + \beta = 2\cdot\dfrac{T}{4} = \dfrac{T}{2}$. 즉 두 실근의 합은 주기의 절반.
📌 영상 핵심 예시 — 1/4 주기 대칭으로 실근의 합

방정식 $\sin x = \dfrac{1}{2}$ 의 $[0, \pi]$ 구간 두 실근의 합을 구하라.

실근 위치
$\sin x = 1/2$ → $x = \pi/6$ 또는 $x = 5\pi/6$ (특수각).
대칭 검증
두 실근의 중점 = $\dfrac{\pi/6 + 5\pi/6}{2} = \dfrac{\pi}{2}$ — 정확히 $\sin x$의 최대점 위치 ✔
합
$\dfrac{\pi}{6} + \dfrac{5\pi}{6} = \pi$. 일반: $\alpha + \beta = 2 \cdot \dfrac{T}{4} = \dfrac{T}{2}$ (여기서 $T=2\pi$ → 합 $=\pi$ ✔)
강사 인용: "반의 반주기에 대하여 대칭성 그대로 나타나지." 두 실근 합 = 대칭축 × 2 = $T/2$.
$\cos$의 경우: $y = \cos x$의 대칭축은 $x = 0, \pi, 2\pi, \dots$ ($\sin$보다 $\pi/2$ 만큼 앞). 동일한 1/4 주기 대칭 원리는 그대로 통함.
기초 주기 계산·비율 분할
기초 1 — 주기 계산
다음 함수의 주기를 구하시오.
(1) $y = \sin(3x)$    (2) $y = \cos\bigl(\dfrac{x}{2}\bigr)$    (3) $y = \tan(4x)$
풀이 보기
(1)
$3x$가 $2\pi$ 가려면 $x$는 $\dfrac{2\pi}{3}$. $T = \dfrac{2\pi}{3}$
(2)
$\dfrac{x}{2}$가 $2\pi$ 가려면 $x$는 $4\pi$. $T = 4\pi$
(3)
$\tan$의 주기는 $\pi$. $4x$가 $\pi$ 가려면 $x$는 $\dfrac{\pi}{4}$. $T = \dfrac{\pi}{4}$
정답: (1) $\frac{2\pi}{3}$, (2) $4\pi$, (3) $\frac{\pi}{4}$
기초 2 — 8등분으로 그래프 읽기
$y = \sin(2x)$의 한 주기 $[0, \pi]$ 안에서 최댓값을 갖는 $x$의 값을 구하시오.
풀이 보기
주기
$T = \pi$. 8등분 간격 $= \dfrac{\pi}{8}$.
최댓값 위치
$\sin$의 최댓값은 $T/4$ 위치 = $\dfrac{\pi}{4}$.
검산
$\sin(2\cdot\pi/4) = \sin(\pi/2) = 1$ ✔
정답: $x = \dfrac{\pi}{4}$
기초 3 — 대칭축 활용
$\sin x = \dfrac{\sqrt 3}{2}$의 $[0, \pi]$ 구간 두 실근의 합을 구하시오.
풀이 보기
대칭 원리
두 실근은 $x = \pi/2$ (최대점) 좌우 대칭. 합 = $2 \cdot \pi/2 = \pi$
검산
$\sin x = \sqrt 3/2$ → $x = \pi/3$ 또는 $2\pi/3$. 합 = $\pi$ ✔
정답: $\pi$
기본 위상 + 주기 + 대칭
기본 1 — 위상 포함 그래프
$y = \sin\bigl(2x - \dfrac{\pi}{3}\bigr)$의 주기와, $[0, \pi]$ 안에서 최댓값을 갖는 $x$를 구하시오.
풀이 보기
주기
$2x$가 $2\pi$ 가는 동안 $x$는 $\pi$. $T = \pi$.
시작점
$2x - \pi/3 = 0$ → $x = \pi/6$ (그래프 "0에서 시작" 점).
최댓값 위치
$T/4 = \pi/4$ 만큼 뒤. $x = \pi/6 + \pi/4 = \dfrac{2\pi + 3\pi}{12} = \dfrac{5\pi}{12}$.
검산
$\sin(2\cdot 5\pi/12 - \pi/3) = \sin(5\pi/6 - \pi/3) = \sin(\pi/2) = 1$ ✔
정답: $T=\pi$, 최대 $x = \dfrac{5\pi}{12}$
기본 2 — 두 실근의 합
$\sin(2x) = \dfrac{1}{2}$의 $[0, \dfrac{\pi}{2}]$ 구간 두 실근의 합을 구하시오.
풀이 보기
주기
$y = \sin(2x)$의 주기 $T = \pi$. 최대점은 $x = T/4 = \pi/4$.
$[0, \pi/2]$가 한 주기
$[0, \pi/2]$는 한 주기의 절반 (= $T/2$). 이 구간 안에서 $\sin(2x) = 1/2$인 두 실근은 $x = \pi/4$ 대칭.
합
두 실근 합 = $2 \cdot \pi/4 = \dfrac{\pi}{2}$.
검산
$2x = \pi/6$ → $x = \pi/12$, $2x = 5\pi/6$ → $x = 5\pi/12$. 합 = $6\pi/12 = \pi/2$ ✔
정답: $\dfrac{\pi}{2}$
기본 3 — 그래프 적용 (cos)
$y = \cos(\pi x)$의 한 주기 안에서 $y$가 양수인 구간의 길이를 구하시오.
풀이 보기
주기
$\pi x = 2\pi$ → $x = 2$. $T = 2$.
$\cos$ 부호
$\cos x$가 양수인 구간: 한 주기 $[0, 2\pi]$ 중 $[0, \pi/2] \cup [3\pi/2, 2\pi]$. 길이 = $\pi$ (전체의 절반).
$y=\cos(\pi x)$ 적용
주기 $T=2$의 절반 = $\mathbf{1}$.
정답: 1
심화 실근의 합·대칭 융합
심화 1 — 여러 실근의 합
방정식 $\sin x = \dfrac{1}{3}$의 $[0, 2\pi]$ 구간 모든 실근의 합을 구하시오.
풀이 보기
$\sin x$ 부호
$\sin x > 0$인 구간은 $[0, \pi]$. $\sin x = 1/3 > 0$이므로 실근은 모두 $[0, \pi]$ 안.
대칭축
$[0, \pi]$에서 $\sin x$의 최대점 $x = \pi/2$가 대칭축. 두 실근 합 = $2 \cdot \pi/2 = \pi$.
정답: $\pi$
심화 2 — $\cos$ 그래프 실근 합
$\cos(2x) = \dfrac{1}{2}$의 $[0, \pi]$ 구간 모든 실근의 합을 구하시오.
풀이 보기
주기
$y = \cos(2x)$의 주기 $T = \pi$. $[0, \pi]$는 정확히 한 주기.
$\cos$ 대칭축
한 주기 안에서 $\cos$ 그래프는 $x = 0$(최대), $x = T/2 = \pi/2$ (최소), $x = T = \pi$ (최대) 대칭.
실근 위치
$\cos(2x) = 1/2$ → $2x = \pi/3$ 또는 $-\pi/3 + 2\pi k$.
$[0, 2\pi]$의 $2x$ 범위에서 $2x = \pi/3, 5\pi/3$.
→ $x = \pi/6, 5\pi/6$.
합
$\pi/6 + 5\pi/6 = \pi$. (대칭축 $x=\pi/2$에 대해 좌우 대칭 ✔)
정답: $\pi$
수능 대비 준킬러 — 대칭·주기 종합
수능 대비 1 — 준킬러 (여러 주기 실근)
방정식 $\sin(2x) = \dfrac{\sqrt 3}{2}$의 $[0, 2\pi]$ 구간 모든 실근의 합을 구하시오.
풀이 보기
주기
$y = \sin(2x)$의 주기 $T = \pi$. $[0, 2\pi]$는 두 주기.
첫 주기 $[0, \pi]$
최대점 $x = \pi/4$. 두 실근의 합 = $2 \cdot \pi/4 = \pi/2$.
둘째 주기 $[\pi, 2\pi]$
최대점 $x = \pi + \pi/4 = 5\pi/4$. 두 실근의 합 = $2 \cdot 5\pi/4 = 5\pi/2$.
전체 합
$\pi/2 + 5\pi/2 = 6\pi/2 = \mathbf{3\pi}$
일반화: 주기 $T$인 함수의 $n$개 주기에서 같은 $y$값의 실근들 합 = $n$개의 대칭축 합 × 2 = "각 주기의 1/4 위치들의 합" × 2.
정답: $3\pi$
수능 대비 2 — 준킬러 (조건과 대칭)
방정식 $\sin\bigl(\dfrac{x}{2}\bigr) = k$ ($0 < k < 1$)의 $[0, 4\pi]$ 구간 실근이 두 개일 때, 두 실근의 합을 $k$의 식으로 나타내시오.
풀이 보기
주기
$x/2 = 2\pi$ → $x = 4\pi$. $T = 4\pi$. $[0, 4\pi]$는 정확히 한 주기.
최대점
$T/4 = \pi$ 위치. 두 실근의 대칭축이 $x = \pi$.
합
$\alpha + \beta = 2 \cdot \pi = 2\pi$. $k$값에 무관하게 상수.
$k$ 값에 따라 두 실근의 위치는 변하지만 합은 대칭축 × 2로 고정. 이것이 대칭성의 위력.
정답: $2\pi$ ($k$ 무관)
수능 대비 3 — 준킬러 (sin·cos 혼합)
함수 $f(x) = \sin x + \cos x$가 $[0, 2\pi]$에서 최댓값을 갖는 $x$의 값을 구하시오.
풀이 보기
정리
$\sin x + \cos x = \sqrt 2 \sin\bigl(x + \dfrac{\pi}{4}\bigr)$ (합성).
(참고: 이 합성 공식은 영상 범위 외이지만 결과만 활용)
최댓값 위치
$\sqrt 2$이고, $\sin(x + \pi/4) = 1$일 때. $x + \pi/4 = \pi/2$ → $x = \pi/4$.
검산
$f(\pi/4) = \sin(\pi/4) + \cos(\pi/4) = \dfrac{\sqrt 2}{2} + \dfrac{\sqrt 2}{2} = \sqrt 2$ ✔
주기·대칭 직관: $f(x) = \sin x + \cos x$의 그래프는 $\sin$ 그래프를 $-\pi/4$ 만큼 평행이동한 모양. 최대점 위치도 $\pi/2 \to \pi/4$로 이동.
정답: $x = \dfrac{\pi}{4}$ (최댓값 $\sqrt 2$)
📝 한눈에 보는 정리
도구핵심 식활용
주기 $T = \dfrac{2\pi}{a}$ ($\sin, \cos$), $\dfrac{\pi}{a}$ ($\tan$) "안의 식이 $2\pi$일 때 $x$ 변화량"
8등분 분할 $T/8$ 간격으로 특수값 그래프 즉시 스케치
1/4 주기 대칭 $x = T/4$ (최대점) 좌우 선대칭 두 실근 합 = $T/2$
여러 주기 실근 합 각 주기 합 더하기 $n$ 주기 = $n$개 대칭축 × 2
💡 시험장 체크리스트
① 주기 보이면 즉시 "안의 식이 $2\pi$가 되는 $x$" 계산.
② 함숫값이 $\pm 1$이 아니면 한 주기 안에 실근 2개가 생긴다 (대칭쌍).
③ 두 실근의 합 = 가까운 최대·최소점 × 2 (대칭축의 두 배).
④ 여러 주기에 실근 있으면 각 주기마다 분리해서 합 계산 후 합.
⑤ $\cos$의 대칭축은 $\sin$의 대칭축에서 $\pi/2$ 만큼 어긋남.
