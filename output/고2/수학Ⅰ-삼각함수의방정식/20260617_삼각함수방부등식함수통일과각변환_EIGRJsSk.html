📌 도입: 영상이 던진 질문

예시 — 영상의 핵심 문제 (수능 대비형)

문제: $2$ 이상 $10$ 이하의 두 자연수 $a, b$ 가 다음 조건을 만족할 때, 순서쌍 $(a, b)$ 의 개수는?

(가) $a+b$ 는 짝수이다.
(나) $\sin\!\left(a\pi + \dfrac{\pi}{b}\right) \le \cos\!\left(b\pi + \dfrac{\pi}{a}\right)$

📖 풀이 보기 (영상의 결론)

STEP 1 — 분기: $a+b$ 짝수 → (둘 다 짝수) 또는 (둘 다 홀수).

STEP 2 — 둘 다 짝수: 짝수 $\pi$ 는 주기 $2\pi$ 의 정수배라 소거. $\sin\!\left(\dfrac{\pi}{b}\right) \le \cos\!\left(\dfrac{\pi}{a}\right)$. 우변을 각 변환 $\cos\theta = \sin(\pi/2 - \theta)$ 로 sin 통일: $\sin\!\left(\dfrac{\pi}{b}\right) \le \sin\!\left(\dfrac{\pi}{2} - \dfrac{\pi}{a}\right)$. 두 각 모두 1사분면 → $\dfrac{\pi}{b} \le \dfrac{\pi}{2} - \dfrac{\pi}{a}$ → $\dfrac{1}{a} + \dfrac{1}{b} \le \dfrac{1}{2}$. 짝수 $\{4,6,8,10\}$ 어느 조합이든 만족 → $4\times4 = 16$ 개.

STEP 3 — 둘 다 홀수: 홀수 $\pi$ 소거 시 부호 반전 → $-\sin\!\left(\dfrac{\pi}{b}\right) \le -\cos\!\left(\dfrac{\pi}{a}\right)$. 양변 $\times(-1)$ 부등호 뒤집기: $\dfrac{1}{a} + \dfrac{1}{b} \ge \dfrac{1}{2}$. 홀수 자연수 대입 → $(3,3), (3,5), (5,3)$ → $3$ 개.

정답: $16 + 3 = \mathbf{19}$

핵심 포인트: 그래프 $\sin 5x$, $\cos 3x$ 를 그릴 생각부터 버릴 것. ① 정수 $\pi$ 소거 ② 한 함수로 통일 ③ 같은 사분면에서 단순 비교 — 세 도구가 전부다.

💡 개념 설명 (함수의 통일 · 각 변환 · 사분면)

① 큰 그림 — "처음 5초의 판단"

방정식 $\sin x = \sin 2x$ 는 그래프 두 개를 그려서 교점으로 풀 만합니다. 그런데 $\sin 2x = \sin 5x$ 처럼 각이 커지면? $\sin 5x$ 같은 건 우리는 안 그려요.

강사 메시지: "하나의 함수에 서로 다른 값 두 개가 들어갔다" 고 관점을 바꾸자. 들어간 두 값이 주기 · 대칭 관계에 놓이는지 보는 것이 훨씬 빠릅니다.

$\sin 2x = \cos 3x$ 처럼 사인 · 코사인이 섞이고 각도 다르면 더 불편 → 각 변환으로 함수를 통일해 한 함수로 만들고 끝.

처음 5초에 결정해야 할 것은 그래프를 그릴 것인가, 통일 · 각변환으로 식을 다룰 것인가. 복잡한 각이 보이면 거의 항상 후자.

② 선수 지식 1 — 주기성과 정수 $\pi$ 소거

$\sin(x + 2\pi) = \sin x$, $\cos(x + 2\pi) = \cos x$ — 주기 $2\pi$.

그래서 $\sin(2n\pi + \theta) = \sin\theta$, $\cos(2n\pi + \theta) = \cos\theta$ (짝수 $\pi$ 는 그냥 소거).

홀수 $\pi$ 는? $\sin(\pi + \theta) = -\sin\theta$, $\cos(\pi + \theta) = -\cos\theta$ → 부호 반전.

한 줄: 각 안의 $n\pi$ 는 $n$ 의 짝/홀에 따라 소거 vs 부호반전.

영상 문제의 첫 단추 — $\sin(a\pi + \pi/b)$ 의 $a\pi$ 는 $a$ 짝수면 그냥 사라지고, $a$ 홀수면 부호가 뒤집힌다. 이걸 먼저 정리해야 식이 보입니다.

③ 선수 지식 2 — 각 변환 공식 (단위원에서)

$\sin\!\left(\dfrac{\pi}{2} - \theta\right) = \cos\theta$, $\cos\!\left(\dfrac{\pi}{2} - \theta\right) = \sin\theta$ (여각 관계).

$\sin\!\left(\dfrac{\pi}{2} + \theta\right) = \cos\theta$, $\cos\!\left(\dfrac{\pi}{2} + \theta\right) = -\sin\theta$.

왜 그런가? — 단위원에서 한 점을 $\pi/2$ 만큼 회전하면 $x, y$ 좌표가 뒤바뀝니다. 회전 방향(+ 또는 -)에 따라 부호가 달라질 뿐.

활용: $\cos B$ 를 $\sin$ 으로 갈아치우는 두 가지 — $\sin(\pi/2 - B)$ 또는 $\sin(\pi/2 + B)$. 둘 다 가능 하고 어느 쪽을 써도 결국 같은 답이 나옵니다 (대칭성). 다만 어느 쪽 쓰느냐에 따라 참조할 사분면이 달라짐.

④ 핵심 정리 1 — 함수의 통일 (cos → sin)

$\sin A \le \cos B$ 같은 부등식에서 두 함수를 그리는 대신 한 함수로 갈무리:

$\cos B = \sin\!\left(\dfrac{\pi}{2} - B\right)$ 대입 → $\sin A \le \sin\!\left(\dfrac{\pi}{2} - B\right)$

이제 좌 · 우 둘 다 $\sin$. 같은 함수 안의 두 값 비교 문제로 환원됨.

함수만 통일되면 부등식 풀이는 두 단계 — (1) 두 각이 같은 사분면에 들어가는지 확인 (2) 그 사분면의 증감으로 단순 비교. 그래프 안 그려도 됨.

⑤ 핵심 정리 2 — 같은 사분면에서 단순 비교

$\sin$ 은 사분면마다 증감 방향이 다릅니다.

1사분면 ($0 \le \theta \le \pi/2$): $\sin\theta$ 는 증가. 각 큰 쪽이 sin 값 큼.

2사분면 ($\pi/2 \le \theta \le \pi$): $\sin\theta$ 는 감소. 각 큰 쪽이 sin 값 작음.

전제 — 두 각이 같은 사분면 안에 있어야 단순 대소 비교가 됩니다.

영상 문제 둘 다 짝수 케이스: $\dfrac{\pi}{b}$, $\dfrac{\pi}{2} - \dfrac{\pi}{a}$ 둘 다 1사분면 (양수, $\pi/2$ 이하).

$\sin$ 단조증가 적용 → $\dfrac{\pi}{b} \le \dfrac{\pi}{2} - \dfrac{\pi}{a}$.

이항해서 $\dfrac{1}{a} + \dfrac{1}{b} \le \dfrac{1}{2}$ 한 줄로 정리.

⑥ 핵심 정리 3 — 다른 각 변환을 써도 결과는 같다

$\cos(\pi/a) = \sin(\pi/2 + \pi/a)$ 를 대신 쓰면 어떻게 되나? — 이번엔 우변 각이 2사분면.

식: $\sin(\pi/b) \le \sin(\pi/2 + \pi/a)$. 좌변은 1사분면, 우변은 2사분면.

2사분면 $\sin$ 의 대칭성: $\sin(\pi/2 + \pi/a) = \sin(\pi - (\pi/2 + \pi/a)) = \sin(\pi/2 - \pi/a)$.

결국 좌변과 우변 모두 1사분면 비교로 환원 → 같은 부등식 $\dfrac{1}{a} + \dfrac{1}{b} \le \dfrac{1}{2}$.

어느 각 변환을 골라도 결국 같은 결론. 다만 중간 사분면 확인 한 번이 더 들어갑니다. 빠르고 안전한 선택은 $\sin(\pi/2 - \theta)$ — 한 번에 1사분면 정렬.

⑦ 함정 — 흔히 놓치는 4가지

① 홀수 $\pi$ 의 부호 반전 잊기. $\sin(3\pi + \theta) = -\sin\theta$. 영상 문제 "둘 다 홀수" 케이스에서 부등호가 뒤집히는 이유.

② 사분면 미확인. 두 각이 다른 사분면에 있으면 단순 대소 비교 불가. 한 번 더 대칭성으로 같은 사분면으로 옮겨야 함.

③ 각 변환 선택이 문제 풀이의 성패를 가르는 게 아님. 어느 쪽 골라도 풀린다 — 단 참조 사분면이 달라질 뿐.

④ "그래프 그리려 함" — 처음 5초에 멈춰서 식 자체를 정리할 수 있는지 먼저 살핀다. 복잡한 각이 보이면 거의 항상 그래프 ❌.

🟢 기초

기초 1 — 정수 $\pi$ 소거 (짝수)

문제: $\sin\!\left(4\pi + \dfrac{\pi}{3}\right)$ 의 값을 구하시오.

📖 풀이 보기

STEP 1: $4\pi = 2 \cdot 2\pi$ 는 $\sin$ 의 주기 $2\pi$ 의 정수배 → 그냥 소거.

STEP 2: $\sin\!\left(4\pi + \dfrac{\pi}{3}\right) = \sin\!\left(\dfrac{\pi}{3}\right) = \dfrac{\sqrt{3}}{2}$.

정답: $\dfrac{\sqrt{3}}{2}$

기초 2 — 각 변환 ($\cos \to \sin$)

문제: $\cos\theta$ 를 $\sin$ 함수의 두 가지 형태로 표현하시오 ($\theta$ 는 일반각).

📖 풀이 보기

여각 관계: $\cos\theta = \sin\!\left(\dfrac{\pi}{2} - \theta\right)$.

또는 평행이동: $\cos\theta = \sin\!\left(\dfrac{\pi}{2} + \theta\right)$.

정답: $\cos\theta = \sin\!\left(\dfrac{\pi}{2} - \theta\right) = \sin\!\left(\dfrac{\pi}{2} + \theta\right)$

핵심: 둘 다 맞는데, 변환 후 참조할 사분면이 다름. $\pi/2 - \theta$ 면 1사분면, $\pi/2 + \theta$ 면 2사분면.

🔵 기본

기본 1 — 정수 $\pi$ 소거 (홀수, 부호 반전)

문제: $\sin\!\left(5\pi + \dfrac{\pi}{4}\right)$ 의 값을 구하시오.

📖 풀이 보기

STEP 1: $5\pi = 4\pi + \pi = 2\cdot 2\pi + \pi$. 주기 $4\pi$ 는 소거되고 $\pi$ 만 남음.

STEP 2: $\sin\!\left(\pi + \dfrac{\pi}{4}\right) = -\sin\!\left(\dfrac{\pi}{4}\right)$ (홀수 $\pi$ 부호 반전).

STEP 3: $-\sin(\pi/4) = -\dfrac{\sqrt{2}}{2}$.

정답: $-\dfrac{\sqrt{2}}{2}$

오답주의: "$5\pi$ 도 다 소거" ❌ — $5$ 가 홀수라 $\pi$ 가 하나 남음. 그 $\pi$ 가 부호를 뒤집습니다.

기본 2 — 함수의 통일로 방정식 풀기

문제: $0 \le x < 2\pi$ 에서 $\sin x = \cos x$ 를 만족하는 $x$ 를 모두 구하시오.

📖 풀이 보기

STEP 1 — 함수 통일: $\cos x = \sin(\pi/2 - x)$ 대입 → $\sin x = \sin(\pi/2 - x)$.

STEP 2 — 두 사인이 같을 조건:

(i) 각이 같음 ($+2n\pi$): $x = \pi/2 - x + 2n\pi$ → $2x = \pi/2 + 2n\pi$ → $x = \pi/4 + n\pi$.

(ii) 보각 ($\sin\theta = \sin(\pi-\theta)$): $x = \pi - (\pi/2 - x) + 2n\pi$ → $x = \pi/2 + x + 2n\pi$ — 정수 해 없음.

STEP 3 — 범위 적용: $0 \le x < 2\pi$ 에서 $n=0,1$: $x = \pi/4$, $x = 5\pi/4$.

정답: $x = \dfrac{\pi}{4}$ 또는 $x = \dfrac{5\pi}{4}$

검산: $\sin(\pi/4) = \cos(\pi/4) = \sqrt{2}/2$ ✓, $\sin(5\pi/4) = \cos(5\pi/4) = -\sqrt{2}/2$ ✓.

🟠 심화

심화 1 — 함수 통일로 부등식의 자연수 풀이

문제: $\sin\!\left(\dfrac{\pi}{a}\right) \le \cos\!\left(\dfrac{\pi}{b}\right)$ 를 만족하는 $2$ 이상 $10$ 이하 자연수 순서쌍 $(a, b)$ 의 개수를 구하시오.

📖 풀이 보기

STEP 1 — 함수 통일: 우변 $\cos(\pi/b) = \sin(\pi/2 - \pi/b)$. 식: $\sin(\pi/a) \le \sin(\pi/2 - \pi/b)$.

STEP 2 — 사분면 확인: $a, b \ge 2$ → $\pi/a \le \pi/2$ (1사분면), $\pi/2 - \pi/b \in (0, \pi/2]$ (1사분면). 둘 다 1사분면이라 $\sin$ 단조증가 적용.

STEP 3 — 대소 비교: $\dfrac{\pi}{a} \le \dfrac{\pi}{2} - \dfrac{\pi}{b}$ → $\dfrac{1}{a} + \dfrac{1}{b} \le \dfrac{1}{2}$.

STEP 4 — 자연수 풀이: $a=2$ 면 $1/a = 1/2$ 이라 $1/b \le 0$ 불가. $a \ge 3$ 일 때 가능한 $b$ 확인:

$a=3$: $1/b \le 1/6$ → $b \ge 6$ → $b \in \{6,7,8,9,10\}$ — 5개.
$a=4$: $1/b \le 1/4$ → $b \ge 4$ → $b \in \{4,5,...,10\}$ — 7개.
$a=5$: $1/b \le 3/10$ → $b \ge 10/3 \approx 3.33$ → $b \in \{4,...,10\}$ — 7개.
$a=6$: $1/b \le 1/3$ → $b \ge 3$ → $b \in \{3,...,10\}$ — 8개.
$a=7$: $1/b \le 5/14$ → $b \ge 14/5 = 2.8$ → $b \in \{3,...,10\}$ — 8개.
$a=8$: $1/b \le 3/8$ → $b \ge 8/3 \approx 2.67$ → $b \in \{3,...,10\}$ — 8개.
$a=9$: $1/b \le 7/18$ → $b \ge 18/7 \approx 2.57$ → $b \in \{3,...,10\}$ — 8개.
$a=10$: $1/b \le 2/5$ → $b \ge 5/2 = 2.5$ → $b \in \{3,...,10\}$ — 8개.

합: $5 + 7 + 7 + 8 + 8 + 8 + 8 + 8 = \mathbf{59}$.

정답: $59$ 개

핵심: 문제의 본질은 그래프가 아니라 분수 부등식의 자연수 풀이. 함수 통일이 그 환원의 열쇠.

심화 2 — 두 각이 다른 사분면일 때

문제: $\sin\!\left(\dfrac{\pi}{4}\right)$ 과 $\sin\!\left(\dfrac{3\pi}{4}\right)$ 의 대소를 비교하시오. 두 각이 서로 다른 사분면에 있다는 사실에 어떻게 대처해야 하는지 함께 설명하시오.

📖 풀이 보기

STEP 1 — 사분면 확인: $\pi/4$ 는 1사분면, $3\pi/4$ 는 2사분면. 단순 대소 비교 불가.

STEP 2 — 대칭성으로 한 사분면으로 옮기기: $\sin(3\pi/4) = \sin(\pi - 3\pi/4) = \sin(\pi/4)$ ($\sin$ 의 보각 대칭).

STEP 3 — 결론: $\sin(\pi/4) = \sin(3\pi/4) = \dfrac{\sqrt{2}}{2}$.

정답: 두 값은 같다. 이유 — $\sin$ 의 보각 대칭 $\sin\theta = \sin(\pi - \theta)$.

핵심: 영상 강사가 강조한 부분 — 어떤 각 변환을 선택하느냐에 따라 두 각이 같은 사분면에 들어갈지 다른 사분면에 들어갈지가 달라짐. 다른 사분면이면 한 번 더 대칭성으로 옮기면 끝.

🔴 수능 대비

수능 대비 1 — 영상의 핵심 문제 (준킬러 패턴)

문제: $2$ 이상 $10$ 이하의 두 자연수 $a, b$ 가 다음 조건을 만족할 때, 순서쌍 $(a, b)$ 의 개수를 구하시오.

(가) $a+b$ 는 짝수이다.
(나) $\sin\!\left(a\pi + \dfrac{\pi}{b}\right) \le \cos\!\left(b\pi + \dfrac{\pi}{a}\right)$

📖 풀이 보기

STEP 1 — 분기: $a+b$ 짝수 → (둘 다 짝수) 또는 (둘 다 홀수).

STEP 2 — 둘 다 짝수 케이스 ($a, b \in \{2,4,6,8,10\}$):

$a\pi$, $b\pi$ 모두 짝수 $\pi$ → 소거. 식: $\sin(\pi/b) \le \cos(\pi/a)$. 우변 sin 통일: $\sin(\pi/b) \le \sin(\pi/2 - \pi/a)$. 두 각 1사분면 → $\dfrac{\pi}{b} \le \dfrac{\pi}{2} - \dfrac{\pi}{a}$ → $\dfrac{1}{a} + \dfrac{1}{b} \le \dfrac{1}{2}$.

$a=2$: $1/2 + 1/b \le 1/2$ — 불가. $a \ge 4$ 일 때 짝수 $b$ 별:

$a=4$: $1/b \le 1/4$ → $b \ge 4$ → $b \in \{4,6,8,10\}$ — 4개.
$a=6$: $1/b \le 1/3$ → $b \ge 3$ → $b \in \{4,6,8,10\}$ — 4개.
$a=8$: $1/b \le 3/8$ → $b \ge 8/3 \approx 2.67$ → $b \in \{4,6,8,10\}$ — 4개.
$a=10$: $1/b \le 2/5$ → $b \ge 2.5$ → $b \in \{4,6,8,10\}$ — 4개.

짝수 합계: $4 \times 4 = \mathbf{16}$ 개.

STEP 3 — 둘 다 홀수 케이스 ($a, b \in \{3,5,7,9\}$):

$a\pi$, $b\pi$ 모두 홀수 $\pi$ → 부호 반전. $\sin(a\pi + \pi/b) = -\sin(\pi/b)$, $\cos(b\pi + \pi/a) = -\cos(\pi/a)$. 식: $-\sin(\pi/b) \le -\cos(\pi/a)$. 양변 $\times(-1)$ 부등호 뒤집기: $\sin(\pi/b) \ge \cos(\pi/a) = \sin(\pi/2 - \pi/a)$. 1사분면 단조증가 → $\dfrac{1}{a} + \dfrac{1}{b} \ge \dfrac{1}{2}$.

홀수 직접 확인:
$(3,3)$: $2/3 \ge 1/2$ ✓.
$(3,5)$: $1/3 + 1/5 = 8/15 \approx 0.533 \ge 0.5$ ✓.
$(5,3)$: 대칭으로 ✓.
$(3,7)$: $1/3 + 1/7 = 10/21 \approx 0.476 < 0.5$ ✗.
$(5,5)$: $2/5 = 0.4 < 0.5$ ✗.
나머지는 모두 더 작음.

홀수 합계: $\mathbf{3}$ 개 — $(3,3), (3,5), (5,3)$.

정답: $16 + 3 = \mathbf{19}$

핵심: 그래프 $\sin 5x$, $\cos 3x$ 그릴 생각부터 버린다. ① 정수 $\pi$ 소거 (짝수 = 그냥, 홀수 = 부호반전) ② sin 통일 ③ 1사분면 단조증가 — 세 도구가 전부다.

오답주의: 홀수 케이스에서 양변 $\times(-1)$ 시 부등호 방향 뒤집기 잊으면 해 개수가 정반대로 나옴.

수능 대비 2 — 영상 문제의 변형 (반대 방향 부등호)

문제: $2$ 이상 $10$ 이하의 두 자연수 $a, b$ 가 다음 조건을 만족할 때, 순서쌍 $(a, b)$ 의 개수를 구하시오.

(가) $a+b$ 는 짝수이다.
(나) $\sin\!\left(a\pi + \dfrac{\pi}{b}\right) \ge \cos\!\left(b\pi + \dfrac{\pi}{a}\right)$

📖 풀이 보기

관찰: 부등호만 반대. 같은 분기 · 통일 과정 적용.

둘 다 짝수: $\dfrac{1}{a} + \dfrac{1}{b} \ge \dfrac{1}{2}$, $a, b \in \{2,4,6,8,10\}$.

$a=2$: $1/2 + 1/b \ge 1/2$ ⇔ $1/b \ge 0$ — 모든 $b$ 만족 → 5개.
$a=4$: $1/4 + 1/b \ge 1/2$ ⇔ $1/b \ge 1/4$ ⇔ $b \le 4$ → $b \in \{2,4\}$ — 2개.
$a=6$: $1/b \ge 1/3$ ⇔ $b \le 3$ → $b \in \{2\}$ — 1개.
$a=8$: $1/b \ge 3/8$ ⇔ $b \le 8/3 \approx 2.67$ → $b \in \{2\}$ — 1개.
$a=10$: $1/b \ge 2/5$ ⇔ $b \le 5/2 = 2.5$ → $b \in \{2\}$ — 1개.

짝수 합계: $5 + 2 + 1 + 1 + 1 = \mathbf{10}$ 개.

둘 다 홀수: $\dfrac{1}{a} + \dfrac{1}{b} \le \dfrac{1}{2}$, $a, b \in \{3,5,7,9\}$.

$a=3$: $1/b \le 1/6$ ⇔ $b \ge 6$ → $b \in \{7,9\}$ — 2개.
$a=5$: $1/b \le 3/10$ ⇔ $b \ge 10/3 \approx 3.33$ → $b \in \{5,7,9\}$ — 3개.
$a=7$: $1/b \le 5/14$ ⇔ $b \ge 14/5 = 2.8$ → $b \in \{3,5,7,9\}$ — 4개.
$a=9$: $1/b \le 7/18$ ⇔ $b \ge 18/7 \approx 2.57$ → $b \in \{3,5,7,9\}$ — 4개.

홀수 합계: $2 + 3 + 4 + 4 = \mathbf{13}$ 개.

정답: $10 + 13 = \mathbf{23}$

검산: 짝수 쌍 전체 $5 \times 5 = 25$ 가지, 홀수 쌍 전체 $4 \times 4 = 16$ 가지 — 합 $41$. 본 문제(≥, 23개) + 수능 대비 1(≤, 19개) = 42, 중복(등호) 1쌍 차감 → 41 일치 ✓.

핵심: "처음 5초" — 영상 강사가 강조한 부분. 식 형태가 익숙해지면, 부등호 방향만 보고 패턴을 잡아낼 수 있게 됩니다.

✅ 학습자 가이드라인 적용 결과

본 자료는 영상에서 강사가 압축한 부분을 중수준(하위 40%) 학습자도 따라갈 수 있도록 풀어 작성:

  • "sin 5x 안 그려요" → 그래프를 안 쓰는 이유와 대안(함수 통일 · 각변환)을 개념 ① ② 에서 분리 설명.
  • "각 안의 짝수 파이 그냥 지워" → 주기성 $\sin(2n\pi+\theta)=\sin\theta$, 홀수 $\pi$ 는 부호 반전 — 개념 ② 에서 단계별 유도.
  • 각 변환 공식 → $\cos\theta = \sin(\pi/2 \pm \theta)$ 두 형태 모두 단위원에서 유도 (개념 ③). 선택에 따른 사분면 차이도 짚음 (개념 ⑥).
  • "같은 사분면에서 단순 비교" → $\sin$ 의 사분면별 증감을 표로 정리 (개념 ⑤).
  • 홀수 케이스 부등호 뒤집기 → 영상 강사가 빠르게 지나간 부분을 양변 $\times(-1)$ 절차로 명시 (수능 대비 1, STEP 3).
  • 변형 문제 자체 검산 → 수능 대비 2 에서 본 문항과 합산해 등호 케이스로 정합성 확인.
