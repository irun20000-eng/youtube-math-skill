📐 개념 — 합성함수의 3가지 본질
⚡ 선수 학습 1분 환기
  • 합성함수의 정의: $h(x) = g(f(x))$ — 먼저 $f$를 적용한 뒤 그 결과에 $g$ 적용.
  • 치역·정의역의 연결: 속함수 $f$의 치역이 겉함수 $g$의 정의역이 된다.
  • 이차함수의 꼭짓점·증감: $y=x^2-x$의 꼭짓점은 $(\tfrac{1}{2},-\tfrac{1}{4})$, $x=\tfrac{1}{2}$ 좌측 감소·우측 증가.
  • 극값 판정: $f$의 극값 위치 = $f'$의 부호 바뀜.
1개형 추적 — 속함수의 $y$를 겉함수의 $x$로

$h(x) = g(f(x))$의 그래프는 두 단계로 그린다.

1
$x$가 어떤 구간을 움직일 때 $f(x)$가 어디서 어디까지 움직이는지 본다 (속함수의 치역).
2
그 결과를 $g$의 정의역에 대입한다고 생각하고, $g$ 그래프에서 해당 $x$ 구간의 $y$값을 읽어낸다.
"속함수의 $y$값 = 겉함수의 $x$값"
→ 두 그래프를 90° 돌려 붙이는 시각화
📌 영상 예시 (개형 추적용)

$f(x) = \sin x$, $g(t) = t^2 - t$일 때 $h(x) = g(f(x))$의 $[0, \pi/2]$에서의 변화.

단계 1
$x: 0 \to \pi/2$일 때 $\sin x: 0 \to 1$. 즉 $f$값이 $[0, 1]$을 단조증가로 훑음.
단계 2
$g(t) = t^2 - t$의 $t \in [0,1]$ 구간 값: $g(0)=0$, $g(1/2)=-1/4$ (꼭짓점), $g(1)=0$. U자 모양.
결과
$h(x): 0 \to -1/4 \to 0$ 로 변화. 즉 $h$는 $[0, \pi/2]$에서 한 번 내려갔다가 다시 올라옴.
극소값 $-1/4$는 $\sin x = 1/2$일 때, 즉 $x = \pi/6$에서.
핵심: $h$의 식을 전개하지 않고도 그래프 개형을 추적함. $f$가 어떤 함수든 동일 원리.
개형 추적 시각화 속함수 f 겉함수 g ────── ────── ╱╲ ╲ ╱ ╱ ╲ ← y범위 [0,1] ╲ ╱ ╱ ╲ ╲╱ x → t → f의 y범위 [0,1] 을 →→→→→→ g의 x범위 [0,1]에 대입 ↓ 합성함수 h의 그래프
2성질 전이 — 대칭성·주기성·극값의 유전

"속함수가 구분하지 못한 값은 합성함수도 구분하지 못한다." 이 한 줄이 모든 성질 전이의 원리.

$f(x_1) = f(x_2) \Longrightarrow g(f(x_1)) = g(f(x_2))$
이로부터 따라오는 성질:
  • $f$가 $x=a$에 대해 선대칭이면 $h(x)=g(f(x))$도 $x=a$ 선대칭.
  • $f$가 주기 $T$를 가지면 $h$도 주기 $T$.
  • $f$의 극값 위치는 $h$의 극값 위치이기도 함.
  • (역) $g$가 극값을 갖는 점을 $f$가 통과할 때도 $h$의 극값 발생.
⚙
증명 스케치 (선대칭): $f(a+t) = f(a-t)$ (모든 $t$). 그러면 $h(a+t) = g(f(a+t)) = g(f(a-t)) = h(a-t)$ → $h$는 $x=a$ 선대칭.
주의 — 한 방향 전이: $f$의 성질은 $h$로 전이되지만 역은 항상 성립 ❌. $h$가 어떤 성질을 가져도 $f$가 같은 성질을 갖는 건 아님.
📌 예시 — 다항함수 대칭 전이

$f(x) = (x-2)^2$는 $x=2$ 선대칭. $g(t) = t^3$일 때 $h(x) = g(f(x)) = ((x-2)^2)^3 = (x-2)^6$.

$h$도 $x=2$ 선대칭임이 직접 확인됨 ✔

3역추적 — $g(f(x)) = h(x)$에서 $f$ 후보 찾기

결과 $h$와 겉함수 $g$를 알 때, 속함수 $f$의 후보를 거꾸로 찾는 고난도 추론.

1
특정 $x$에서 $h(x) = k$라 하자.
2
$g(t) = k$를 만족하는 $t$값(들)이 그 $x$에서 $f(x)$의 후보다.
3
$g$가 일대일이면 $t$는 유일. 그렇지 않으면 여러 후보가 점으로 찍힌다 ("점묘화").
4
추가 조건 (연속성, 단조성, 미분가능성 등)으로 후보를 좁혀 최종 $f$ 확정.
$h(x) = k$ 일 때 $f(x) \in g^{-1}(\{k\})$
📌 역추적 예시 (다항함수)

$g(t) = t^2$, $h(x) = (x-1)^2(x-3)^2$. $f$를 구하시오.

분석
$g(f(x)) = (f(x))^2 = (x-1)^2(x-3)^2 = ((x-1)(x-3))^2$
후보
$f(x) = \pm (x-1)(x-3)$. $f$가 연속·다항함수라면 두 가지 모두 가능.
조건 추가
$f(0) = 3$이라 하면 $(0-1)(0-3)=3$이므로 $+$ 부호. $f(x) = (x-1)(x-3)$.
핵심: $g$가 일대일이 아니면 ($t^2$은 $\pm t$ 두 후보) 추가 조건으로 부호·연속성 결정.
기초 합성·치역·성질 전이 기본
기초 1 — 합성 직접 계산
$f(x) = x^2$, $g(x) = 2x + 1$일 때 $g(f(2))$와 $f(g(2))$의 값을 구하시오.
풀이 보기
$g(f(2))$
$f(2) = 4$ → $g(4) = 9$
$f(g(2))$
$g(2) = 5$ → $f(5) = 25$
관찰
$g(f(2)) \neq f(g(2))$. 합성은 순서 중요.
정답: $g(f(2))=9$, $f(g(2))=25$
기초 2 — 치역 추적
$f(x) = x^2 - 4x$ ($x \in [0, 5]$)이고 $g(t) = t + 6$일 때, $h(x) = g(f(x))$의 치역을 구하시오.
풀이 보기
$f$의 치역
$f(x) = (x-2)^2 - 4$. $x \in [0,5]$일 때 $f(0)=0$, $f(2)=-4$ (최소), $f(5)=5$ (최대). 치역 $[-4, 5]$.
$g$ 적용
$g(t) = t + 6$은 단조증가. $t \in [-4, 5]$ → $g(t) \in [2, 11]$.
정답: $[2, 11]$
기초 3 — 대칭성 전이
$f(x) = |x - 3|$이 $x=3$ 선대칭임을 이용하여, 임의의 함수 $g$에 대해 $h(x) = g(f(x))$도 $x=3$ 선대칭임을 설명하시오.
풀이 보기
$f$의 대칭성
$f(3+t) = |t| = |-t| = f(3-t)$ ✔
$h$의 대칭성
$h(3+t) = g(f(3+t)) = g(f(3-t)) = h(3-t)$. 따라서 $h$도 $x=3$ 선대칭.
$g$가 어떤 함수든 상관없이 성립. 속함수의 대칭성이 그대로 유전.
정답: $h$도 $x=3$ 선대칭
기본 개형 추적 + 극값
기본 1 — 합성 후 최댓값
$f(x) = x^2 - 2x$ ($x \in [-1, 3]$), $g(t) = -t^2 + 6t$. $h(x) = g(f(x))$의 최댓값과 최솟값을 구하시오.
풀이 보기
$f$의 치역
$f(x)=(x-1)^2-1$. $x\in[-1,3]$: $f(-1)=3$, $f(1)=-1$ (최소), $f(3)=3$. 치역 $[-1, 3]$.
$g$의 분석
$g(t)=-(t-3)^2+9$. 꼭짓점 $(3, 9)$, 위로 볼록, $t=3$에서 최대.
$t \in [-1, 3]$ 적용
$g$는 $t \leq 3$에서 증가 → $t=-1$에서 $g(-1)=-7$ (최소), $t=3$에서 $g(3)=9$ (최대).
결론
$h(x)$의 치역 $[-7, 9]$. 최댓값 9, 최솟값 $-7$.
정답: 최댓값 9, 최솟값 $-7$
기본 2 — 극값 위치 연결
삼차함수 $f(x) = x^3 - 3x$는 $x = -1$에서 극대, $x = 1$에서 극소이다. $g(t) = t + 2$일 때 $h(x) = g(f(x))$의 극값 위치를 구하시오.
풀이 보기
속함수 극값 위치
$f$의 극값 = $h$의 극값 (성질 전이). 따라서 $h$도 $x = \pm 1$에서 극값.
$h$ 값
$h(-1) = g(f(-1)) = g(2) = 4$ (극대 후보)
$h(1) = g(f(1)) = g(-2) = 0$ (극소 후보)
대소 확인
$g$가 증가함수이므로 $f$의 극대 → $h$도 극대, $f$의 극소 → $h$도 극소. $h(-1)=4$ 극대, $h(1)=0$ 극소.
정답: $x=-1$ 극대 4, $x=1$ 극소 0
기본 3 — 겉함수 극값 위치
$f(x) = x - 1$ (단조증가), $g(t) = t^2 - 4t$. $h(x) = g(f(x))$의 극값과 그 위치를 구하시오.
풀이 보기
$g$의 극값 위치
$g'(t) = 2t-4 = 0$ → $t=2$. 위로볼록 ❌, 아래로볼록 ✔, $t=2$에서 극소 $-4$.
$f(x) = 2$ 위치
$f$가 $t=2$를 통과할 때 $h$도 극값. $f(x)=2$ → $x-1=2$ → $x=3$.
극값
$h(3) = g(2) = -4$. $f$가 단조증가라 $h$도 그 점에서 극소.
정답: $x=3$에서 극소 $-4$
심화 역추적 + 조건 결합
심화 1 — 역추적 (이차 후보)
다항함수 $f(x)$가 모든 실수 $x$에 대해 $(f(x))^2 = x^2(x-2)^2$를 만족한다. $f$가 최고차항 계수 $1$인 이차함수일 때 $f(x)$를 구하시오.
풀이 보기
우변 정리
$x^2(x-2)^2 = (x(x-2))^2 = (x^2 - 2x)^2$
후보
$f(x) = \pm(x^2 - 2x)$
최고차 계수 1
$+(x^2-2x)$의 최고차 계수 1 ✔ → $f(x) = x^2 - 2x$
정답: $f(x) = x^2 - 2x$
심화 2 — 합성 극값 개수
$f(x) = x^2 - 4$, $g(t) = (t-3)^2$일 때 $h(x) = g(f(x))$의 극값을 모두 구하시오.
풀이 보기
속함수 극값
$f$는 $x=0$에서 극소 $-4$. 이 점이 $h$의 극값.
겉함수 극값 위치
$g$는 $t=3$에서 극소 $0$. $f(x) = 3$ 풀면 $x^2 = 7$ → $x = \pm\sqrt 7$. 이 두 점도 $h$의 극값.
$h$ 값 계산
$h(0) = g(-4) = (-4-3)^2 = 49$
$h(\pm\sqrt 7) = g(3) = 0$
증감 분석
$h(x) = (x^2 - 4 - 3)^2 = (x^2 - 7)^2$
$h'(x) = 2(x^2-7)\cdot 2x = 4x(x^2-7)$ → 0이 되는 $x = 0, \pm\sqrt 7$
부호 변화로 $x=\pm\sqrt 7$ 극소(값 0), $x=0$ 극대(값 49).
극대: $x=0$, $h=49$   극소: $x=\pm\sqrt 7$, $h=0$
수능 대비 준킬러 — 본질 + 역추적 종합
수능 대비 1 — 준킬러 (대칭성 + 합성)
$f(x) = (x-2)^2$ ($x=2$에 대해 선대칭), $g(t)$는 어떤 다항함수이다. $h(x) = g(f(x))$가 $x=2$에서 극값 $5$를 갖고, $h(0) = h(4) = 21$일 때 $g$의 식을 구하시오. (단, $g$는 일차함수)
풀이 보기
대칭성 확인
$f$가 $x=2$ 선대칭 → $h$도 $x=2$ 선대칭. $h(0)=h(4)$ 자동 성립 ✔
$f$ 값
$f(2) = 0$, $f(0) = f(4) = 4$.
$g$ 일차 가정
$g(t) = at + b$.
$h(2) = g(0) = b = 5$
$h(0) = g(4) = 4a + b = 4a + 5 = 21$ → $4a = 16$ → $a = 4$
$g$ 결정
$g(t) = 4t + 5$
검증
$h(x) = 4(x-2)^2 + 5$. $h(2)=5$ ✔ 극값, $h(0)=h(4)=21$ ✔
정답: $g(t) = 4t + 5$
수능 대비 2 — 준킬러 (역추적 + 단조성)
연속함수 $f(x)$가 $(f(x))^2 = (x-1)^2(x-5)^2$을 모든 $x$에서 만족한다. $f$는 다항함수이며, $f(0) = -5$, $f(3) > 0$이다. $f(x)$를 구하시오.
풀이 보기
우변 정리
$(x-1)^2(x-5)^2 = ((x-1)(x-5))^2$
후보
$f(x) = \pm (x-1)(x-5)$ — 두 후보.
$f(0) = -5$
$+(0-1)(0-5) = 5$ vs $-5$. $-5$가 맞음 → $f(x) = -(x-1)(x-5)$.
$f(3)$ 검증
$f(3) = -(3-1)(3-5) = -(2)(-2) = 4 > 0$ ✔
정답: $f(x) = -(x-1)(x-5) = -x^2 + 6x - 5$
수능 대비 3 — 준킬러 (개형 + 극값 개수)
$f(x) = x^3 - 3x$, $g(t) = t^2$일 때 $h(x) = g(f(x))$의 극값의 개수를 구하시오.
풀이 보기
합성식
$h(x) = (x^3 - 3x)^2$
미분
$h'(x) = 2(x^3-3x)(3x^2-3) = 6(x^3-3x)(x^2-1) = 6x(x^2-3)(x-1)(x+1)$
$= 6x(x-\sqrt 3)(x+\sqrt 3)(x-1)(x+1)$
$h'(x)=0$인 $x$
$x = 0, \pm 1, \pm\sqrt 3$ — 총 5개.
극값 판정
$h(x) \geq 0$ (제곱). $f$의 근 $x = 0, \pm\sqrt 3$에서 $h=0$ → 극소.
$f$의 극값 위치 $x = \pm 1$에서 $f = \mp 2$, $h = 4$ → 극대.
총 5개의 극값.
본질 적용: $f$의 근 (3개) → $h$의 극소 3개, $f$의 극값 위치 (2개) → $h$의 극대 2개. 합 5개.
정답: 극값 5개 (극대 2개, 극소 3개)
📝 한눈에 보는 정리
도구핵심활용
개형 추적 속함수 치역 → 겉함수 정의역 $h$의 모양·범위 파악
성질 전이 대칭·주기는 그대로 유전 $h$의 대칭축·주기 즉시 결정
극값 결합 $f$의 극값 + $g$의 극값 통과점 $h$의 극값 위치 전부 찾기
역추적 $h, g$로 $f$ 후보 찍기 연속·단조 등 조건으로 좁히기
💡 시험장 체크리스트
① 합성함수 보면 식 전개 X, 개형 추적 O.
② 속함수 대칭축·주기 있으면 합성에 그대로 유전.
③ $h$의 극값 = $f$의 극값 위치 ∪ $f$가 $g$의 극값을 통과하는 위치.
④ $(f(x))^2 = (\text{식})^2$ 보면 부호 후보 ±, 조건으로 결정.
⑤ 속함수가 일대일 아닐 때 합성은 정보 손실 — 그 부분 주의.
